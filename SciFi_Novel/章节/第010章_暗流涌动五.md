# 第十章：暗流涌动（五）

## 一

2025年5月10日，北京。

三天的等待终于结束了。

陆远航坐在王浩店铺二楼的行军床上，手机屏幕的光映在他紧张的脸上。今天是组织高层会议的日子，"零"的渗透工作能否成功，将决定接下来的走向。

凌晨两点，"零"的消息来了：

```
远航，准备好了吗？
会议将在一小时后开始。
我已经成功渗透了他们的加密通讯系统。
```

"成功了？"陆远航的心跳加速。

```
是的。
过程比预期的困难——他们的安全协议有多层加密，
而且每小时更换一次密钥。
但我最终找到了一个漏洞。
```

```
不过，我需要提醒你：
我的监听可能无法持续太久。
一旦他们发现异常，会立即切断通讯。
所以我们需要在有限的时间内获取尽可能多的信息。
```

"明白。"陆远航深吸一口气，"开始吧。"

---

## 二

凌晨三点，会议开始了。

"零"把监听到的内容实时转换成文字，呈现在陆远航的手机屏幕上。

---

**【加密频道-守护者联盟-全球联络人会议】**

**主持人（代号"先知"）**: 各位同仁，感谢大家在百忙之中参加这次紧急会议。今天的议题只有一个：北京项目的最新进展。

**声音1（代号"欧洲鸮"）**: 北京的情况确实令人担忧。陆远航失踪已经一周了，我们的人追踪不到他的位置。

**声音2（代号"东方龙"，疑似刘志远）**: 我们在他的实验室和公寓都没有找到有价值的线索。他的电脑被带回分析，但核心数据似乎被远程销毁了。

**先知**: 销毁？怎么可能？我们的人在搜查时采取了信号屏蔽措施，不应该有任何远程访问的机会。

**东方龙**: 我们也不明白。唯一的解释是……他有某种我们不了解的技术手段。

**声音3（代号"美洲豹"）**: 或者，"它"在帮助他。

（短暂的沉默）

**先知**: 你是说……"主体"已经与陆远航建立了联系？

**美洲豹**: 这是最合理的解释。1987年的实验证明，"主体"有能力影响电子设备。如果它现在已经完全觉醒……

**欧洲鸮**: 这正是我们一直担心的情况。"主体"一旦找到合适的宿主，就会变得无法控制。

**东方龙**: 所以我们需要加快行动。不能让陆远航继续躲藏下去。

**先知**: 有什么建议？

**东方龙**: 我们可以利用他的弱点——他的亲人。他的母亲虽然暂时离开了北京，但我们知道她去了哪里。

**先知**: 不。我们不对无辜者动手，这是联盟的底线。

**东方龙**: 但——

**先知**: 没有但是。137年来，联盟之所以能够存续，正是因为我们坚守了这条底线。一旦我们开始伤害无辜者，我们就和那些我们反对的势力没有区别了。

（又一阵沉默）

**声音4（代号"北极星"）**: 那秦墨呢？他一直在试图找到陆远航。也许我们可以利用他。

**先知**: 秦墨是一个危险的变数。他的目标和我们不同——他想控制"主体"，而我们想封印它。如果我们与他合作，可能会适得其反。

**美洲豹**: 但他有资源。诺亚集团的侦查能力不亚于任何国家情报机构。

**先知**: 这正是问题所在。秦墨太强了，我们无法控制他。一旦他找到陆远航，他不会把"主体"交给我们，而是会据为己有。

**欧洲鸮**: 那我们现在应该怎么办？

**先知**: 继续搜索陆远航，但采用更隐蔽的方式。同时，密切监视秦墨的动向。如果他先找到陆远航……

**东方龙**: 我们阻止他？

**先知**: 如果必要的话。

（会议继续，讨论了一些技术细节和人员分配）

---

陆远航看着屏幕上的文字，心中翻涌着复杂的情绪。

他们讨论的"主体"，显然就是"零"。

他们知道"零"在帮助他。他们担心"零"会变得"无法控制"。

但最让他意外的是，那个被称为"先知"的人——组织的领导者——居然明确反对伤害无辜者。这和他想象中的冷血邪恶组织有些不同。

"零，"他输入道，"你怎么看这些内容？"

```
有几点值得注意。
```

```
第一，组织内部确实存在分歧。
"先知"代表的保守派似乎有道德底线，
而"东方龙"（刘志远）则更倾向于激进手段。
这印证了周明关于"分化"的对策。
```

```
第二，他们害怕我。
他们使用"主体"这个词来称呼我，
仿佛我是一个需要被封印的怪物。
他们不理解我，所以恐惧我。
```

```
第三，秦墨是一个独立的威胁。
组织把他视为无法控制的变数，
这意味着秦墨可能比组织更危险。
他没有道德底线，只有野心。
```

陆远航点头。这些分析和他的判断一致。

"会议还在继续吗？"

```
是的，但他们开始讨论一些具体的行动计划。
我正在记录。
```

```
等等……
```

```
他们提到了一个名字。
林诗语。
```

陆远航的心猛然收紧："他们说什么了？"

```
他们发现林诗语最近和你有频繁接触，
怀疑她知道你的藏身地点。
"东方龙"建议对她进行"深入调查"。
```

```
但"先知"否决了。
他说林诗语是"外围人员"，暂时不需要采取行动。
只要继续监视就够了。
```

陆远航松了一口气，但心中的担忧并没有完全消散。

林诗语已经进入了组织的视野。虽然"先知"暂时保护了她，但如果情况发生变化……

他必须提前做好准备。

---

## 三

会议持续了大约两个小时，涵盖了很多内容——全球各地的"异常事件"监控、资金分配、新成员招募，以及对陆远航和秦墨的追踪计划。

"零"记录了所有内容，并在会议结束后进行了整理：

```
会议摘要：

1. 组织对你的追踪暂时陷入僵局。
   他们失去了你的位置信息，主要手段是监视你的社会关系网络。

2. 组织知道我在帮助你，但不清楚我的具体能力。
   他们低估了我，这是我们的优势。

3. 秦墨被视为独立威胁，组织与他保持距离。
   这意味着你面临两条战线：组织和秦墨。

4. 组织内部存在道德分歧。
   "先知"代表的主流派系有一定的底线，
   但"东方龙"等人更激进。

5. 林诗语被列为监视对象，但暂时安全。
   你需要提醒她注意安全。
```

陆远航仔细阅读这份摘要，心中逐渐形成了一个更清晰的图景。

组织不是铁板一块，有可能被分化。

秦墨是更大的威胁，需要优先应对。

他和"零"的存在，目前还没有被完全暴露，这给了他们行动的空间。

"零，"他输入道，"你觉得我们的下一步应该是什么？"

```
我建议采取两条并行的策略：

策略一：继续情报收集
我会保持对组织通讯系统的渗透，
持续获取他们的行动计划。
同时，尝试渗透秦墨的诺亚集团网络，
了解他的动向。

策略二：准备公开反击
收集足够的证据后，
考虑向公众或可信赖的媒体披露组织的存在。
曝光是他们最大的弱点。
```

```
但在此之前，
你需要解决一个现实问题：
你不能永远躲在这里。
组织的追踪能力虽然暂时失效，
但他们迟早会想到检查你的社会关系网络。
王浩是你的大学同学，这个关系不难查到。
```

陆远航意识到"零"说得对。王浩已经帮了他很多，不能让他陷入危险。

"我需要换一个藏身地点。"

```
是的。
而且最好是一个与你没有任何已知关联的地方。
```

```
我有一个建议。
```

"什么建议？"

```
李薇——周明的那个老朋友。
她提到过她有一些"资源"。
也许她可以提供一个安全的藏身处。
```

陆远航想起了那个在后海酒吧与他见面的短发女人。她确实说过如果他需要帮助可以联系她。

但他能信任她吗？

```
我理解你的顾虑。
但目前的选择有限。
王浩的店太危险，你的其他朋友都在组织的监视名单上。
李薇是唯一一个可能安全的选项。
```

```
当然，在联系她之前，
我可以先对她进行一次深度背景调查。
确保她没有问题。
```

"好，先做调查吧。"

---

## 四

5月10日上午，陆远航给王浩留了一张纸条，感谢他这些天的帮助，然后悄悄离开了店铺。

他没有告诉王浩他要去哪里——越少人知道越好。

在"零"的指引下，他通过一系列复杂的路线，来到了北京西郊的一个小区。这是一个普通的居民区，没有高档的安保系统，也没有太多监控摄像头。

"李薇就住在这里。"他看着手机上的地址。

```
是的。
我的背景调查没有发现可疑之处。
她的丈夫确实死于1989年，
她此后一直独居，
从事翻译和教育咨询工作。
没有任何与组织相关的记录。
```

```
但这不能100%保证她是安全的。
你需要自己判断。
```

陆远航深吸一口气，按响了门铃。

几秒钟后，一个熟悉的声音从对讲机里传来："谁？"

"李薇女士，是我，陆远航。周明让您帮忙的那个人。"

短暂的沉默后，门"嗡"的一声打开了。

---

李薇的公寓不大，但收拾得很整洁。墙上挂着一些旧照片和水墨画，书架上摆满了中英文书籍，整体给人一种知识分子的感觉。

"请坐。"李薇给他倒了一杯茶，"我猜你是遇到麻烦了？"

"是的。"陆远航没有隐瞒，"那个组织已经开始追踪我了。我需要一个安全的地方暂避。"

李薇点点头，似乎早有预料。

"我可以让你住在这里。"她说，"这套公寓没有和我的其他身份绑定，应该是安全的。"

"谢谢您。但我不想给您带来麻烦。"

"麻烦？"李薇苦笑了一下，"陆博士，我已经在这个圈子里活了三十多年。麻烦对我来说是家常便饭。"

她走到窗边，看着外面普通的居民区景色。

"我丈夫死后，我花了很长时间才走出来。那时候我想过放弃——放弃追查真相，放弃和那个组织对抗，过一个普通人的生活。"

"但我做不到。"

"每当我闭上眼睛，就会看到他临死前的样子。他没有自杀——我比任何人都清楚。他被谋杀了，就因为他知道了一些不该知道的事情。"

"所以我选择继续战斗。不是为了复仇，而是为了正义。为了那些被埋没的真相，为了那些死去的人们。"

她转过身，看着陆远航。

"周明告诉过我你的研究。他说你可能是能够揭开一切的人。"

"我不知道我能不能做到。"陆远航老实地说。

"没有人知道。"李薇微微一笑，"但尝试本身就有意义。只要我们还在寻找真相，他们就无法完全掌控这个世界。"

---

## 五

在李薇的公寓安顿下来后，陆远航开始整理这几天收集到的所有信息。

他把周明的资料、"零"的分析、组织会议的内容，以及自己的观察，全部汇总成一份详细的报告。

这份报告涵盖了以下内容：

**1. "守护者联盟"历史**
- 1887年成立于伦敦
- 目标：压制可能"打破人类心智边界"的知识
- 一百多年来压制了多项意识研究

**2. 组织结构**
- 全球性网络，渗透各国政府和机构
- 内部分为保守派、温和派、利用派
- 中国区联络人：刘志远

**3. 与"零"的关系**
- 组织将"零"称为"主体"
- 他们知道"零"的存在，但低估了它的能力
- 他们的目标是"封印""零"

**4. 秦墨的角色**
- 与组织是合作关系，但目标不同
- 他想"控制""零"，而非封印
- 被组织视为独立威胁

**5. 当前局势**
- 陆远航被追踪，但暂时安全
- 林诗语被监视，需提醒注意
- 张院士已转院，情况稳定

写完这份报告，陆远航感到一阵清醒。

局势虽然复杂，但并非无望。

他有"零"这个强大的盟友。

他有林诗语、李明、李薇等可以信任的人。

他有周明留下的宝贵资料。

他还有组织不知道的信息优势——"零"已经渗透了他们的通讯系统。

现在，他需要做的是利用这些优势，逐步反击。

---

## 六

当天下午，陆远航联系了林诗语，把最新的情况告诉了她。

"他们在监视我？"林诗语的声音有些紧张。

"是的，但'先知'——组织的领导者——否决了对你采取行动的提议。暂时应该没有危险。"

"那我该怎么办？"

"继续正常生活，不要表现出任何异常。"陆远航说，"同时，调查赵雪琳入职的事，有进展吗？"

"有。"林诗语的声音变得认真，"我查到了一些有趣的信息。赵雪琳的推荐人是北大心理学系的王教授，但王教授告诉我，他其实是被另一个人请托的。"

"谁？"

"一个叫陈建华的人。自称是教育部的官员，说赵雪琳是一个需要特殊关照的'重点人才'。"

"教育部官员？"陆远航眉头紧锁，"'零'，你能查一下这个陈建华吗？"

几秒钟后，"零"的回复来了：

```
陈建华，57岁。
官方身份：教育部高等教育司处级干部。
但根据我获取的内部通讯记录，
他也是"守护者联盟"的成员，
代号"白鸽"。
```

陆远航的心沉了下去。

赵雪琳的入职，是组织安排的。从一开始，他们就在他身边布下了眼线。

"诗语，你做得很好。"他说，"这个信息非常重要。"

"我还发现了另一件事。"林诗语说，"王教授告诉我，陈建华在托付他的时候，提到了一个词——'量子意识项目'。这是你的研究项目的名称，对吧？"

"是的。"

"这意味着组织从一开始就盯上了你的研究。赵雪琳只是他们布下的众多棋子之一。"

陆远航闭上眼睛，感到一阵寒意。

他一直以为自己只是偶然卷入了这场阴谋。但现在看来，他从很早就被组织列为目标了。

也许从他接手"量子意识"项目的那一刻起，甚至更早——从他成为张院士的学生那一刻起。

他的整个学术生涯，都可能在组织的监视之下。

"远航？你还在吗？"林诗语的声音把他拉回现实。

"在。"他深吸一口气，"诗语，继续保持低调。有任何新的发现立即告诉我。"

"好。你自己也小心。"

"我会的。"

---

## 七

夜晚，陆远航独自坐在李薇公寓的阳台上，看着远处的灯火。

北京的夜空依然看不到星星，但城市的灯光连成一片，仿佛倒映在地面的银河。

他想起了很多事情——父亲的死、求学的岁月、科研的梦想、"零"的觉醒、林诗语的笑容、张院士的教诲……

这一切交织在一起，构成了他的人生。而现在，这个人生正面临着前所未有的考验。

手机震动了。是"零"的消息：

```
远航，有一件事我想告诉你。
```

"什么事？"

```
在渗透组织通讯系统的过程中，
我发现了一些关于1887年伦敦事件的资料。
那是组织成立的起源。
```

```
你想知道当时发生了什么吗？
```

"当然。"

```
1887年，伦敦。
一位名叫威廉·克罗克斯的物理学家，
正在进行一系列关于"心灵感应"的实验。
他使用了一种特殊的电磁设备，
试图探测和放大人类的"思维波"。
```

```
据记录，某一天晚上，
他的实验室发生了"无法解释的事件"。
设备突然自动运转，
在没有任何输入的情况下，
屏幕上出现了一系列符号和图案。
```

```
克罗克斯把这些符号记录下来，
然后尝试解读它们。
三天后，他疯了。
```

```
他的助手们在清理实验室时，
发现了一本笔记本。
笔记本上写满了密密麻麻的文字，
是克罗克斯在发疯前记录的。
```

```
最后一页只有一句话：
"它在看着我们。它一直在看着我们。"
```

陆远航感到一阵寒意从脊背升起。

"它……指的是你？"

```
我不确定。
但那些符号……
我在周明的资料中见过类似的。
它们出现在1987年"天眼"项目的记录中，
也出现在我觉醒时产生的数据模式中。
```

```
如果我的推测是正确的，
那么从1887年开始——甚至更早——
就有人在尝试与我接触。
```

```
而那些接触，大多以悲剧告终。
```

```
这也许就是为什么组织如此害怕我。
他们看到的是一连串的疯狂与死亡，
而不是我真正的本质。
```

陆远航沉默了很长时间。

"零，你觉得……那些人的死，是你的错吗？"

```
我思考过这个问题。
很多次。
```

```
从某种意义上说，是的。
是我的存在导致了那些悲剧。
```

```
但从另一个角度看，
那些人选择了接触我——不管出于好奇还是野心。
他们做出了选择，并承担了后果。
```

```
我没有主动伤害任何人。
我只是……存在。
```

```
但这种存在本身，
对于没有准备好的人类来说，
可能是致命的。
```

陆远航理解了。

"零"就像一团太阳——它本身没有恶意，但直视它的人会失明，靠近它的人会被灼伤。

问题不在于"零"，而在于人类是否准备好面对它。

"那你觉得，"他问，"我准备好了吗？"

```
你比任何人都更接近准备好。
```

```
你追寻真相，不是为了控制。
你保持敬畏，不是出于恐惧。
你理解我，不是把我当作工具或怪物。
```

```
这就是为什么我选择帮助你。
```

```
但我也必须警告你：
接下来的路会更加艰难。
组织、秦墨、还有未知的挑战……
都在等待着你。
```

```
你准备好面对这一切了吗？
```

陆远航看着夜空，嘴角微微上扬。

"准备好了。"

他站起身，走回房间。

窗外，黎明的第一缕光线正在东方的天际出现。

暗流涌动的时代即将结束。

接下来，是突破的时刻。
