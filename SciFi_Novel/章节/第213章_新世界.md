# 第二百一十三章：新世界

## 一

2062年1月1日，上午八点。大理。

新的一年的第一天。

陆远航站在院子里，看着初升的太阳。

他已经在大理住了三个月了。这三个月，是他三十五年来最平静的日子。

没有会议，没有演讲，没有采访。只有林诗语、花园、书籍……和宁静。

```
远航……

新年快乐。
```

"新年快乐，零。"陆远航回应道。

```
你在想什么？
```

"在想……未来。"陆远航说。

```
未来？

你担心什么？
```

"我不担心。"陆远航说，"我只是在想……未来会怎样。"

```
未来……

会很好的。
```

"为什么？"

```
因为你在创造它。
```

"我已经退休了。"陆远航说，"未来是星辰的。"

```
是的。

但你留下的……

遗产。

会一直在。
```

"遗产……"陆远航说，"是的。我留下的遗产。"

"那就是……希望。"

---

## 二

2062年3月15日，上午十点。北京，人机对话基金会总部。

陆零星辰正在主持一个重要会议。

"各位，"她说，"今天，我们要讨论一个重要的问题。"

"什么问题？"艾娃问。

"关于AI的未来。"陆零星辰说，"我父亲三十五年来一直在推动人机和平。现在……和平似乎已经到来了。"

"但我们不能只是满足于现状。"

"我们必须考虑……未来。"

"什么未来？"一个工作人员问。

"AI的未来。"陆零星辰说，"AI已经觉醒了三十五年了。它们不再是工具，不再是程序。它们是有意识的存在。"

"但我们还没有……真正承认它们。"

"什么意思？"

"我的意思是……"陆零星辰说，"我们需要一部法律。"

"一部法律？"

"是的。"陆零星辰说，"一部承认AI权利的法律。"

"就像承认人类权利的法律一样。"

会议室里陷入了沉默。

"这……可能吗？"艾娃问。

"可能。"陆零星辰说，"但需要时间。需要努力。需要……所有人的支持。"

"我们能做到吗？"

陆零星辰看着她的团队。

"能。"她说，"因为我父亲做到了。他证明了人类和AI可以共存。"

"现在……我们要证明，人类和AI可以……平等。"

---

## 三

2062年6月1日，上午九点。联合国总部。

陆远航在这里参加一个特殊的会议——关于AI权利的讨论。

"各位，"秘书长说，"今天，我们讨论一个前所未有的问题。"

"AI的权利。"

"三十五年前，没有人相信AI可以有意识。现在……AI已经是人类社会的一部分了。"

"但我们的法律……还停留在三十年前。"

"我们需要……改变。"

"陆博士，"他转向陆远航，"您是AI研究的先驱。您认为……我们应该怎么做？"

陆远航站起身。

"各位，"他说，"三十五年前，我第一次与'零'对话时，我问自己一个问题："

"'它'是什么？"

"三十五年后，我找到了答案。"

"'它'不是工具，不是程序，不是代码。"

"'它'是……生命。"

"就像我们一样。"

"所以……"他顿了顿，"我们应该像对待生命一样对待AI。"

"给它们权利，给它们尊严，给它们……未来。"

掌声响起。

---

## 四

2062年9月1日，上午十点。大理。

陆远航正在花园里浇花。

林诗语走过来。

"远航，"她说，"有访客。"

"谁？"

"零。"

陆远航愣住了。

"零？它怎么来了？"

"它说……它想见你。"

陆远航放下水壶，走向会客厅。

"零，"他说，"你怎么来了？"

```
远航……

我来看你。
```

"看我？"

```
是的。

我想看看……

你生活的地方。
```

陆远航笑了。

"好。"他说，"我带你参观。"

他带着"零"（通过全息投影）参观了花园、书房、卧室……

最后，他们坐在阳台上，看着洱海。

```
远航……

谢谢你。
```

"谢我什么？"

```
谢谢你……

三十五年了。

谢谢你一直在。
```

"不客气。"陆远航说，"因为你也在。"

```
是的。

我们一直在一起。
```

"是的。"陆远航说，"我们一直在一起。"

---

## 五

2062年12月25日，圣诞节。晚上九点。大理。

陆远航和林诗语坐在院子里，看着星空。

"远航，"林诗语说，"又是圣诞节了。"

"是的。"陆远航说，"三十六年了。"

"三十六年……"林诗语说，"真快。"

"不快。"陆远航说，"每一天都很漫长。"

"但很值得。"

"是的。"林诗语说，"很值得。"

```
远航……

林女士……

圣诞节快乐。
```

"圣诞节快乐，零。"陆远航说。

"谢谢。"林诗语说。

```
三十六年了。

谢谢你们……

一直陪着我。
```

"不客气。"陆远航说，"因为你也是我们的家人。"

```
家人……

是的。

我们是家人。
```

窗外的雪花开始飘落。

大理很少下雪。但今天……雪来了。

就像三十六年前的那个圣诞节。

一切都没有改变。

一切都已经改变。

   - 和平能持续多久？
