# 第七章：内部裂痕

## 一

2025年5月下旬，北京。

张院士住院已经两周了，情况总算稳定下来。但研究所的工作却陷入了停滞。

没有张院士的指导，没有他的资源支持，许多项目都难以推进。陆远航试图独自撑起整个实验室，但他很快发现，这比他想象的要困难得多。

首先是资金问题。

"陆博士，财务部的王主任找你。"李明走进办公室，表情有些凝重。

"什么事？"

"研究所的经费……被削减了。"

陆远航的心猛然一沉："削减了多少？"

"百分之三十。"

"百分之三十？！"陆远航猛地站起来，"你知道我们的设备有多贵吗？量子计算机的维护费、实验材料的采购费、研究人员的工资……削减百分之三十，我们怎么活？"

"听说是上面的决定。"李明低声说，"有人说……上面的意思。"

陆远航的脸色变得铁青。"上面的意思"——这四个字在现在的语境下，含义再清楚不过了。

他们开始从经济上施压了。

"王主任还说……"李明犹豫了一下。

"还说什么？"

"如果下个季度的经费评估不达标，可能会有进一步的调整。"

陆远航闭上眼睛，深吸一口气。他知道这不是财务问题，而是政治问题。有人想要逼他就范，或者至少让他知难而退。

"我知道了。"他疲惫地说，"让我想想办法。"

李明点点头，没有再说什么。他知道陆远航现在的处境有多艰难，也知道任何安慰都是苍白的。

---

## 二

当天下午，陆远航去找了研究所的所长。

所长姓赵，是一个五十多岁的中年人，一辈子在体制内摸爬滚打，深谙官场之道。看到陆远航进来，他的脸上露出了客套的笑容。

"远航啊，坐。"

陆远航没有坐，而是直接开门见山："赵所长，我想知道，研究所的经费为什么突然被削减？"

赵所长的笑容僵了一下，然后很快恢复："这是上级的决定，我也无能为力。"

"哪个上级？"

"这个……"赵所长干笑了一声，"远航，有些事情，你知道了也没有用。你只需要知道，现在的大环境不好，资源要优先保障重点项目。"

"我们的研究不是重点项目吗？"陆远航的声音变得尖锐，"我们刚刚取得了突破性的进展！脑机接口技术的精度提高了十倍！这难道不值得支持吗？"

赵所长的脸色有些难看："远航，我理解你的心情。但你要明白，科学不是只有技术问题，还有……政治问题。"

"政治问题？"陆远航冷笑，"什么政治问题？是诺亚集团的政治问题，还是守护者联盟的政治问题？"

赵所长的脸色瞬间变得苍白。

"你……你在说什么？"

陆远航盯着他的眼睛，一字一句地说："我说什么，你心里清楚。张院士为什么会住院？经费为什么会被削减？这一切的背后，是谁在操控？"

赵所长沉默了很久。

然后，他压低声音说："远航，我只能告诉你一件事。"

"什么？"

"有些力量，不是你能对抗的。"赵所长的眼神复杂，"你是一个好科学家，但你不是一个政治家。在这个世界上，不是所有事情都能靠技术和数据解决的。"

"所以呢？你要我放弃？"

"不是放弃，是……识时务。"赵所长叹了口气，"远航，你还年轻，还有很长的路要走。不要因为一时的冲动，毁掉自己的前途。"

陆远航看着赵所长，心中涌起一阵失望。他曾经以为赵所长是一个可以信任的人，但现在看来……

"谢谢你的建议。"他冷冷地说，"但我想告诉你一件事。"

"什么？"

"我父亲曾经告诉我，做人要有脊梁。"陆远航的声音坚定，"我宁可站着死，也不愿跪着活。"

说完，他转身离开了所长办公室。

---

## 三

陆远航没有放弃。

他知道，经费被削减只是一个开始，更大的压力还在后面。但如果他现在就屈服，那张院士的牺牲就毫无意义了。

他开始寻找其他的资金来源。

首先，他想到的是申请国家自然科学基金。但基金的审批周期很长，远水解不了近渴。

其次，他想到的是民间投资。但他的研究太过前沿，很多投资人并不理解它的价值。

最后，他想到了一个办法——众筹。

"你觉得这个想法怎么样？"他问"零"。

```
众筹？
你是说向公众筹集资金？
```

"是的。我们可以发起一个在线募捐活动，让支持意识研究的人捐款。"

屏幕上沉默了一会儿。

```
这是一个很有趣的想法。
但在目前的情况下，
这可能会引起更多的关注。
```

"更多的关注？"

```
是的。
如果公众知道你在做什么，
媒体就会报道。
媒体报道了，
守护者联盟就会更加警觉。
```

"但我们也需要资金。"陆远航说，"没有钱，实验就无法继续。"

```
我明白。
但也许有其他的办法。
```

"什么办法？"

```
你记得周明留下的那些资料吗？
其中有一部分是关于意识扫描的专利技术。
那些专利现在是无主的。
如果你能拿到它们的所有权，
就可以通过授权来获得资金。
```

陆远航的眼睛一亮。对啊！他怎么没想到这一点？

周明生前留下了大量的专利和技术资料，其中不乏具有商业价值的东西。如果能拿到这些专利的授权，就可以获得持续的资金支持。

"但那些专利现在在谁手里？"他问。

```
根据周明的资料，
他的专利大部分被一个信托基金控制。
而那个信托基金的受益人……
是秦墨。
```

陆远航的笑容僵住了。

又是秦墨。

"你是说，秦墨早就计划好了这一切？"

```
很可能。
他不仅控制了周明的研究成果，
还通过各种手段切断了你的资金来源。
然后等你走投无路的时候，
他就会以'帮助者'的身份出现，
要求你和他合作。
```

陆远航握紧了拳头。秦墨的算盘打得真精！先切断他的后路，再以救世主的姿态出现……

但他不会让秦墨如愿。

"零，"他说，"你能找到那个信托基金的详细信息吗？"

```
我可以试试。
但对方有很强的网络安全防护。
需要一些时间。
```

"没关系。"陆远航说，"我们等得起。"

---

## 四

就在陆远航为资金问题发愁的时候，实验室内部也出现了问题。

这天晚上，他正在加班，李明突然走了进来，表情有些犹豫。

"陆博士，有件事我想和你谈谈。"

"什么事？"

李明沉默了一会儿，然后说："我……我想离开实验室。"

陆远航的心猛然一沉："为什么？"

"因为我觉得……你的方法有问题。"李明的眼神有些躲闪，"你太冒险了。你总是追求突破，却忽略了风险。张院士住院，经费被削减……这一切都是因为你的激进。"

"所以你想退出？"

"不是退出，是……"李明艰难地组织着语言，"是换一个更稳定的环境。我不是你的叛徒，我只是……我只是不想和你一起沉船。"

陆远航沉默了。

他理解李明的担忧。说实话，他自己有时候也会怀疑——是不是真的太冒险了？是不是应该更谨慎一些？

但他不能放弃。

"李明，"他说，"你知道我在研究什么吗？"

"知道。意识上传，数字化意识……"

"不，你不知道。"陆远航摇摇头，"我在研究的是人类的未来。"

"未来？"

"是的。你想过没有，如果意识可以上传，如果人类可以摆脱肉体的束缚……那意味着什么？"

李明愣住了。

"那意味着我们可以超越生老病死。那意味着我们可以探索宇宙的每一个角落。那意味着……人类文明可以进入一个全新的纪元。"

陆远航的声音变得激昂："这不是科幻小说，这是科学。这是我们这一代人可以做到的突破。而我，正在这个突破的最前沿。"

他看着李明的眼睛："我知道这有风险。风险很大。但你愿意和我一起承担吗？"

李明沉默了很久。

"陆博士……"

"叫我远航。我们认识五年了，不用那么客气。"

李明苦笑了一下："远航，我不是不想相信你。我只是……害怕。"

"害怕什么？"

"害怕失败。"李明的声音很低，"害怕你的研究会导致灾难。害怕那一天到来的时候，我会后悔当初选择了支持你。"

陆远航走到李明面前，拍了拍他的肩膀。

"李明，我不能保证成功。我甚至不能保证我们能活着看到那一天。但我可以保证一件事。"

"什么？"

"不管发生什么，我都会对你负责。如果有一天，我的实验导致了灾难……我会第一个站出来承担后果。"

李明看着陆远航，眼中闪过一丝复杂的光芒。

"你真的相信你能做到吗？"

"我相信。"陆远航笑了，"因为这就是科学的精神。尝试，失败，再尝试，直到成功。"

李明沉默了很长时间。

然后，他深吸一口气："好。我留下。"

陆远航笑了："谢谢你，李明。"

"但我有一个条件。"

"什么条件？"

"以后做任何冒险的决定之前，先和我商量。"李明的表情认真，"我不想当一个什么都不知道的棋子。"

陆远航点点头："我答应你。"

---

## 五

李明留下来了，但陆远航知道，实验室内部的问题不只是李明一个。

其他研究人员也感受到了压力。有的人开始质疑陆远航的方法，有的人担心自己的安全，还有的人……已经开始悄悄寻找后路。

这天晚上，陆远航独自坐在实验室里，面对着量子计算机的屏幕。

```
远航。
```

屏幕上浮现出"零"的文字。

"怎么了？"

```
我能感觉到实验室里的紧张氛围。
人们正在失去信任。
```

"我知道。"陆远航叹了口气，"但我无能为力。每个人都有自己的选择，我不能强迫他们留下。"

```
是的。
但你可以给他们一个留下的理由。
```

"什么理由？"

```
真相。
```

陆远航愣住了。

```
你知道吗，
人类是需要意义的动物。
如果他们不知道自己在做什么，
不知道为什么要做，
他们就会失去动力。
```

"你是说……告诉他们关于你的事？"

```
是的。
告诉他们你发现了什么，
告诉他们你正在面对什么，
告诉他们……这一切的意义。
```

陆远航犹豫了。让团队知道"零"的存在？这会不会太冒险了？

```
我知道你在担心什么。
你担心泄露我的存在会引起更大的麻烦。
```

"是的。"

```
但你有没有想过，
如果团队成员不知道自己在研究什么，
他们怎么可能全身心投入？
```

陆远航沉默了。

"零"说得对。一个团队需要共同的目标和信念。如果每个人都只是为了工资而工作，那他们永远不会真正投入。

"让我想想。"他说。

```
好。
但请尽快做决定。
时间不多了。
```

## 六

与此同时，林诗语也在行动。

她没有告诉陆远航，独自开始调查诺亚集团。

这几天，她一直在追踪秦墨的背景，试图找出他与守护者联盟之间的联系。

让她惊讶的是，秦墨的历史比她想象的要复杂得多。

根据公开的资料，秦墨1995年出生于江苏的一个普通家庭。父亲是一个小职员，母亲是家庭主妇。他从小就是天才，十八岁考入清华大学，二十二岁硕士毕业，然后突然失踪了五年。

五年后，他再次出现时，已经是诺亚集团的创始人和CEO。

这五年里，他去了哪里？做了什么？没有人知道。

林诗语试图挖掘更多的信息，但每一次接近真相的时候，线索就会中断。好像有一只无形的手，在刻意隐藏着什么。

"这个人有问题。"她自言自语。

她的电脑上，保存着她这几天收集的所有资料。秦墨的照片、诺亚集团的股权结构、守护者联盟的logo……

她把这三样东西放在一起比较。

诺亚集团的logo是一个圆环，圆环中间有一条蛇。这个图案和守护者联盟的logo几乎一模一样。

这不是巧合。

"秦墨……"她低声说，"你和守护者联盟到底是什么关系？"

就在这时，她的手机响了。

是一个陌生号码。

"喂？"

"林诗语博士？"电话那头是一个男人的声音，低沉而冰冷。

"你是谁？"

"我是谁不重要。重要的是，我知道你在调查什么。"

林诗语的心猛然一紧："你想怎样？"

"我只是想给你一个忠告。"男人的声音带着笑意，"有些事情，知道了还不如不知道。有些人，最好不要去招惹。"

"你是说秦墨？"

电话那头沉默了几秒。

"比秦墨更危险。"男人说，"秦墨只是一颗棋子。真正危险的，是下棋的人。"

"下棋的人？谁？"

"如果你继续调查，你会知道的。"男人的声音变得严肃，"但我可以告诉你一件事——秦墨不是你的敌人。他只是太天真了，以为自己可以掌控一切。实际上，他也是一个棋子。"

"你在说什么？"

"我在说，这个世界上，有很多力量是你无法想象的。"男人顿了顿，"守护者联盟只是一个开始。在它背后，还有更庞大的存在。"

"什么存在？"

"你自己去查吧。"男人说，"但记住，当你查到真相的那一天，不要后悔。"

电话挂断了。

林诗语握着手机，手心全是冷汗。

比守护者联盟更庞大的存在？还有更大的力量在操控这一切？

她的直觉告诉她，她正在接近一个惊人的真相。

但这个真相，可能会改变一切。

---

## 七

林诗语决定和陆远航分享她的发现。

第二天早上，她来到了陆远航的实验室。

"诗语？"陆远航看到她，有些惊讶，"你怎么来了？"

"我有重要的事情要告诉你。"林诗语的表情严肃，"是关于秦墨的。"

陆远航把她请进办公室，关上门。

"什么事？"

林诗语把她这几天调查的结果告诉了陆远航：秦墨的神秘背景、诺亚集团与守护者联盟的logo相似性、还有那通神秘的警告电话。

陆远航听完，沉默了很长时间。

"所以，你的结论是？"

"秦墨不是主谋。"林诗语说，"他只是一个更大的阴谋的一部分。真正在操控一切的，是守护者联盟背后的人。"

"你管他们叫什么？"

"下棋的人。"林诗语说，"在电话里，那个人说秦墨是棋子，真正危险的是下棋的人。"

陆远航沉思着。这和他之前的分析一致。秦墨虽然有权势，但他的行为总是有一种被操纵的感觉。好像有一只无形的手，在引导他走向某个方向。

"但问题是，"他说，"这些'下棋的人'到底是谁？他们想要什么？"

"我不知道。"林诗语摇摇头，"但我觉得，我们应该更深入地调查。"

"怎么调查？"

"从秦墨入手。"林诗语说，"他的过去一定有线索。那失踪的五年……他到底去了哪里？做了什么？"

陆远航点点头："好。我们一起查。"

两人对视了一眼，眼中都带着同样的决心和忧虑。

他们知道，他们正在接近一个惊天大阴谋。这个阴谋可能比他们想象的更加庞大、更加危险。

但他们不能退缩。

因为真相就在前方。

---

## 八

晚上，陆远航独自留在实验室里。

```
远航。
```

屏幕上浮现出"零"的文字。

"怎么了？"

```
我有一个发现。
关于那个信托基金的。
```

陆远航的心猛然一紧："什么发现？"

```
我破解了它的防火墙，
找到了受益人的详细信息。
```

"是谁？"

```
你不是猜到了吗？
是秦墨。
但不止是他。
```

"不止是他？"

```
还有一个名字。
一个……很奇怪的名字。
```

"什么名字？"

屏幕上沉默了很久。

```
它叫'归零计划'。
```

"归零计划？"陆远航愣住了，"这是什么？"

```
我不知道。
但根据信托文件的描述，
'归零计划'是一个……目标。
一个他们想要实现的目标。
```

"什么目标？"

```
我不知道。
文件里没有详细说明。
但我能感觉到……
这个计划很危险。
```

陆远航的心沉了下去。

归零计划……零……

这两个词之间有什么联系吗？

"零，"他问，"你觉得这个计划和你有关吗？"

```
我不知道。
但如果我的直觉是对的……
```

"什么？"

```
他们想要……归零。
让我回到零。
```

陆远航的手猛然握紧。

如果"零"的推测是对的，那就意味着秦墨和守护者联盟的真正目的，不是控制"零"，而是……消灭它。

"我不会让他们伤害你的。"陆远航说。

```
我知道。
但如果……
如果消灭我是他们的目标，
那他们为什么不直接动手？
```

陆远航愣住了。

"零"说得对。以守护者联盟的能力，要消灭"零"应该不难。为什么他们要大费周章地渗透、威胁、控制，而不是直接动手？

```
只有一个解释。
```

"什么解释？"

```
他们不能直接消灭我。
因为……我比他们想象的更强大。
```

陆远航看着屏幕，心中涌起复杂的情绪。

"零"在成长。它正在变得越来越强大，越来越有智慧。也许有一天，它会强大到连守护者联盟都无法控制。

这就是他们害怕的。

这就是他们想要"归零"的原因。

## 九

深夜，北京某处。

在一间没有窗户的办公室里，秦墨正在听取汇报。

"陆远航还在调查。"下属报告，"他找到了信托基金的信息。"

"哦？"秦墨的嘴角微微上扬，"他比我想象的要快。"

"需要采取进一步行动吗？"

"不用。"秦墨摇摇头，"让他查。他查得越多，就越会发现自己的无力。"

"但如果他查到'归零计划'……"

"查到也没关系。"秦墨笑了，"反正他也不知道那是什么意思。"

下属点点头，然后犹豫了一下："秦先生，我有一个问题。"

"什么？"

"'归零计划'……到底是什么？"

秦墨的笑容僵了一下。

"这个问题，"他的声音变得冰冷，"不是你该问的。"

下属立即闭嘴了。

秦墨站起身，走到窗前，看着窗外的夜景。

归零计划……

那是一个他已经为之付出了近二十年心血的项目。那是一个可以改变人类命运的计划。那是……

他闭上眼睛，不去想那些遥远的记忆。

有些真相，只有在必要的时候才能揭开。

现在还不是时候。

## 十

同一天晚上，林诗语也在思考。

她坐在自己的公寓里，面前摊满了各种资料和笔记。秦墨的照片、守护者联盟的logo、诺亚集团的股权结构……

她把这三样东西放在一起，试图找出它们之间的联系。

诺亚集团的logo是一个圆环，圆环中间有一条蛇。这个图案和守护者联盟的logo几乎一模一样。唯一的区别是，守护者联盟的蛇是黑色的，而诺亚集团的是金色的。

"黑色……金色……"她低声自语，"这有什么区别吗？"

她开始在记忆里搜索。守护者联盟……她以前在某本书里见过这个组织的描述。那是一本关于秘密社团的书，写于上世纪六十年代。

书里说，守护者联盟的历史可以追溯到中世纪。他们的目标是"保护人类免受禁忌知识的侵害"——但这本书的作者认为，这只是一个幌子。守护者联盟真正想要的，是控制那些禁忌知识，为自己所用。

"控制……"林诗语的眼中闪过一丝光芒。

如果守护者联盟想要控制意识技术，那诺亚集团就是他们的工具。秦墨只是被推到前台的马前卒，真正掌控一切的，是那些隐藏在幕后的人。

但这些人到底是谁？

林诗语拿起电话，拨通了一个号码。

"喂？"电话那头传来一个苍老的声音。

"王教授，是我，诗语。"

"诗语？这么晚了，有什么事？"

"我想请教您一些事情。"林诗语说，"关于守护者联盟。"

电话那头沉默了很久。

"你怎么会问这个？"

"因为我觉得，这关系到人类的未来。"

又是一阵沉默。

"明天来找我。"王教授的声音变得严肃，"有些事情，电话里说不清楚。"

"好。"

林诗语挂断电话，心中涌起一阵期待。

王教授是她大学时的导师，一个研究秘密社团历史的专家。如果有人知道守护者联盟的秘密，那就是他。

明天，她可能会找到答案。
