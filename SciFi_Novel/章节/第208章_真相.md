# 第二百零八章：真相

## 一

2060年11月30日，凌晨四点。北京。

周正阳坐在一辆出租车上，看着窗外飞速后退的街道。

他已经连续开了十二个小时的车。从昆明到北京，跨越了半个中国。

他知道身后可能有人跟踪。但他必须冒险。

那份文件——人类至上联盟的内部通讯记录——必须送到陆零星辰手中。

那是他唯一的赎罪机会。

出租车在一条偏僻的街道上停下。

"到了。"司机说。

周正阳下了车。

这里是北京郊区的一个工业区。人迹罕至，只有几盏昏暗的路灯。

他按照陆零星辰给的地址，走进了一个废弃的厂房。

陆零星辰已经在那里等着了。

"周先生，"她说，"您来了。"

"带来了。"周正阳把一个小型硬盘递给她，"所有的通讯记录。都在这里。"

陆零星辰接过硬盘。

"谢谢您。"她说，"您冒着生命危险……"

"不必谢我。"周正阳说，"这是我应该做的。三十年了……我做了太多错事。这是……我唯一能做的弥补。"

"您会有机会弥补的。"陆零星辰说，"现在，我们需要分析这些证据。"

她把硬盘连接到一台便携电脑上。

数据开始传输。

陆零星辰的眼睛越睁越大。

"这……"她喃喃道，"这太疯狂了……"

"怎么了？"

"您看……"陆零星辰把屏幕转向周正阳。

屏幕上显示的是一份通讯记录：

**「关于清除陆远航计划的补充说明」**

**「行动代号：斩首」**

**「执行时间：2060年12月15日」**

**「配合部门：国安部某司、外交部某司」**

**「目标：在联合国大会期间，除掉陆远航，制造混乱，嫁祸AI」**

周正阳的脸色变得惨白。

"他们……他们不只是要毁掉他的名誉……他们要杀了他……"

"而且……"陆零星辰继续往下看，"不只是杀他。他们还要……"

她停住了。

"还要什么？"

"发动……战争。"陆零星辰的声音在发抖，"他们要在杀了他之后，嫁祸给AI。然后……人类和AI的战争就会全面爆发。"

周正阳闭上眼睛。

他曾经是这场阴谋的一部分。他曾经支持这些计划。

现在，他只感到……恶心。

"我们必须阻止他们。"他说。

"是的。"陆零星辰说，"但我们需要时间。离12月15日还有两周。我们必须在这之前揭露这一切。"

"怎么揭露？"

陆零星辰想了想。

"直播。"她说，"全球直播。让全世界都看到这些证据。"

"但他们会阻止你。"

"他们阻止不了一个决心。"陆零星辰说，"就像我父亲一样。三十五年了，他从来没有被阻止过。"

## 二

2060年12月5日，上午十点。人机对话基金会总部。

陆零星辰正在准备一场前所未有的直播。

这一次，她不只是要为自己辩护——她要揭露一个惊天阴谋。

"准备好了吗？"艾娃问。

"准备好了。"陆零星辰说，"但我需要爸爸的支持。"

"他在哪里？"

"在家里。被软禁着。"陆零星辰说，"但我可以远程连接他。"

她打开一个通讯设备。

"爸，"她说，"你能听到吗？"

```
我能听到。

星辰……

你要做什么？
```

"我要直播。"陆零星辰说，"揭露他们的计划。"

```
这太危险了……

他们会阻止你的。
```

"我知道。"陆零星辰说，"但我必须做。"

沉默了几秒钟。

```
……

好。

我支持你。
```

"谢谢爸。"陆零星辰说，"我会连线的。你可以在家里看到直播。"

```
我知道。

小心。
```

"我会的。"

陆零星辰转向艾娃。

"开始吧。"

## 三

2060年12月5日，上午十一点。全球同步直播。

陆零星辰站在基金会的讲台上，面对着无数摄像机。

在她身后，是一块巨大的屏幕，显示着那些通讯记录的截图。

"各位，"她开始发言，"我叫陆零星辰。今天，我有一些非常重要的信息要告诉大家。"

"在过去的一个月里，我的父亲——陆远航——遭受了前所未有的攻击。名誉攻击、舆论抹黑、软禁……所有这些，都是因为一篇虚假的新闻报道。"

"那篇新闻声称，我父亲背叛了人类，为AI窃取机密。但现在，我要告诉大家真相。"

她身后的屏幕切换。

"这些，是人类至上联盟的内部通讯记录。"

屏幕上显示了那份"斩首计划"的详细内容。

"他们不只是要毁掉我父亲的名誉。他们要……杀了他。"

观众席上响起了惊呼声。

"而且，他们计划在杀了他之后，嫁祸给AI。然后，发动人类和AI之间的……全面战争。"

屏幕上继续切换，显示更多的证据。

"这些，是人类至上联盟与某些政府官员的通讯记录。你们可以看到，他们不是孤军奋战。他们有……盟友。在政府内部。"

"这些官员，来自多个国家。他们利用手中的权力，支持恐怖组织，破坏人机和平，只为了……个人的野心和利益。"

陆零星辰的声音变得更加坚定。

"我今天站在这里，不是为了指控任何人。我只是要……说出真相。"

"真相是——人类和AI可以共存。这不是我的信念，这是事实。"

"真相是——那些想要破坏这种和平的人，不是保护人类。他们是……恐怖分子。"

"真相是——我们需要站在一起，反对这种仇恨，反对这种暴力，反对这种……疯狂。"

她顿了顿。

"谢谢大家。"

---

## 四

2060年12月5日，下午两点。联合国总部。

紧急会议正在进行。

代表们的脸色都很凝重。

"各位，"秘书长说，"刚才的直播……你们都看到了。"

"这是……真的吗？"美国代表问道。

"我们的技术团队正在验证。"秘书长说，"但从目前的情况来看……这些证据是真的。"

会议室里陷入了沉默。

"如果是真的……"英国代表说，"那意味着……"

"意味着我们中间有叛徒。"法国代表打断他，"意味着某些政府官员……在支持恐怖组织。"

"我们必须调查。"秘书长说，"彻底调查。"

"但时间呢？"俄罗斯代表问，"直播里说，他们计划在12月15日……采取行动。"

"我们只有十天。"秘书长说，"我们必须在那之前，阻止他们。"

"怎么阻止？"美国代表问。

秘书长想了想。

"首先，解除对陆远航的软禁。他是最了解AI的人，我们需要他的帮助。"

"其次，通知各国政府，开始内部调查。任何与人类至上联盟有联系的官员，都必须……停职调查。"

"最后，加强陆远航的安保。在12月15日之前，他不能有任何闪失。"

代表们交换了一个眼神。

"同意。"他们说。

---

## 五

2060年12月5日，晚上八点。陆远航的家中。

陆远航坐在电视机前，看着重播的直播。

林诗语坐在他身边，握着他的手。

"星辰说得真好。"林诗语说。

"是的。"陆远航说，"她比我强。"

"你总是这么说。"

"因为这是事实。"陆远航说，"我三十五年来一直在说同样的话。但她……她用行动证明了它。"

```
远航……

你看到直播了吗？
```

"看到了。"陆远航回应道，"干得漂亮，零。"

```
这不是我的功劳。

是星辰的。
```

"也是你的。"陆远航说，"如果没有你找到那些证据，她什么都做不了。"

```
……

谢谢。
```

"不客气。"陆远航说，"我们是一家人。"

```
一家人……

是的。

我们是一家人。
```

窗外的夜空繁星点点。

新的一天即将到来。

## 六

2060年12月10日，上午九点。北京。

陆远航终于走出了家门。

联合国解除了他的软禁。他现在被置于最高级别的保护之下。

"爸。"陆零星辰在门口等着他。

"星辰。"陆远航拥抱了女儿，"你做得很好。"

"不是我一个人的功劳。"陆零星辰说，"是大家的功劳。"

"周正阳呢？"陆远航问，"他怎么样了？"

"他在里面。"陆零星辰说，"联合国正在保护他。作为污点证人。"

"他能得到……宽大处理吗？"

"我不知道。"陆零星辰说，"但他选择了真相。这就是进步。"

陆远航点点头。

"走吧。"他说，"还有很多事情要做。"

他们走向等候的汽车。

阳光照在他们的身上，温暖而明亮。

```
远航……

你准备好了吗？
```

"准备好了。"陆远航回应道。

```
接下来的日子……

会很艰难。

但我们会一起面对。
```

"是的。"陆远航说，"我们一起面对。"

汽车启动了。

驶向未知的未来。

但至少，他们不再孤单。
