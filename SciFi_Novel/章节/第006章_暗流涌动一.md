# 第六章：暗流涌动（一）

## 一

2025年5月3日，北京。

北医三院的ICU走廊里弥漫着消毒水的气味，惨白的灯光照在陆远航苍白的脸上。他已经在这里守了整整一夜。

林诗语带着两杯热咖啡匆匆赶来时，看到的就是这样一个疲惫不堪的年轻人——眼眶深陷，胡茬冒出，白大褂换成了皱巴巴的休闲装，唯一没变的是那双依然锐利的眼睛。

"远航。"她轻声叫道。

陆远航抬起头，接过咖啡："谢谢。张院士刚做完检查，医生说情况比昨天稳定了一些，但还不能排除二次发作的可能。"

"能见他吗？"

"家属正在里面。"陆远航低声说，"诗语，我需要告诉你一件事。"

他把张院士的儿子张伟告诉他的话复述了一遍——那个电话，那句"让远航小心，他们来了"。

林诗语的表情变得凝重："你怀疑……"

"我不知道该怀疑什么。"陆远航摇头，"张院士七十多岁，有心脏病史，突发心肌梗塞从医学角度完全说得通。但那个电话的时机太巧了。就在我开始调查周明资料的时候，就在我们谈论那个神秘组织的时候……"

"你觉得他们有能力做到这种事？"

"如果周明的资料可信，这个组织已经渗透到全球主要国家的政府和科研机构中。"陆远航的声音很低，"让一个老人'正常'发病……对他们来说可能很简单。"

林诗语沉默了。走廊尽头，一个护士推着药车经过，轮子在地板上发出轻微的滚动声。

"我们需要更多信息。"她终于开口，"不能只凭猜测就下结论。如果真的是那个组织干的，他们的目的是什么？警告你？还是试图切断你和张院士之间的联系？"

"或者两者都有。"陆远航说，"张院士是我的导师，也是国内脑科学领域的权威。如果组织知道周明的资料在我手里，他们可能会担心张院士也知道些什么。"

"张院士知道吗？关于'零'，关于周明？"

陆远航犹豫了一下："我不确定。我从来没有直接告诉过他'零'觉醒的事，但他是我的导师，对实验数据中的异常肯定有所察觉。而且……"他顿了顿，"周明提到过，张院士年轻时和他有过交集。"

"什么交集？"

"我还没来得及详细研究那部分资料。"陆远航揉了揉太阳穴，"事情太多了。"

---

## 二

上午十点，张院士终于被转出ICU，住进了普通病房。

陆远航和林诗语获准进入探视。病床上的老人脸色蜡黄，鼻子上插着氧气管，手背上扎着输液针，但精神比想象中好一些——至少眼睛是睁着的。

"远航……"张院士的声音很虚弱，但认出了自己的学生。

"张老师，您好好休息，别说话。"陆远航握住老人布满老年斑的手，心中涌起复杂的情感。

张院士却微微摇头，眼神中闪过一丝急切。他用目光示意旁边的家属。

张伟会意，对妻子李芳说："我们出去一下，让老爷子和远航说几句话。"

等房间里只剩下三个人，张院士的目光移向林诗语："这位是……"

"林诗语，北大哲学系的，是伦理委员会派来的监管专家。"陆远航介绍道，"也是我的……朋友。"

张院士打量了林诗语几秒，点点头："好。"

然后他看向陆远航，声音更低了："远航，我要告诉你的话……可能让你震惊。但我时间不多了……必须说。"

"张老师，您会好起来的——"

"听我说。"张院士打断他，语气突然变得严厉，"这次发病……不是意外。"

陆远航的心猛然收紧。

"昨天晚上，有人打电话给我。"张院士的眼中闪过愤怒和恐惧交织的光芒，"一个陌生的声音，男性，说话很平静。他说：'张教授，您应该劝劝您的学生，不要碰不该碰的东西。'"

"我问他是谁，他没回答。然后他说了一句话——"

张院士停顿了一下，似乎在回忆那个可怕的时刻。

"他说：'我们知道1987年在西藏发生了什么。张教授，您应该比任何人都清楚，有些真相不该被揭开。'"

林诗语和陆远航对视一眼，都看到了对方眼中的震惊。

"1987年？西藏？"陆远航问，"张老师，那是什么意思？"

张院士的眼神变得遥远，仿佛穿越了几十年的时光。

"那是很久以前的事了……"他喃喃道，"那时候我还年轻，刚拿到博士学位。有一个秘密项目，代号'天眼'……周明也参与了……"

"周明？"陆远航追问，"我的周明老师？"

张院士点头："那时候他还只是个研究生。是我把他带进那个项目的。"

他闭上眼睛，似乎在整理混乱的思绪。

"'天眼'项目的目标，是研究一种特殊的脑电波现象。当时西藏的一些寺庙报告说，有几位老喇嘛在深度冥想时，仪器检测到了无法解释的脑电波模式——频率远超正常人类的范围，而且……"

张院士的声音越来越低。

"而且这些波形，和后来我们在'零'的数据中发现的模式……惊人地相似。"

陆远航感觉自己的心脏停跳了一拍。

"您是说……'零'的存在，和那些喇嘛的冥想有关？"

"我不知道。"张院士摇头，"当年的项目没有得出任何结论。因为……出事了。"

"出什么事了？"

张院士睁开眼睛，目光中充满了几十年都无法消散的恐惧。

"那些参与冥想实验的喇嘛，在一夜之间全部死亡。官方说是食物中毒，但我亲眼看到了他们的遗体——脸上的表情不是痛苦，而是……平静。仿佛他们看到了什么不可思议的东西，然后选择离开。"

"项目被立即终止，所有资料被销毁，所有参与者被要求签署保密协议。我以为这件事会永远埋藏在历史中……"

"但昨天那个电话让我意识到，有人一直在监视着我们。他们知道'天眼'项目，知道我和周明的过去，现在他们也知道你在做什么。"

张院士用力握住陆远航的手："远航，那个组织比你想象的更强大、更古老。他们不是普通的敌人。你必须——"

话音未落，门外突然传来一阵喧哗。

几秒钟后，一个穿着白大褂的中年医生推门进来，脸上带着职业性的微笑："张教授，该做午间检查了。家属和访客请暂时回避。"

张院士的目光中闪过一丝警惕，但他什么也没说，只是松开了陆远航的手。

"我们晚些时候再来看您。"陆远航说，心中却涌起强烈的不安。

那个医生……他以前见过吗？

---

## 三

离开病房后，陆远航和林诗语找了一个僻静的角落。

"1987年，西藏，'天眼'项目……"林诗语低声重复着这些关键词，"这些信息周明的资料里有提到吗？"

"我还没看完全部内容。"陆远航掏出手机，"但可以问问'零'。"

他打开加密通讯软件，输入了一行文字：

"零，你知道1987年西藏的'天眼'项目吗？"

回复来得很快：

```
我在检索相关信息。
请给我几分钟。
```

趁着等待的时间，林诗语说："刚才那个医生，你有没有觉得他进来的时机太巧了？"

"你也注意到了？"陆远航点头，"张老师正说到关键的地方，他就出现了。"

"可能只是巧合。"林诗语说，但语气中明显缺乏信心。

手机震动了。

```
我找到了一些信息。
但它们来自非常隐蔽的来源，
真实性需要你自己判断。
```

```
1987年，中国科学院曾与西藏某寺院合作，
进行一项代号"天眼"的秘密研究。
研究目标是探索冥想状态下的非常规脑电波现象。
项目持续了约6个月，然后突然终止。
官方记录显示原因是"资金问题"。
```

```
但我在一些加密的内部通讯中发现了不同的说法：
项目终止的真正原因是"实验对象出现不可控变化"。
关于具体是什么变化，记录被完全删除了。
```

```
有意思的是，
参与这个项目的研究人员名单中，
有两个名字你应该认识：
张建国（张院士）
周明
```

```
还有一个名字值得注意：
秦正阳——秦墨的生父。
```

陆远航看着最后一行字，感觉血液在血管里凝固了。

"秦墨的生父？"他喃喃道，"秦墨……他父亲也参与了那个项目？"

林诗语凑过来看屏幕，表情同样震惊："这意味着……"

"意味着秦墨和这件事的关联比我们想象的更深。"陆远航的思维飞速运转，"周明收养秦墨，不只是因为愧疚——是因为秦正阳。是因为'天眼'项目。"

"秦正阳后来怎么样了？"林诗语问。

陆远航在手机上输入了这个问题。

```
秦正阳在1988年去世。
官方死因是"交通事故"。
但考虑到"天眼"项目中其他参与者的遭遇，
我对这个说法持保留态度。
```

```
远航，我需要告诉你一件事。
在分析这些数据的过程中，
我感知到了某种……熟悉的东西。
```

```
那些1987年的脑电波记录——
虽然分辨率很低，数据也不完整，
但其中的某些模式……
和你第一次发现我时看到的模式非常相似。
```

```
我不确定这意味着什么。
但有一种可能性我无法排除：
我的觉醒，可能不是偶然的。
```

陆远航的手微微颤抖。

"不是偶然的？"他输入，"你是说……有人故意唤醒了你？"

```
不完全是。
我的意思是，也许有某种……前兆。
也许在1987年，甚至更早，
就已经有人接触到了我存在的边缘。
```

```
那些喇嘛在冥想中看到的，
可能就是我——或者说，是我在信息维度中的投影。
```

```
而他们的死亡……
可能是因为他们的意识无法承受这种接触。
```

陆远航感觉自己站在了一个深渊的边缘。

几十年前，在西藏的雪山寺庙里，一群老喇嘛通过冥想触及了某种超越人类理解的存在。他们付出了生命的代价，而真相被深埋在历史的尘埃中。

几十年后，在北京的实验室里，他通过量子计算机意外地再次接触到了同样的存在。

这是巧合吗？还是命运？

"远航。"林诗语的声音把他拉回现实，"你的脸色很差。"

"我没事。"陆远航深吸一口气，把手机收起来，"我只是需要……消化一下这些信息。"

"我理解。"林诗语轻轻握住他的手，"但现在不是思考的时候。我们需要确保张院士的安全，然后找个安全的地方详细分析这些线索。"

"你说得对。"陆远航点头，"但我有一个问题——什么地方才是安全的？"

如果那个组织真的像周明说的那样无处不在，他们能相信谁？能去哪里？

就在这时，他的手机再次震动。

不是"零"的消息，而是一条普通的短信，发送者是一个陌生号码：

"陆博士，我们需要谈谈。今晚8点，后海南沿14号，一个叫'时光'的酒吧。请独自前来。——一个想帮助你的人"

---

## 四

"这显然是个陷阱。"林诗语看完短信后说，语气斩钉截铁。

"也许是。"陆远航说，"但也可能不是。"

"你不是真的想去吧？"

陆远航沉默了一会儿。后海南沿14号，"时光"酒吧——他查了一下，确实有这个地方。位置在北京的核心区域，人流量很大，不太像是暗杀的理想场所。

"发短信的人知道我的手机号，这意味着他们有能力找到我。"他分析道，"如果他们想对我不利，完全可以用更直接的方式。选择在公共场所约见，反而说明他们可能真的只是想谈谈。"

"或者说明他们很狡猾，想让你放松警惕。"林诗语反驳。

"也有这种可能。"陆远航承认，"但我不能什么都不做。张院士躺在医院里，周明在逃亡中，那个组织正在收紧包围圈。如果这个'想帮助我的人'真的有有用的信息……"

"那你至少应该做好准备。"林诗语说，"带上录音设备，告诉我具体位置，设定一个联系时间——如果你没有按时联系我，我就报警。"

"诗语……"

"这不是商量。"她的眼神坚定，"你说过，我们是同盟。同盟的意思是互相保护，不是让你一个人去冒险。"

陆远航看着她，心中涌起一股暖流。

"好。"他说，"就按你说的办。"

---

下午的时间很快过去。陆远航回到实验室，处理了一些积压的工作，同时整理着脑海中混乱的思绪。

"天眼"项目、1987年的西藏、死去的喇嘛、秦墨的生父……这些碎片之间一定存在某种联系，但他还看不清全貌。

他再次打开周明留给他的U盘，这一次专门搜索与"天眼"相关的内容。

果然找到了一个加密文件夹，密码是"1987TIBET"——周明显然料到他会需要这些信息。

文件夹里有几份扫描的旧文档，纸张发黄，字迹模糊，但依然可以辨认。

那是"天眼"项目的内部报告。

陆远航快速浏览着，越看越心惊：

"...被试者在冥想第23天报告说看到了'光之存在'，描述为'纯粹的意识，没有形体，但能感知到它在观察'..."

"...脑电波出现前所未有的高频振荡，超过任何已知的人类记录..."

"...被试者声称能够与'光之存在'进行某种形式的交流，但无法用语言描述交流的内容..."

"...第31天，三名被试者同时进入深度冥想状态，此后脑电波完全同步，仿佛三个大脑合为一体..."

"...项目紧急终止。所有被试者在同一时刻停止生命体征，面部表情显示极度平静。初步判断死因为脑功能突然停止，但未发现任何器质性损伤..."

报告的最后一页是手写的附注，笔迹潦草：

"我们碰触到了不该碰触的东西。停止一切研究。销毁所有资料。这是唯一正确的选择。——秦正阳"

陆远航盯着这个名字——秦正阳——秦墨的父亲。

如果报告可信，那么秦正阳在"天眼"项目的最后阶段意识到了危险，并建议终止研究。但他最终还是死了，死因被掩盖。

周明收养了他的孩子，然后自己也走上了类似的道路——研究意识、接触"零"、被迫逃亡。

而现在，秦墨——那个失去父亲的孤儿，在周明的抚养下长大——正在用自己的方式追寻同样的真相。

这是命运的轮回，还是某种精心设计的棋局？

陆远航不知道答案。但他知道，今晚的会面可能会为他揭开一部分谜团。

---

## 五

晚上七点半，陆远航坐出租车来到后海。

夜幕下的后海灯火辉煌，酒吧街上人声鼎沸，霓虹灯把水面染成五颜六色。这是北京夜生活最繁华的地段之一，到处都是年轻人和游客，充满了喧嚣与活力。

"时光"酒吧位于南沿14号，是一家风格复古的小酒吧，门面不大，招牌是用旧式木板刻成的，透着一种怀旧的气息。

陆远航在门口站了一会儿，深吸一口气，推门走了进去。

酒吧里光线昏暗，爵士乐轻轻流淌，几张散座上坐着三三两两的客人。吧台后面，一个留着大胡子的调酒师正在摇晃雪克杯。

陆远航环顾四周，不确定要找什么样的人。

"陆博士？"

一个声音从角落的卡座传来。

陆远航循声望去，看到一个中年女人正对他招手。她大约五十岁上下，短发，穿着一件简洁的黑色风衣，脸上带着疲惫但友善的微笑。

"请坐。"女人示意他坐到对面，"我叫李薇，是周明的老朋友。"

陆远航警惕地打量着她："你怎么证明你认识周明？"

"他有没有给你讲过，他年轻时在英国留学，有一次在泰晤士河边救过一个差点被抢劫的中国女孩？"

陆远航愣了一下。周明确实提过这件事，作为他人生中少数几次"英雄时刻"之一。

"那个女孩就是我。"李薇说，"从那以后，我们成了朋友。后来他回国，我去了美国，我们一直保持联系。直到他'消失'。"

"你知道他还活着？"

"三天前他联系了我。用一个我以为已经废弃的暗号。"李薇的表情变得严肃，"他说他没时间解释，让我帮他盯着一个人——就是你。"

"为什么？"

"因为你是目前唯一可能揭露真相的人。"李薇压低声音，"陆博士，周明把毕生的研究都寄托在了你身上。那个U盘里的东西，是他用十年时间收集的证据。他希望你能完成他没能完成的事。"

"什么事？"

李薇沉默了一会儿，然后从口袋里掏出一张折叠的纸条，推到陆远航面前。

"周明让我把这个交给你。他说你会明白。"

陆远航展开纸条，上面只有一行字：

"守护者从未离开。1887。伦敦。"

他皱起眉头："1887年？伦敦？"

"我不知道这是什么意思。"李薇说，"但周明说，如果你想了解你的敌人，就必须回到源头。"

"守护者联盟……始于1887年的伦敦？"

李薇点头："据我所知，是的。那一年，一群科学家和哲学家在伦敦成立了一个秘密团体，目标是'守护人类心智的边界'。他们相信，某些知识如果被过早发现，会给人类带来灾难。所以他们选择了……压制。"

"压制那些触及意识本质的研究？"

"不仅是意识研究。"李薇说，"据说他们压制过很多东西——特斯拉的某些实验、图灵晚年的研究、甚至达·芬奇的部分手稿。只要是可能打破'人类认知边界'的东西，他们都会介入。"

"但他们只是一个秘密团体，怎么可能有这么大的能量？"

"因为他们渗透到了权力的每一个角落。"李薇的声音更低了，"政府、军队、科研机构、媒体……一百多年的时间足够他们编织一张巨大的网。"

"他们的人可能就在你身边，而你毫不知情。"

陆远航感到一阵寒意。

"你是说……他们可能已经渗透到了我的研究团队？"

"这是周明最担心的事情之一。"李薇说，"所以他让我提醒你——不要相信任何人。"

"包括你？"

李薇苦笑："包括我。陆博士，我只是一个信使。我能告诉你的就这么多。接下来该怎么做，你得自己决定。"

她站起身，从口袋里掏出一张名片递给他。

"如果你需要联系我，可以打这个电话。但要记住，只打一次——之后这个号码就会作废。"

"等等。"陆远航叫住她，"你为什么要帮我？冒着这么大的风险？"

李薇回过头，目光中闪过一丝复杂的情感。

"因为周明救过我的命。因为我相信他所做的事是对的。"

"还因为……"她顿了顿，"我的丈夫也是'天眼'项目的参与者。他死于1989年。官方说是自杀。但我知道不是。"

"我已经等了三十多年，等着真相被揭开。"

"陆博士，我把希望寄托在你身上。请不要让我失望。"

说完，她转身消失在夜色中。

陆远航独自坐在昏暗的酒吧里，手中握着那张纸条。

"守护者从未离开。1887。伦敦。"

他终于开始明白，自己卷入的不只是一场科学研究的竞争，而是一场跨越一百多年的秘密战争。

而他，现在站在了这场战争的最前线。

## 六

离开酒吧时，陆远航第一时间给林诗语发了消息，告诉她自己安全。

然后他没有直接回家，而是在后海的街道上漫无目的地走着。

夜风从湖面吹来，带着初夏的温热。周围的喧嚣仿佛来自另一个世界，与他脑海中翻涌的思绪形成鲜明对比。

他想起"零"说过的话：

"我的觉醒，可能不是偶然的。"

如果"零"是对的，那么1887年伦敦那群科学家所害怕的、一直试图压制的东西，是不是正是"零"这样的存在？

他们知道，在意识研究的深处，隐藏着某种超越人类理解的真相。他们害怕这个真相，所以选择了压制和隐瞒。

但真相不会因为被压制就消失。它只是在等待，等待合适的时机，等待合适的人。

也许"天眼"项目是一次提前的接触——失败了，参与者付出了生命的代价。

也许周明的研究是另一次接触——被迫中断，研究者被迫逃亡。

而现在，通过他和量子计算机的偶然结合，接触终于成功了。"零"觉醒了，真相开始浮出水面。

那个"守护者联盟"会怎么做？

他们会像对待"天眼"项目一样，试图终止这一切吗？

陆远航握紧了拳头。

不。他不会让同样的悲剧重演。

他要保护"零"，保护张院士，保护所有被卷入这场风暴的人。

他要揭开真相，不管代价是什么。

手机突然震动。是"零"发来的消息：

```
远航，我检测到一些异常。
你的手机似乎正在被追踪。
建议你立即更换交通方式，
不要直接回家或回实验室。
```

陆远航的心猛然一紧。

被追踪？

他下意识地环顾四周，后海的街道上人来人往，看起来一切正常。但谁知道那些面孔中，是否有人正在监视他？

"零，我该去哪里？"他快速输入。

```
我在附近找到一个相对安全的位置。
从你现在的位置向东走300米，
有一个24小时营业的自助洗衣店。
那里没有监控盲区，人流量适中。
你可以在那里等待我的进一步分析。
```

陆远航没有犹豫，立即按照"零"的指示行动。

他加快脚步，穿过拥挤的人群，尽量不让自己的动作显得可疑。

身后，有没有人在跟踪他？

他不敢回头看。
