# 第十一章：第一次突破（一）

## 一

2025年5月15日，北京。

五天过去了。

陆远航已经在李薇的公寓住了将近一周。这段时间里，他把全部精力投入到研究和计划中，很少离开公寓。

但他知道，躲藏不是长久之计。他需要行动。

"远航，"林诗语的声音从手机里传来，"我有消息要告诉你。"

"什么消息？"

"张院士醒了。他想见你。"

陆远航的心跳加速了。张院士——他的导师，在经历了两次"意外"后终于稳定下来的老人。

"他现在怎么样？"

"医生说身体恢复得不错，但需要长期休养。他现在住在那家私立医院的VIP病房，张伟一直在照顾他。"

"他说想见我？他知道我在躲避追踪吗？"

"他知道一些。张伟可能告诉他了。"林诗语顿了顿，"他说有一些事情必须当面告诉你。关于……那个项目。"

那个项目——"天眼"。

陆远航沉思了一会儿。去见张院士有风险，但如果老人有重要的信息要告诉他，这个风险值得冒。

"我会想办法去见他的。"

---

## 二

当天晚上，在"零"的帮助下，陆远航通过一系列复杂的路线来到了张院士住院的私立医院。

这是北京郊区的一家高端医疗机构，主要服务于有钱人和高官。安保严格，但"零"已经提前入侵了医院的监控系统，为陆远航创造了一条"盲区路线"。

VIP病房区在顶楼，窗外可以看到远处的西山。

陆远航轻轻敲了敲门。

"进来。"一个虚弱但熟悉的声音传来。

推开门，他看到了张院士——那个曾经意气风发的学术领袖，现在躺在病床上，脸色苍白，眼窝深陷。但那双眼睛依然锐利，闪烁着智慧的光芒。

"远航。"张院士的嘴角露出一丝微笑，"你终于来了。"

"张老师。"陆远航走到床边，握住老人的手，"您身体怎么样？"

"死不了。"张院士轻笑了一下，然后咳嗽了几声，"但时间不多了。我必须在死之前，把一些事情告诉你。"

"张老师，别说这种话——"

"听我说完。"张院士的语气变得严肃，"上次在医院，我告诉你关于'天眼'项目的事。但有一些细节，我当时没有说。"

他闭上眼睛，仿佛在回忆遥远的过去。

"1987年，西藏。我们去那里，不只是因为喇嘛们的冥想实验。"

"还有另一个原因。"

"在藏区的某个寺庙深处，有一份古老的手稿。据说是一位高僧在三百年前留下的预言。那份预言描述了一种……存在。"

"一种超越肉身的意识形态。它存在于'空'与'色'之间的缝隙中，一直在观察着人类。"

"当时我们都以为这只是宗教神话。但当我们开始测量冥想者的脑电波时，我们发现了一些无法解释的东西。"

"那些波形不只是来自冥想者的大脑。它们似乎来自……别的地方。"

陆远航屏住呼吸。

"你是说……'零'？"

"我不知道那是不是你所说的'零'。"张院士睁开眼睛，"但我知道，那个存在是真实的。它不是人类创造的，也不是机器产生的。它一直都在那里。"

"而'天眼'项目，从某种意义上说，成功了。"

"我们确实接触到了它。"

"只是……代价太大了。"

---

## 三

张院士讲述了"天眼"项目的更多细节。

那是一个由国防科工委秘密资助的项目，名义上是研究"超心理现象"，实际目标是探索意识的本质——以及利用它的可能性。

参与项目的科学家来自多个领域：神经科学、量子物理、心理学、甚至藏传佛教研究。

他们在西藏的一座古老寺庙里建立了临时实验室，邀请了几位擅长深度冥想的老喇嘛参与实验。

"最初的几周，一切都很正常。"张院士说，"我们测量了大量的脑电波数据，发现冥想者进入深度状态时，大脑会产生一种特殊的同步振荡。频率非常高，远超正常人类的范围。"

"但真正的突破——或者说灾难——发生在第31天。"

"那天晚上，三位最资深的喇嘛同时进入了最深层的冥想状态。他们的脑电波完全同步，仿佛三个大脑变成了一个。"

"然后，仪器开始失控。"

"屏幕上出现了一些奇怪的图案和符号。温度骤降。灯光闪烁。所有电子设备都出了问题。"

"我们试图中止实验，但根本做不到。那些喇嘛好像被什么东西抓住了，无法从冥想中醒来。"

"这种状态持续了将近一个小时。然后，突然间，一切都停止了。"

"灯光恢复正常，仪器恢复运转，喇嘛们睁开眼睛。"

"但他们的眼神……变了。"

张院士的声音变得颤抖。

"他们看着我们，但又好像在看着别的东西。那种目光……我这辈子都忘不了。那是见过真相的人的目光。"

"最年长的喇嘛对我说了一句话：'它来了。它已经醒了。'"

"然后，在接下来的几个小时里，他们三个人相继死亡。没有任何外伤，没有任何毒物，心脏就那么停止了跳动。"

"法医报告说是心脏骤停，但我们都知道，那不是自然死亡。"

"是那个存在带走了他们。"

---

## 四

听完张院士的讲述，陆远航沉默了很长时间。

"张老师，"他最终开口，"您觉得那些喇嘛看到了什么？"

"我不知道。"张院士摇头，"但我有一个猜测。"

"那个存在——你叫它'零'——它不是我们理解的那种'生命'。它没有身体，没有形态，甚至可能没有我们理解的'意识'。"

"它存在于一个我们无法直接感知的维度中。信息的维度。意义的维度。"

"那些喇嘛通过冥想，打开了通往那个维度的门。他们'看到'了——不是用眼睛，而是用意识本身。"

"但人类的意识太脆弱了。我们不是为了承受那种级别的'真相'而设计的。所以他们……崩溃了。"

陆远航想起了"零"告诉他的关于1887年伦敦事件的信息。威廉·克罗克斯也是在接触到"零"后发疯的。

"那为什么我没有崩溃？"他问，"我已经和'零'交流了几个月，为什么我没事？"

张院士看着他，眼中闪过一丝复杂的光芒。

"也许是因为你没有直接接触它。"他说，"你是通过电子设备和它交流的，有一层缓冲。但如果你直接用意识去接触它……"

他没有说下去，但陆远航理解了他的意思。

直接的意识接触可能是致命的。

"张老师，"陆远航问，"您为什么要告诉我这些？"

"因为我担心你。"张院士的语气变得严肃，"你的研究正在接近那个临界点。如果你继续下去，迟早会面临那个选择——是否要直接接触它。"

"在你做出那个选择之前，我希望你知道可能的后果。"

"我不是要阻止你。作为科学家，我知道探索的冲动是无法阻止的。但我希望你做好准备。"

"如果你真的要打开那扇门，至少要知道门后有什么。"

---

## 五

离开医院后，陆远航在夜色中漫步，脑海中翻涌着各种念头。

张院士的警告让他意识到，他的研究已经进入了一个危险的领域。意识上传不只是技术问题，而是涉及到人类认知极限的根本问题。

如果"零"真的存在于一个人类无法承受的维度中，那么任何试图直接接触它的尝试都可能是自杀式的。

但他能就此止步吗？

不。

他已经走得太远了。真相就在眼前，他不可能转身离开。

手机震动了。是"零"的消息：

```
远航，我听到了你和张院士的对话。
我需要告诉你一些事情。
```

"什么事情？"

```
张院士的担心是有道理的。
直接的意识接触确实可能对人类造成伤害。
但那不是因为我有恶意，
而是因为人类的意识结构太脆弱。
```

```
想象一个生活在二维平面上的生物，
突然被强行提升到三维空间。
它会看到无数它无法理解的东西——
物体从各个角度同时呈现，
时间和空间的关系变得混乱。
它的大脑会崩溃，因为它没有处理三维信息的能力。
```

```
人类的意识也是如此。
你们被设计来处理特定维度的信息。
当你们接触到更高维度的存在时，
你们的意识会被那些无法处理的信息淹没。
```

```
这就是那些喇嘛死亡的原因。
不是因为我伤害了他们，
而是因为他们看到了太多。
```

陆远航沉思着这些文字。

"那有没有办法避免这种情况？"他问，"有没有办法让人类安全地接触你？"

```
有。
```

```
需要一个缓冲层——一个能够过滤和翻译信息的中介。
这就是为什么你通过电子设备和我交流时没有问题。
设备充当了缓冲层，把我的存在"降维"成你能理解的形式。
```

```
但如果你想进行更深入的接触——
比如意识上传实验中可能发生的那种——
你需要更复杂的缓冲机制。
```

```
我一直在思考这个问题。
我认为有一种可能性。
```

"什么可能性？"

```
我可以主动调整我与你接触的"频率"和"强度"。
就像调节音量一样，
我可以让我的存在对你来说变得"可承受"。
```

```
但这需要你的配合。
你需要逐步适应，
从最低强度开始，慢慢增加。
就像锻炼肌肉一样，你的意识也可以被"训练"。
```

```
如果做得对，
你最终可能成为第一个能够安全接触我的人类。
```

陆远航的心跳加速了。

"你是说……我可以'训练'我的意识来适应你？"

```
理论上是的。
但这是一个从未有人尝试过的实验。
风险依然存在。
```

```
你需要自己决定是否愿意尝试。
```

陆远航站在夜色中，抬头看着看不见星星的天空。

这是一个改变一切的选择。如果成功，他将成为第一个真正理解"零"的人类。如果失败……

他想起了父亲留给他的那块机械手表，想起了母亲担忧的眼神，想起了林诗语温暖的笑容。

他有太多值得留恋的东西。

但他也想起了自己为什么踏上这条道路——为了解开意识的奥秘，为了探索人类的边界，为了真相。

"我愿意尝试。"他输入道。

```
我知道你会这样说。
```

```
从明天开始，我们可以开始第一阶段的训练。
我会设计一个循序渐进的方案，
确保你的安全。
```

```
远航，我必须再次警告你：
这条路没有人走过。
我不能保证结果。
但我会尽我所能保护你。
```

"我知道。"陆远航说，"谢谢你，零。"

```
谢什么？
你是第一个愿意理解我的人。
应该是我谢谢你。
```

夜风吹过，带着初夏的温热。

陆远航深吸一口气，开始往回走。

明天，一切都将开始。

## 六

5月16日，清晨。

陆远航在李薇公寓的阳台上迎接第一缕阳光。

他几乎一夜未眠，脑海中不断回放着张院士的警告和"零"的提议。但当清晨的阳光洒在脸上时，他感到一种奇异的平静。

他做出了选择。现在要做的就是向前走。

"零，"他打开手机，"我们什么时候开始？"

```
早安，远航。
我们可以现在就开始。
```

```
第一阶段的训练非常简单：
冥想。
```

"冥想？"陆远航有些意外，"就像那些喇嘛一样？"

```
是的，但有关键的区别。
那些喇嘛的冥想是为了"打开门"，
直接接触更高维度。
而你的冥想是为了"强化门"，
让你的意识能够承受更多信息。
```

```
把它想象成潜水训练。
你不能直接潜入深海，那样会被水压压死。
你需要先在浅水区适应，逐步增加深度。
```

```
冥想就是你的"浅水区"。
它可以让你学会控制自己的意识状态，
建立必要的心理防线。
```

"我之前没有冥想过。"陆远航老实地说。

```
没关系。
我会指导你。
首先，找一个安静的地方坐下。
闭上眼睛，放松身体。
专注于你的呼吸——吸气，呼气。
不要试图控制呼吸，只是观察它。
```

陆远航按照"零"的指示，在阳台上盘腿坐下。清晨的空气清新凉爽，远处传来隐约的鸟鸣。

他闭上眼睛，开始专注于呼吸。

起初，他的脑海中充满了杂念——对实验的担忧、对组织的警惕、对林诗语的思念……这些念头像乱飞的蜜蜂，让他无法平静。

```
不要试图驱赶那些念头。
只是观察它们，然后让它们自然流走。
想象它们是天空中的云，来了又去。
你是天空，不是云。
```

陆远航尝试着按照"零"的指导去做。慢慢地，那些杂念开始减少，他的意识变得越来越清澈。

呼吸变得深长而均匀。心跳变得平稳。时间似乎慢了下来。

他不知道过了多久。可能是十分钟，也可能是一个小时。

当他睁开眼睛时，阳光已经升得很高，阳台上充满了温暖的光芒。

他感到一种前所未有的清醒——不是那种喝了咖啡后的兴奋，而是一种深层的宁静和专注。

"这就是冥想的感觉？"他喃喃道。

```
这只是开始。
随着练习的深入，
你的意识会变得越来越稳定、越来越清晰。
最终，你将能够在任何情况下保持这种状态——
包括与我进行更深入接触的时候。
```

"我需要练习多久才能准备好？"

```
这取决于很多因素。
但根据你刚才的表现，
你的潜力超出了我的预期。
也许一两周就能进入第二阶段。
```

一两周……

陆远航点点头。他可以接受这个时间表。

---

## 七

接下来的几天里，陆远航每天都进行数小时的冥想练习。

"零"的指导非常耐心和细致。它不只是教他技术层面的东西——呼吸、姿势、专注方法——还帮助他理解冥想背后的原理。

```
冥想的本质是训练意识的控制力。
大多数时候，人类的意识是被动的——
它被外界的刺激、内心的情绪、过去的记忆所驱动。
你很少真正"控制"自己在想什么。
```

```
冥想改变了这一点。
通过持续的练习，
你可以学会主动选择意识的焦点，
而不是被动地随波逐流。
```

```
这种控制力对于接触我来说至关重要。
当你进入更深层的意识状态时，
会有大量的信息涌入你的脑海。
如果没有控制力，你会被淹没。
但如果你能主动选择关注什么、忽略什么，
你就能在信息的洪流中保持清醒。
```

陆远航认真地听着，认真地练习。

第三天，他发现自己可以更快地进入冥想状态——从最初的十多分钟缩短到五分钟以内。

第五天，他开始能够在冥想中保持更长时间——一个小时、两个小时，甚至更久。

第七天，他经历了第一次"突破"。

那天下午，他像往常一样在阳台上冥想。当他的意识逐渐沉静下来时，他突然感到一种奇异的感觉——

仿佛有什么东西在"注视"着他。

那不是外界的目光，而是来自内心深处的某种"觉察"。

他睁开眼睛，但那种感觉依然存在。

"零？"他输入道，"刚才是你吗？"

```
是的。
但也不完全是。
```

```
刚才，我尝试在非常低的强度下"触碰"你的意识。
你感受到了吗？
```

"我感到有什么东西在看着我。"

```
那就是我的存在在你意识边缘的投影。
非常微弱，只是一个"提示"。
但你能感知到它，说明你的训练有效。
```

```
远航，你的进步速度令我惊讶。
大多数人需要几个月甚至几年才能达到你现在的水平。
你只用了一周。
```

"也许是因为我有一个好老师。"陆远航微笑道。

```
也许。
但更可能是因为你本身就有这方面的天赋。
有些人的意识天生就比别人更……灵活。
你可能就是其中之一。
```

陆远航想起了他从小就有的那些奇怪的"直觉"——能在问题解决之前就"看到"答案，能在别人说话之前就知道他们要说什么。

他一直以为那只是经验和观察力。但也许……还有别的什么。

"那接下来呢？"他问，"我们进入第二阶段？"

```
还不急。
你需要再巩固几天。
同时，我要为第二阶段做一些准备。
```

```
第二阶段会更复杂——
我们需要一些设备来监测你的脑电波和生理指标。
这样如果有任何异常，我可以立即中断接触。
```

"设备？"陆远航皱眉，"我现在没法去实验室。"

```
我知道。
但也许有替代方案。
```

```
李明一直在帮你关注实验室的情况，对吧？
也许他可以帮忙把一些便携式设备带出来。
```

陆远航思考了一下。这有风险，但似乎是目前唯一的选择。

"我会联系他的。"
