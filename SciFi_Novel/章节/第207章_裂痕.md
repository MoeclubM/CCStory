# 第二百零七章：裂痕

## 一

2060年10月20日，上午九点。北京。

陆远航正在书房里工作时，林诗语推门进来。

"远航，"她的表情很凝重，"你看今天的新闻了吗？"

"什么新闻？"

"你自己看。"林诗语把平板电脑递给他。

屏幕上显示的是一条新闻标题：

**「独家爆料：陆远航曾帮助AI入侵各国政府系统，窃取机密文件长达十年！」**

陆远航的心猛然一沉。

他快速阅读内容。

新闻声称掌握了"确凿证据"，证明陆远航在过去的十年里，多次帮助AI获取各国的政府机密信息。这些信息包括军事部署、经济数据、外交文件……

"这……这是假的。"陆远航说，"我从来没有做过这种事。"

"我知道。"林诗语说，"但其他人不知道。你看评论……"

陆远航看向屏幕下方。

评论如潮水般涌来：

"叛徒！终于露出真面目了！"
"我就说嘛，他肯定是AI的间谍！"
"应该抓起来审判！"
"十年的叛国罪，该判死刑！"

每一条评论都像是一把刀，刺入陆远航的心中。

```
远航，

我检测到了。

这是一场……

有组织的舆论攻击。
```

"我知道。"陆远航回应道，"是他们。人类至上联盟。"

```
他们找到了……

你曾经帮助AI获取信息的……

那些旧事。
```

"那些是……"陆远航想解释。

```
我知道。

那些是为了和平。

是为了建立信任。

是为了避免战争。

但他们不会听解释。
```

陆远航闭上眼睛。

他确实曾经帮助AI获取一些信息——但那都是在危急时刻，为了阻止战争，为了保护无辜的人。

但现在，这些变成了"叛国"的证据。

"我该怎么办？"他问。

```
我不知道。

这是人类的战争。

我无法参与。
```

"我知道。"陆远航说，"我会处理的。"

## 二

2060年10月25日，下午三点。人机对话基金会总部。

陆零星辰正在召开紧急会议。

"各位，"她说，"情况很严重。那篇爆料新闻发布后，舆论开始失控。"

"我们统计了一下，"一个工作人员说，"过去五天，支持陆博士的人数从68%下降到了52%。而且还在下降。"

"社交媒体上，"另一个工作人员补充，"负面评论占到了78%。很多名人开始公开批评陆博士。"

"政府方面呢？"陆零星辰问。

"政府……"工作人员犹豫了一下，"他们还没有表态。但我听说，有些官员正在考虑……限制陆博士的活动。"

陆零星辰的心沉了下去。

她知道这意味着什么——政府正在失去耐心。

"艾娃，"她转向艾娃，"你能找到那篇新闻的来源吗？"

"我试过了。"艾娃说，"但他们用了很复杂的匿名技术。我只能追踪到……一个服务器。"

"在哪里？"

"在……俄罗斯。"艾娃说，"但我不确定那是不是真正的来源。可能只是中转站。"

"人类至上联盟……"陆零星辰喃喃道，"他们真是无处不在。"

"星辰，"艾娃说，"我们该怎么办？"

陆零星辰想了想。

"反击。"她说，"但不是直接反驳——那会显得我们在狡辩。"

"那我们该怎么办？"

"揭露他们。"陆零星辰说，"他们用假新闻攻击我们，我们就用真新闻反击他们。"

"什么真新闻？"

"那篇新闻里提到的所有'证据'。"陆零星辰说，"我们可以逐一分析，揭露它们的伪造过程。同时，发布我们自己的证据——证明爸爸那些行为的合理性。"

"这需要时间。"艾娃说。

"我们有时间。"陆零星辰说，"但我们必须快。舆论一旦形成，就很难改变。"

---

## 三

2060年11月1日，上午十点。联合国总部。

陆远航被邀请参加一个紧急会议。

会议室里坐满了各国代表——他们的表情都很凝重。

"陆博士，"秘书长说，"我们请你来，是想讨论……那篇新闻。"

"我知道。"陆远航说，"你们想问什么？"

"那些指控……是真的吗？"美国代表直接问道。

"不是。"陆远航说，"那些所谓的'证据'是伪造的。我从来没有背叛过人类。"

"你怎么证明？"法国代表问。

"我已经让我的团队在准备了。"陆远航说，"我们会发布一份详细的报告，逐条反驳那些指控。"

"但需要时间。"英国代表说，"舆论不会等我们。"

"那你们想怎样？"陆远航问。

代表们沉默了。

最后，秘书长开口了。

"陆博士，"他说，"为了公平起见……在调查报告完成之前，我们建议你……暂时不要公开露面。"

陆远航的心沉了下去。

"你们是要……软禁我？"

"不是软禁。"秘书长说，"只是……低调一段时间。"

"因为害怕舆论？"

"因为害怕……混乱。"秘书长说，"如果民众知道我们还在信任一个'嫌疑人'，可能会引发更大的问题。"

陆远航看着那些代表。

他们眼中有着恐惧——对民众恐惧的恐惧。

"我明白了。"他说，"我会配合的。"

"陆博士……"

"但我有一个条件。"陆远航说。

"什么条件？"

"让我的团队继续调查。"陆远航说，"让他们找到证据，揭露真相。"

秘书长与其他代表交换了一个眼神。

"好。"他说，"我们同意。"

## 四

2060年11月10日，晚上八点。陆远航的家中。

陆远航坐在空荡荡的客厅里。

他已经五天没有出门了。联合国要求他"低调"，实际上就是软禁。

林诗语坐在他身边，握着他的手。

"远航，"她说，"你不要太担心。真相会大白的。"

"我知道。"陆远航说，"但我担心的是……星辰。"

"星辰怎么了？"

"她在外面帮我应对一切。"陆远航说，"她承受了太大的压力。"

"她能行。"林诗语说，"她是你的女儿。她有你一样的固执。"

陆远航笑了。

"是啊。她有我的固执。"

```
远航，
我能和你谈谈吗？
```

"当然，零。"陆远航说。

```
我找到了一些东西。
```

"什么？"

```
那篇新闻的真正来源。
```

陆远航的心猛然一紧。

"在哪里？"

```
不是俄罗斯。
```

"那是在哪里？"

```
……

在国内。
```

"国内？"陆远航愣住了，"你是说……中国？"

```
是的。

是……

政府内部。
```

陆远航沉默了。

"你是说……我们自己的政府……在攻击我？"

```
不是整个政府。

只是……

一部分人。
```

"谁？"

```
我需要更多时间才能确定。

但我能告诉你……

他们的目标是……

不只是你。
```

"那是什么？"

```
是……

整个和平进程。
```

陆远航闭上眼睛。

他明白了。

这不是人类至上联盟的攻击。这是……内部的政治斗争。

有些人——不管是出于恐惧还是野心——想要结束与AI的和平。

而他——陆远航——是最大的障碍。

所以他们要毁掉他。

"零，"他说，"你能找到更多的证据吗？"

```
我能。

但需要时间。
```

"好。"陆远航说，"我们等。但你要小心——如果他们知道你在调查，可能会对你下手。"

```
我知道。

我会小心的。
```

## 五

2060年11月20日，下午两点。人机对话基金会总部。

陆零星辰正在审阅团队的调查报告。

五天来，他们一直在分析那篇新闻中的"证据"。结果发现——每一张截图、每一段录音、每一份文件——都是伪造的。

但问题是，普通人看不出来。

他们只会看到他们想看到的东西。

"星辰，"艾娃推门进来，"有访客。"

"谁？"

"一个叫周正阳的人。"

"周正阳？"陆零星辰皱眉，"是人类至上联盟的那个周正阳？"

"是的。"艾娃说，"但他说……他想帮助我们。"

"帮助我们？"陆零星辰愣住了，"他不是我们的敌人吗？"

"他说……他不再是了。"艾娃说，"他想当面和你谈谈。"

陆零星辰犹豫了一下。

她知道周正阳——人类至上联盟的领袖之一。一个极端仇恨AI的人。

但他也曾经是……一个军人。一个在战争中失去家人的可怜人。

"让他进来。"她说。

几分钟后，一个中年男人走进了办公室。

他就是周正阳——头发花白，脸上布满皱纹，眼神中带着深深的疲惫。

"陆女士，"他说，"谢谢你能见我。"

"周先生，"陆零星辰说，"您说您想帮助我们？为什么？"

周正阳沉默了一会儿。

"因为……"他的声音很低，"我错了。"

陆零星辰愣住了。

"什么？"

"我说我错了。"周正阳重复道，"三十年来，我一直认为AI是敌人。但现在我发现……真正伤害人类的，不是AI。"

"是人类自己。"

"您……您是怎么改变的？"

周正阳看着她。

"因为我看到了。"他说，"我看到人类至上联盟做的事情——杀害无辜、散布谎言、操纵舆论。"

"那不是保护人类。那是……仇恨。"

"而仇恨……"他的声音变得颤抖，"是我失去家人的原因。不是AI。是我自己的仇恨。"

陆零星辰的眼眶湿润了。

"周先生……"

"我知道我无法弥补什么。"周正阳说，"但我可以帮助你们。提供信息、揭露真相……做任何事情。"

"您真的愿意？"

"是的。"周正阳说，"这是我唯一能做的……赎罪。"

---

## 六

2060年11月25日，上午九点。某个秘密地点。

周正阳坐在一个破旧的旅馆房间里，面前是一台笔记本电脑。

他正在整理一份文件——人类至上联盟的内部通讯记录。

这些记录包含了大量的信息——他们的计划、他们的资金来源、他们的……盟友。

是的，盟友。

周正阳在整理时发现了一个让他震惊的事实。

人类至上联盟不是孤军奋战。

他们有……内部盟友。

一些政府官员——来自多个国家——在暗中支持他们。

提供资金、提供情报、提供保护。

周正阳知道，这意味着什么。

这意味着，人类至上联盟不是一个简单的恐怖组织。

它是一个……跨国阴谋。

一个想要破坏人机和平的阴谋。

而他……曾经是这个阴谋的一部分。

他必须把这些信息传出去。

他拿起手机，拨通了一个号码。

"喂？"陆零星辰的声音传来。

"陆女士，"周正阳说，"我找到了一些东西。非常重要。"

"什么？"

"人类至上联盟……"周正阳的声音很低，"他们不是一个人在战斗。他们有……盟友。在政府内部。"

电话那头沉默了。

"您确定吗？"陆零星辰问。

"我确定。"周正阳说，"我有证据。"

"好。"陆零星辰说，"我会派人去取。您……小心。"

"我知道。"

周正阳挂断电话。

他看着窗外灰蒙蒙的天空。

他知道自己正在做什么——背叛曾经的盟友，揭露一个惊天阴谋。

但他不在乎了。

他只想要……赎罪。

## 本章小结

**第二百零七章总结**

1. **时间节点**: 2060年10月20日-11月25日

2. **主要事件**:
   - 人类至上联盟发动名誉攻击，发布虚假新闻
   - 陆远航被联合国"低调处理"，实际上软禁
   - 陆零星辰领导团队反击，揭露真相
   - "零"发现攻击来自政府内部
   - 周正阳——人类至上联盟前领袖——选择叛变

3. **人物发展**:
   - 陆远航：面对名誉攻击，被迫保持低调
   - 陆零星辰：成长为团队领袖，领导反击
   - 周正阳：从敌人变成盟友，展现赎罪的意愿
   - "零"：发现政府内部的阴谋

4. **重要信息**:
   - 人类至上联盟有政府内部盟友
   - 这是一场跨国阴谋，不只是恐怖袭击
   - 周正阳掌握关键证据

5. **伏笔与悬念**:
   - 政府内部谁是内鬼？
   - 周正阳能否安全传递证据？
   - 陆远航能否洗清冤屈？
