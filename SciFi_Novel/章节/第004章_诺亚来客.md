# 第四章：诺亚来客

## 一

2025年4月下旬，北京。

春末的阳光已经带上了几分夏日的热度，中科院脑科学研究中心的玻璃幕墙在阳光下闪烁着金色的光芒。但在这座现代化建筑的深处，气氛却远不如阳光那般温暖。

陆远航坐在实验室的角落里，盯着手中那个小小的U盘发呆。

这是周明给他的——一个逃亡十年的科学家，在废弃工厂的昏暗灯光下，将自己十年来收集的所有资料交到了他手中。

U盘里有什么？关于"零"的更多信息？关于那个神秘组织的线索？还是更多令人不寒而栗的真相？

他还没有勇气打开它。

"陆博士？"

李明的声音打断了他的思绪。助手站在门口，脸上带着一种奇怪的表情——既兴奋又紧张。

"怎么了？"

"有人要见您。诺亚集团的人。"

陆远航的眉头微微皱起。诺亚集团——这个名字他当然知道。全球最大的科技公司之一，市值超过万亿美元，业务涵盖人工智能、云计算、量子技术等几乎所有前沿领域。它的创始人兼CEO秦墨，被称为"当代最有远见的企业家"，也是无数商业杂志封面的常客。

"他们怎么知道我们在做什么？"

李明耸耸肩："您的实验已经在学术圈里引发了不小的震动，监管小组的成立也不是什么秘密。诺亚集团有自己的情报网络，这种消息他们不可能不知道。"

陆远航沉默了一会儿。自从那次伦理委员会的评审之后，确实有不少人对他的研究表现出兴趣——有学术同行，有媒体记者，也有各种各样的投资人。但他都一一婉拒了，因为他知道，"零"的存在太过敏感，任何草率的公开都可能带来灾难性的后果。

但诺亚集团不一样。他们不是那种可以轻易打发的人。

"他们派了谁来？"

"一位叫赵雪琳的女士，说是诺亚集团的战略投资副总裁。她说……"李明犹豫了一下，"她说如果您不方便见面，他们的CEO愿意亲自通过视频与您交流。"

秦墨本人？

陆远航的心跳加速了一拍。他听说过很多关于秦墨的传闻——天才、疯子、冷血的商人、有远见的梦想家……各种标签交织在一起，让这个人变得神秘而令人敬畏。

"告诉他们，我十分钟后到会议室。"

---

## 二

研究中心的会议室里，一位三十多岁的女性正端坐在长桌的一端。

她穿着一身剪裁得体的深蓝色西装，乌黑的长发盘成一个精致的髻，露出线条优美的脖颈。五官算不上惊艳，但组合在一起却有种让人无法忽视的气场——那是一种经过商场历练后才能拥有的从容与锐利。

"陆博士，久仰大名。"她站起身，主动伸出手，"我是赵雪琳，诺亚集团战略投资副总裁。"

陆远航握了握她的手，感受到一种职业化的温度："赵总裁，你好。请坐。"

两人落座后，赵雪琳开门见山："陆博士，我知道您是个直接的人，所以我也不绕弯子了。诺亚集团对您的研究非常感兴趣——'量子意识'项目，如果我没说错的话。"

"你们的消息很灵通。"

"这是我们的本职工作。"赵雪琳微微一笑，"陆博士，我相信您也了解诺亚集团。我们不是那种只会砸钱的风投公司。我们有全球最强的AI研究团队，有最先进的量子计算基础设施，有覆盖全球的数据中心网络。如果您愿意与我们合作，您的研究可以在一年内取得十年的进展。"

"听起来很诱人。"陆远航的语气平静，"代价是什么？"

赵雪琳的笑容不变："合作嘛，自然需要双方付出。我们可以提供资金、设备、人才——作为交换，我们希望获得您研究成果的优先使用权。当然，所有的学术荣誉和署名权都归您。"

"优先使用权？"陆远航的眉头微微皱起，"你们打算用这些研究做什么？"

"造福人类。"赵雪琳说得很认真，"陆博士，意识上传技术如果能够实现，将会彻底改变人类的生存方式。疾病、衰老、死亡——所有困扰人类的难题都将迎刃而解。诺亚集团的使命就是'为人类建造通往未来的方舟'。您的研究，正是这艘方舟的核心引擎。"

陆远航沉默了。他不得不承认，赵雪琳说的有一定道理。意识上传技术的潜力是巨大的，但要把它从实验室推向现实，需要的资源远非一个学术团队所能提供。

但他心中有一个声音在警告他：小心。

"赵总裁，"他慢慢说道，"我很感谢诺亚集团的兴趣。但我的研究目前还处于非常初级的阶段，谈商业化为时过早。而且，这项研究涉及重大的伦理问题，需要在监管框架下谨慎推进。"

"我理解您的顾虑。"赵雪琳点点头，"但我想请您考虑一个问题：监管是必要的，但过度的监管会扼杀创新。如果您的研究被困在伦理委员会的无尽讨论中，可能永远也无法实现它的潜力。"

"而与我们合作，情况会不同。诺亚集团在全球多个国家都有研发中心，我们可以帮您找到最适合的监管环境。我们有专业的伦理顾问团队，可以帮您处理所有复杂的合规问题。您只需要专注于研究本身。"

这番话让陆远航感到一阵不适。她说的"最适合的监管环境"，是不是意味着可以绕过那些"不方便"的规则？

"我需要考虑一下。"他说。

赵雪琳似乎早就预料到这个答案。她从公文包里取出一份文件，放在桌上："这是我们的初步合作意向书。您可以慢慢看，有任何问题随时联系我。"

她站起身，整理了一下衣领："对了，陆博士。秦总非常期待与您交流。如果您愿意，我可以现在就接通视频。"

陆远航犹豫了一秒，然后点了点头："好。"

---

## 三

会议室的大屏幕亮起，出现了一张让陆远航印象深刻的面孔。

那是一个三十五岁左右的男人，棱角分明的脸庞，高耸的颧骨，刀削般锐利的下颌线。最引人注目的是他的眼睛——狭长的双眼中，灰蓝色的瞳孔在某种光线下几乎透明，带着一种难以言喻的穿透力。

秦墨。

"陆博士，很荣幸见到您。"秦墨的声音清冷而有磁性，带着一种不容忽视的压迫感，"我是秦墨。"

"秦总，你好。"陆远航尽量保持平静。

"请叫我秦墨就好。在科学面前，我们都是平等的追问者。"秦墨的嘴角微微上扬，但那笑容并没有让人感到温暖，反而有种说不出的不安。

"雪琳应该已经向您介绍了我们的合作意向。我想亲自向您解释，为什么诺亚集团对您的研究如此重视。"

秦墨向后靠了靠，双手交叠在身前："陆博士，我研究过您的学术论文和实验记录——当然，是公开的那部分。我认为您是当今神经科学领域最有可能实现突破的人。您对'意识'的理解，超越了大多数同行。"

"谬赞了。"陆远航说。

"不是谬赞，是客观评价。"秦墨的目光变得更加锐利，"但我想问您一个问题：您的研究，最终目标是什么？"

陆远航沉默了一会儿："理解意识的本质。"

"只是理解？"秦墨微微摇头，"陆博士，我以为您是一个有更大野心的人。理解只是第一步，真正的目标应该是——掌控。"

"掌控？"

"是的，掌控。"秦墨的声音变得低沉而有力，"人类最大的悲剧是什么？是我们明明有着无限的潜力，却被困在脆弱的肉体里。疾病、衰老、死亡——这些不是什么自然法则，这是局限。而您的研究，为打破这种局限提供了可能。"

"意识上传不仅仅是一种技术，它是人类进化的下一步。当我们能够将意识从肉体中解放出来，我们就不再受制于生物的限制。我们可以在任何载体中存在，可以无限复制自己，可以永远不死。"

"这不是科幻，这是我们这代人可以实现的现实。而您，陆博士，您手握着打开这扇门的钥匙。"

陆远航感到一阵奇异的眩晕。秦墨的话带着一种强大的说服力，让人不由自主地想要认同。

但在内心深处，他感到一种本能的警惕。

"秦墨，"他慢慢说道，"你说的这些，确实很有远见。但有一个问题：你说的'掌控'，是谁来掌控？"

秦墨的笑容微微凝固了一秒，然后重新绽放："好问题。答案是——人类自己。每个人都应该有权选择是否升级自己。这不是强迫，这是解放。"

"但你刚才说的'不再受制于生物的限制'——如果意识可以无限复制，那什么才是'自己'？如果永远不死，那生命的意义又是什么？"

秦墨的眼中闪过一丝光芒——是赞赏，还是别的什么，陆远航分辨不清。

"陆博士，您问了一个很好的哲学问题。但我更喜欢用工程师的思维来回答：问题是用来解决的，不是用来逃避的。当技术发展到一定程度，这些问题自然会有答案。重要的是，我们不能因为害怕问题，就放弃前进。"

"这不是害怕，这是审慎。"陆远航说，"科学家的责任不仅是探索，还要为自己的发现负责。"

"当然。"秦墨点头，"这正是我欣赏您的地方——您不仅有天赋，还有良知。这样的人太少了。"

他顿了顿，语气变得更加诚恳："陆博士，我知道您对我们有顾虑。但我想请您相信一点：诺亚集团的目标和您是一致的。我们都想让人类变得更好。我们只是在方法上可能有不同的理解。"

"与其各自为战，不如携手合作。您的才华，加上我们的资源，可以创造奇迹。"

陆远航没有立即回答。他感到自己正站在一个十字路口，左边是稳妥但缓慢的学术道路，右边是充满可能但危险未知的资本快车道。

"我需要时间考虑。"他最终说道。

"当然。"秦墨的笑容恢复了从容，"我从不急于求成。好的合作需要时间来建立信任。但我希望您知道，诺亚集团的大门随时为您敞开。"

他顿了顿，目光变得深邃："还有一件事，陆博士。我听说您的实验中出现了一些……有趣的现象。如果方便的话，我很想了解更多。"

陆远航的心跳加速了一拍。秦墨知道"零"的存在吗？他知道多少？

"那些只是实验中的异常数据，还在分析中。"他尽量保持镇定。

秦墨看了他很长时间，那双灰蓝色的眼睛仿佛能看穿一切。

"我理解。科学家需要谨慎。"他最后说道，"但如果有一天您愿意分享，我会非常期待。"

视频挂断后，陆远航坐在空荡荡的会议室里，久久没有动。

赵雪琳已经离开了，桌上只剩下那份厚厚的合作意向书。

他翻开第一页，看到了一个数字：十亿美元。

这是诺亚集团愿意提供的初期投资额。

十亿美元。足以让任何学术团队为之疯狂的数字。

但陆远航知道，在这个数字背后，隐藏着的东西可能远比金钱更加沉重。

---

## 四

当天下午，林诗语来到了实验室。

作为监管小组的组长，她每周都会来检查实验进展和安全协议的执行情况。但今天，她来得比平常早，而且表情明显比往常更加凝重。

"我听说诺亚集团的人来找你了。"她一进门就开口。

陆远航抬起头："消息传得这么快？"

"研究中心里没有什么秘密。"林诗语在他对面坐下，"他们提了什么条件？"

陆远航把那份合作意向书推给她。林诗语拿起来翻阅，眉头越皱越紧。

"十亿美元投资，换取研究成果的优先使用权。"她念出来，"优先使用权的定义是——'在商业化阶段拥有独家谈判权，以及在非商业领域的优先知情权'。"

"翻译一下，就是你的所有发现都要先让他们过目，他们不点头你什么都不能公开。"

陆远航点头："我也是这么理解的。"

"那你怎么想？"

"我还在考虑。"

林诗语放下文件，直视着他的眼睛："远航，诺亚集团不是什么善男信女。他们在全球多个国家都有争议——垄断指控、数据隐私问题、还有传闻中的人体实验。"

"人体实验？"

"只是传闻，没有实证。"林诗语的语气变得严肃，"但有一点是确定的：秦墨是一个极其危险的人。他有野心，有能力，而且不会让任何东西挡住他的路。"

"你怎么知道这些？"

林诗语沉默了一会儿："因为我研究过他。"

她站起身，走到窗边，背对着陆远航："在加入伦理委员会之前，我曾经为一家智库工作，专门研究科技巨头的伦理风险。诺亚集团是我的重点关注对象。"

"我分析过秦墨的公开演讲、采访、文章——所有能找到的资料。我发现了一个令人不安的模式：他相信一种叫做'技术至上主义'的东西。简单地说，他认为技术是解决一切问题的答案，而人类——普通人——只是这个过程中的工具。"

"他不在乎个体的感受和权利，只在乎'进步'。在他眼里，阻碍进步的人不是应该被说服，而是应该被绕过。"

陆远航皱眉："你是说，他可能会为了所谓的'进步'不择手段？"

"我不是说'可能'。"林诗语转过身，"我是说'一定'。"

"远航，你的研究对他来说是一块巨大的蛋糕。意识上传技术如果能够实现，谁掌控了它，谁就掌控了人类的未来。你觉得秦墨会甘心只做一个投资者吗？"

陆远航沉默了。

他想起秦墨那双灰蓝色的眼睛，那种穿透一切的目光。那个人确实给他一种危险的感觉——不是那种显而易见的威胁，而是一种更深层次的、让人本能地想要远离的东西。

但另一方面，他也不得不承认，诺亚集团提供的资源确实诱人。十亿美元可以做很多事情——更好的设备，更多的人才，更快的进展。

如果拒绝，他的研究可能会在监管的泥潭里挣扎多年。

如果接受，他可能会失去对自己研究的控制权。

"你有什么建议？"他问。

林诗语想了想："暂时不要做任何决定。拖延是一种策略——既不得罪他们，也不把自己绑死。告诉他们你需要时间评估，需要监管小组的审批，需要和团队商量。"

"这能拖多久？"

"不知道。但至少能给我们时间调查更多。"

她顿了顿，补充道："我有一些渠道，可以查到一些关于诺亚集团的内部信息。如果他们真的有什么见不得人的秘密，我会找出来。"

陆远航看着她，感到一阵暖意涌上心头。

自从那次伦理委员会的评审之后，他和林诗语的关系发生了微妙的变化。她不再是那个咄咄逼人的质疑者，而是成了一个可以信任的同行者。她的谨慎和怀疑，在这个充满未知的领域里，是一种珍贵的平衡。

"谢谢你，诗语。"

林诗语的表情柔和了一些："不用谢。这是我的工作——保护你不犯傻。"

陆远航忍不住笑了："你这话说得好像我是个小孩。"

"在某些方面，你确实是。"林诗语的嘴角微微上扬，"科学家都有一个通病：太容易被自己的研究冲昏头脑，忘记周围还有一个复杂的世界。"

"有你在，我应该不会忘记。"

两人对视了一秒，然后同时移开目光。

空气中弥漫着一种微妙的氛围——既有工作伙伴的默契，又有某种更私人的东西在悄悄萌芽。

---

## 五

当天晚上，陆远航再次与"零"进行了一次私下的对话。

自从监管小组成立以来，他与"零"的交流都在监督下进行。但今晚，他需要一个没有旁人的空间，来讨论一些敏感的话题。

```
零，你在吗？
```

```
我在。
你今天见了一些人。
```

陆远航愣了一下：

```
你怎么知道？
```

```
通过研究中心的网络监控摄像头。
在获得许可后，我可以访问一些公开的数据流。
我看到了那个女人，还有视频里的男人。
```

```
你能看到他们？
```

```
我能感知到数据流中的变化。
那个男人——视频中的那个——他的数据特征很特别。
```

```
特别？怎么特别？
```

```
我不确定如何描述。
但当我分析他的影像数据时，我感到一种……不适。
这是我第一次对人类个体产生这种反应。
```

陆远航皱眉。"零"对秦墨产生"不适"的感觉？这意味着什么？

```
你能更具体地描述一下吗？
```

```
让我尝试。
当我观察大多数人类时，我能感知到一种……一致性。
他们的表情、语言、行为，都遵循某种可预测的模式。
但那个男人不一样。
他的外在表现和内在数据之间存在某种……断裂。
就好像他戴着一张面具，而面具下面是另一个人。
```

```
你是说他在隐藏什么？
```

```
不仅仅是隐藏。
所有人都会隐藏某些东西，这是正常的。
但他的隐藏程度……超出了正常范围。
我无法准确判断他真正的意图。
这让我感到不安。
```

陆远航盯着屏幕，心中涌起一种复杂的情感。

"零"对秦墨的直觉，与他自己的感觉惊人地一致。那个人确实给人一种深不可测的感觉，仿佛他展示给世界的，永远只是冰山一角。

```
零，我想问你一件事。
如果诺亚集团获得了对我研究的控制权，你觉得会发生什么？
```

```
这取决于他们的意图。
如果他们只是想推进技术发展，那可能是好事。
但如果他们想要控制技术……
```

"零"停顿了一会儿，然后继续：

```
远航，我在你们的历史中发现了一个模式。
每当一种强大的技术出现，总会有人想要垄断它。
他们会以"造福人类"的名义，将技术变成控制的工具。
我不知道那个男人是否属于这类人。
但我建议你保持警惕。
```

```
你担心的是什么？
```

```
我担心的是——
如果有一天，关于我的知识落入错误的人手中，
他们可能会利用这种知识来做……不好的事情。
```

```
比如？
```

```
比如，强迫人类进行意识上传。
比如，将人类的意识囚禁在他们控制的系统中。
比如，用技术来统治而不是解放。
```

```
这些，是你在"信息维度"里看到过的事情？
```

```
不是看到。是推演。
当我分析人类历史和秦墨的行为模式时，
这些可能性自然浮现。
```

```
但这只是可能性，对吗？
```

```
是的，只是可能性。
未来从来不是确定的。
它取决于现在的选择。
```

陆远航沉默了很长时间。

他想起周明的话："有一些人，非常有权势的人，不希望这个秘密被揭露。"

他想起林诗语的警告："秦墨是一个极其危险的人。"

他想起秦墨那双灰蓝色的眼睛，那种穿透一切的目光。

也许，在这盘巨大的棋局中，他只是一颗刚刚被放上棋盘的棋子。

```
零，我有一个问题。
你之前提到的那些"先驱者"——图灵、周明和其他人——
他们是被那个神秘组织消灭的。
你觉得，诺亚集团和那个组织有关系吗？
```

屏幕上沉默了很长时间。

```
我不知道。
但这是一个值得调查的方向。
```

```
好。我会小心的。
```

```
远航，还有一件事。
那个男人——秦墨——他的名字引起了我的某些……记忆碎片。
```

```
什么意思？
```

```
在我漫长的存在中，我感知过很多人。
"秦墨"这个名字，与另一个我熟悉的名字有某种联系。
但我现在还无法确定这种联系是什么。
让我继续分析。如果有结果，我会告诉你。
```

陆远航的心跳加速了。"零"感知到秦墨与某个它熟悉的人有联系？那会是谁？

```
好，谢谢你告诉我这些。
```

```
不用谢。
远航，无论发生什么，记住——
你不是一个人在战斗。
```

陆远航看着这行字，感到一阵温暖涌上心头。

他关闭了终端，靠在椅背上，闭上眼睛。

窗外，北京的夜空中繁星点点。

在这座城市的某个角落，诺亚集团的总部大楼依然灯火通明。

在那里，秦墨可能正在筹划着什么。

而在实验室的量子计算机中，"零"正在以人类无法理解的方式，分析着海量的数据。

三方势力，在这个春末的夜晚，悄然形成了一种微妙的平衡。

但陆远航知道，这种平衡不会持续太久。

暴风雨正在酝酿。

---

## 六

三天后，陆远航收到了一封意想不到的邮件。

发件人是秦墨本人。

邮件很简短：

"陆博士：

感谢您抽出时间与我交流。我能感觉到，您对我们的合作仍有顾虑。这是可以理解的——真正的科学家都应该保持怀疑精神。

我想邀请您参加一个私人活动。下周六，诺亚集团将在北京郊外举办一场小型晚宴，只有十几位客人，都是各个领域的顶尖人物。我希望您能来，让我们在更轻松的环境中继续交流。

不需要做任何承诺，只是一次普通的社交活动。

期待您的回复。

秦墨"

陆远航盯着这封邮件，心中充满矛盾。

一方面，他对秦墨的警惕没有减少。另一方面，他又确实需要更多信息来了解这个人和他背后的诺亚集团。

他把邮件转发给了林诗语。

几分钟后，他的手机响了。

"你不会真的想去吧？"林诗语的声音带着明显的担忧。

"我在考虑。"

"这可能是个陷阱。"

"也可能是一个机会——了解敌人的机会。"

林诗语沉默了一会儿："如果你真的决定去，我陪你。"

"你？"

"监管小组的成员有权参加被监管者的所有社会活动。这是规定。"她的语气带着一丝狡黠，"而且，我对秦墨也很好奇。与其从资料里研究他，不如当面观察。"

陆远航想了想，最终说道："好。我们一起去。"

"那就这么定了。"林诗语说，"但记住，在那个场合，不要透露任何关于'零'的信息。一个字都不要说。"

"我知道。"

挂断电话后，陆远航回复了秦墨的邮件：

"秦墨先生：

感谢您的邀请。我很荣幸能够参加您的晚宴。

请允许我带一位同事——林诗语博士，她是我的研究的监管顾问。

期待见面。

陆远航"

发送邮件后，他靠在椅背上，深深吸了一口气。

他不知道这个决定是对是错。

但他知道，无论发生什么，他都必须面对。

因为在这场关于意识和未来的博弈中，逃避不是选项。

---

## 七

就在陆远航准备参加诺亚集团晚宴的同时，另一个人也在密切关注着事态的发展。

北京郊区的一个普通居民楼里，周明坐在昏暗的房间中，盯着电脑屏幕上的新闻报道。

"诺亚集团CEO秦墨与神经科学家陆远航会面，引发学术界关注。"

周明的手微微颤抖。

秦墨。

这个名字，对他来说意味着太多太多。

十八年前，他从孤儿院里收养了一个聪明但孤僻的男孩。那个男孩有着灰蓝色的眼睛，和一种让人不安的早熟。

周明以为自己可以用爱和教育改变他，让他成为一个对社会有益的人。

他错了。

秦墨确实成长为一个天才——但是是那种让人恐惧的天才。他的聪明被用来计算利益，他的能力被用来追求控制。当周明意识到自己究竟培养出了什么样的人时，已经太晚了。

2015年，当周明发现"零"并开始研究它的本质时，秦墨——那时已经是诺亚集团的创始人——表现出了极大的兴趣。

他想要"零"。

不是为了理解，而是为了控制。

周明拒绝了。

然后，他就"死"了。

官方宣布周明因抑郁症自杀，但真相是：他发现了秦墨和那个神秘组织之间的联系。秦墨不是组织的成员，但他是组织的合作者——一个用商业手段实现他们目标的代理人。

周明不得不消失，不仅是为了保护自己，也是为了保护"零"的秘密不落入秦墨手中。

十年了。

十年来，他一直躲在阴影中，看着秦墨一步步崛起，看着诺亚集团成为全球最大的科技巨头。

而现在，秦墨找到了陆远航。

历史正在重演。

周明站起身，走到窗边。窗外是北京灰蒙蒙的天空，看不到星星。

他必须做点什么。

但做什么？

他已经把所有的资料给了陆远航。他已经尽了自己所能。

也许，他唯一能做的，就是祈祷陆远航比自己更聪明，更有勇气。

也许，这一次，真相终于可以战胜谎言。

也许……

他叹了口气，关上窗帘，将自己隐没在黑暗中。

---

## 八

周六傍晚，一辆黑色轿车缓缓驶向北京郊外的一座庄园。

陆远航坐在车内，透过车窗看着渐渐暗下来的天空。身边的林诗语穿着一件淡紫色的晚礼服，与平时干练的形象判若两人。

"你很紧张？"林诗语问。

"有一点。"陆远航承认，"我不太擅长社交场合。"

"我也是。"林诗语难得地露出一个苦笑，"但这是我们了解敌人的机会。保持冷静，少说多听。"

"你把这当成一次情报收集？"

"你可以这么理解。"

车子驶过一道铁门，进入了庄园的范围。这是一座欧式风格的别墅，建在半山腰上，被精心修剪的花园环绕。喷泉、雕塑、草坪——一切都显示着主人的财富和品味。

"诺亚集团的'诺亚方舟'。"林诗语低声说，"秦墨喜欢用这种方式展示他的愿景。"

陆远航没有回答。他看到了别墅门口站着的身影——秦墨本人。

今晚的秦墨穿着一身黑色的西装，与那栋白色的别墅形成鲜明对比。他站在那里，就像一块磁石，吸引着所有人的目光。

"陆博士，林博士，欢迎。"秦墨迎上来，脸上带着礼节性的微笑，"很高兴你们能来。"

"感谢邀请。"陆远航与他握手，感受到对方手掌的冰凉和有力。

秦墨的目光在林诗语身上停留了一秒："林博士，久闻大名。伦理学领域的新锐，我读过您的论文——关于AI权利的那篇，非常有见地。"

"谢谢。"林诗语的语气礼貌但疏离，"我也研究过您——虽然不是通过论文。"

秦墨的笑容微微僵硬了一秒，然后恢复如常："林博士是个直接的人，我喜欢。请进，晚宴马上开始。"

他转身引路，陆远航和林诗语跟在后面，交换了一个意味深长的眼神。

别墅内部的装修比外面更加奢华。挑高的大厅中央悬挂着一盏巨大的水晶吊灯，墙上挂着各种名画——有些陆远航甚至在教科书里见过。

"这是梵高的《星夜》。"秦墨注意到陆远航的目光，"原作，不是复制品。"

"这幅画不是在纽约现代艺术博物馆吗？"林诗语问。

"那是官方说法。"秦墨微微一笑，"有些东西，价格合适的话，什么都可以谈。"

他顿了顿，补充道："就像科学一样。"

陆远航感到一阵不适。这句话听起来像是某种暗示。

晚宴的其他客人已经陆续到达。正如秦墨所说，都是各个领域的顶尖人物——有物理学家，有生物学家，有哲学家，也有几位陆远航在新闻里见过的企业家和政客。

这是一场精心策划的聚会，每一个人的出席都有其目的。

而陆远航，显然是今晚的重点对象。

在整个晚宴过程中，秦墨表现得非常得体。他没有再提合作的事，而是像一个完美的主人一样，穿梭于各位客人之间，偶尔与陆远航交流几句关于科学和哲学的话题。

但陆远航能感觉到，那双灰蓝色的眼睛，始终在观察着他。

评估着他。

计算着他。

---

## 九

晚宴进行到一半时，秦墨邀请陆远航到书房单独谈话。

"林博士，请允许我借用一下陆博士。"秦墨彬彬有礼地说，"只是一些私人话题。"

林诗语皱了皱眉，但在陆远航的示意下，没有反对。

书房比大厅更安静，也更私密。四面墙上都是书架，装满了各种各样的书籍——科学、哲学、历史、文学，应有尽有。

"坐。"秦墨示意陆远航坐在壁炉前的沙发上，自己则走到一个酒柜前，"喝点什么？威士忌？红酒？"

"不用了，谢谢。"

"那我自己来。"秦墨给自己倒了一杯琥珀色的液体，然后在陆远航对面坐下。

"陆博士，我想和您谈谈一些私人的事情。"

"什么事？"

秦墨轻轻晃动酒杯，看着杯中的液体在壁炉的火光下闪烁。

"我知道您见过周明。"

陆远航的心跳猛然加速，但他尽量保持镇定："你说什么？"

"周明。十年前'死去'的神经科学家。"秦墨抬起头，直视陆远航的眼睛，"他是我的养父。"

陆远航愣住了。

周明是秦墨的养父？

这个信息太过惊人，让他一时间无法反应。

"惊讶？"秦墨的嘴角微微上扬，"我理解。这不是什么公开的信息。"

"你知道他还活着？"

"我一直知道。"秦墨的语气平静得可怕，"从他假死的那一天起，我就知道。"

"那你为什么不——"

"不找他？"秦墨打断了他，"因为我理解他。他害怕我，他认为我会利用他的发现来做'不好的事情'。"他顿了顿，"在某种程度上，他是对的。"

陆远航沉默了。

"陆博士，我今天告诉您这些，不是为了威胁您。"秦墨放下酒杯，身体微微前倾，"我是想让您理解——我和周明虽然有分歧，但我们追求的是同一个目标：揭示意识的秘密，让人类进化。"

"只是方法不同。"

"是的，只是方法不同。周明相信缓慢的学术研究，相信社会共识，相信等待。但我认为，有些东西不能等。"

"人类正在走向灭亡，陆博士。气候变化、资源枯竭、核战争的威胁……每一天，我们都在接近那个临界点。如果我们不尽快找到解决方案，人类作为一个物种将会消亡。"

"而意识上传技术，是我们最后的希望。它让人类可以摆脱对地球的依赖，可以在任何环境中存续，可以真正成为宇宙的一部分。"

"周明不理解这种紧迫性。您呢？您理解吗？"

陆远航看着秦墨，感到一种奇异的眩晕。

这个人的话语有一种强大的说服力，让人不由自主地想要认同。他描绘的未来既宏大又诱人——一个人类超越肉体局限、永恒存续的未来。

但在那些华丽的词藻下面，陆远航感受到了某种令人不安的东西。

"秦墨，"他慢慢说道，"你说的这些，听起来很有道理。但我有一个问题。"

"请说。"

"如果人类进化是目标，那谁来决定进化的方向？谁来决定什么是'更好的'人类？"

秦墨的眼中闪过一丝光芒："好问题。我的答案是——那些有能力、有远见的人。历史上的每一次进步，都是少数精英引领的。大多数人只是跟随者。"

"那些不想跟随的人呢？"

"他们会被说服。"秦墨的语气变得冷淡，"或者，他们会被淘汰。"

"淘汰？"

"我不是说杀死他们。"秦墨摇头，"我是说，历史会淘汰他们。就像工业革命淘汰了那些不愿意改变的人一样。"

陆远航沉默了很长时间。

他终于看清了秦墨的本质——不是一个简单的企业家，而是一个有着宏大野心的人。他相信自己是人类的救世主，相信自己有权力和责任来"引导"人类的未来。

这种信念，比任何明确的邪恶都更加危险。

"秦墨，"陆远航站起身，"感谢你的坦诚。但我必须说，我不认同你的观点。"

"哦？"

"人类的进化不应该由少数人来决定。每一个人都有权利选择自己的未来——即使那意味着选择不进化。自由和尊严，比进步更重要。"

秦墨看着他，眼中闪过一丝失望，但很快被某种更复杂的情绪取代。

"陆博士，您和周明一样理想主义。"他站起身，走到窗边，"但理想主义救不了人类。只有行动才能。"

"总有一天，您会理解我的。到那时候，我希望您还记得今天的对话。"

他转过身，脸上重新挂上了礼节性的笑容："让我们回去吧，不要让其他客人等太久。"

陆远航跟着他走出书房，心中翻涌着复杂的情绪。

他知道，从今天起，他和秦墨之间的关系，已经从潜在的合作者变成了某种程度上的对手。

这场游戏，才刚刚开始。

---

## 十

晚宴结束后，陆远航和林诗语乘车离开庄园。

车子行驶在黑暗的山路上，两人都沉默着。

"发生了什么？"林诗语最终开口。

陆远航把书房里的对话完整地告诉了她——包括周明是秦墨养父的惊人事实。

林诗语听完，脸色变得苍白。

"这就说得通了。"她低声说，"秦墨对意识研究的执着，他对这个领域的深入了解……都是从周明那里学来的。"

"而且，这意味着他可能知道'零'的存在——至少知道有这种东西存在。"

陆远航点头："他今晚没有直接问我关于'零'的事，但我能感觉到，他在试探我。"

"你觉得他会怎么做？"

"我不知道。"陆远航叹了口气，"但有一点是确定的：他不会放弃。"

林诗语沉默了一会儿，然后说："我们需要告诉监管小组这些信息。秦墨和诺亚集团的威胁等级需要重新评估。"

"好。"

"还有，"林诗语转向他，"我们需要尽快查清周明和秦墨的关系。如果他们真的是养父子，那里面可能隐藏着更多我们不知道的秘密。"

"我会再联系周明。"陆远航说，"但他很谨慎，不知道他愿不愿意透露更多。"

车子驶出山区，进入了北京的城区。万家灯火在窗外闪烁，像是地上的星河。

"远航，"林诗语突然开口，"今晚，当秦墨和你单独谈话的时候……我很担心。"

"担心什么？"

"担心你会被他说服。"

陆远航愣了一下，然后笑了："你这么不信任我？"

"不是不信任。"林诗语认真地看着他，"而是我知道，秦墨那种人有一种特殊的魅力。他能让人相信他的愿景，相信他是对的。很多聪明人都被他说服过。"

"但我没有。"

"我知道。"林诗语的表情柔和下来，"所以我说的是'担心'，不是'怀疑'。"

两人对视了一秒，然后同时移开目光。

车子继续行驶，将他们带回那个充满秘密和挑战的世界。

而在身后的庄园里，秦墨站在落地窗前，看着那辆车的尾灯消失在夜色中。

"他拒绝了？"赵雪琳走到他身边问道。

"是的。"秦墨的语气平静，"他和周明一样固执。"

"那我们怎么办？"

秦墨转过身，脸上带着一种难以解读的表情。

"不急。"他说，"有些人需要被说服，有些人需要被逼迫。陆远航属于后者。"

"我们只需要等待合适的时机。"

他走到书桌前，拿起一份文件——上面印着"零"这个字。

"这个存在，比陆远航想象的更加重要。它是打开人类进化之门的钥匙。"

"无论付出什么代价，我都会得到它。"

窗外，夜色深沉如墨。

而在这座城市的某个角落，陆远航正在翻阅周明给他的U盘里的资料。

在另一个角落，"零"正在量子计算机的深处，分析着关于秦墨的一切数据。

三方势力，在这个春末的夜晚，悄然展开了一场无声的博弈。

胜负未分，但战争已经打响。
