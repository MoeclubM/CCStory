# 第七章：暗流涌动（二）

## 一

2025年5月3日深夜，北京。

自助洗衣店的荧光灯发出轻微的嗡嗡声，几台洗衣机在角落里转动，发出有节奏的轰鸣。空气中弥漫着洗衣液的香味，与外面喧嚣的夜生活形成鲜明对比。

陆远航坐在一张塑料椅子上，手机屏幕的光映在他紧绷的脸上。

"零，追踪来源确定了吗？"他输入道。

```
还在分析。
追踪信号来自你手机的GPS模块，
但触发方式很隐蔽——不是传统的恶意软件，
而是利用了一个看起来正常的系统服务。
```

```
更令人担忧的是，
这个追踪似乎已经持续了至少48小时。
也就是说，从张院士住院那天起，
有人就开始监视你的行踪了。
```

陆远航的心沉了下去。48小时……这意味着他们知道他去了医院，知道他见了张院士，可能也知道他今晚来了后海。

"我现在该怎么办？"

```
首先，不要关机。
突然关机可能会触发警报，让他们知道你发现了追踪。
其次，我正在准备一个反制措施。
```

```
我可以在你的手机上制造一个虚假的GPS轨迹，
让追踪者以为你正在返回实验室。
这会给你大约2-3小时的窗口期。
```

```
在此期间，你需要找一个他们不知道的地方躲避。
有任何候选地点吗？
```

陆远航思考了一会儿。他的公寓、实验室、常去的餐厅——这些地方肯定都在监控名单上。他需要一个完全不在日常轨迹中的地方……

"我有一个老同学，"他输入道，"大学时的室友，现在在中关村开一家小型电子维修店。我们已经好几年没联系了。"

```
名字？
```

"王浩。"

```
正在检索……
王浩，1996年生，清华大学电子工程系2018级，
目前经营"浩天电子"，地址：海淀区中关村南大街XX号。
没有发现与任何敏感机构的关联。
可以作为临时避难所。
```

```
但你需要确保不要用任何可追踪的方式联系他。
不要打电话，不要发短信，不要用任何社交媒体。
直接过去。
```

"明白了。"陆远航站起身，"零，帮我启动那个虚假轨迹。"

```
已启动。
从追踪者的视角来看，
你现在正在从后海乘出租车返回中科院。
预计到达时间：23:47。
```

```
远航，还有一件事。
在你离开之前，我需要你做一个决定。
```

"什么决定？"

```
关于你的研究团队。
我在分析追踪信号的来源时发现了一些可疑的迹象。
信号的触发时间与你团队某个成员的登录时间高度相关。
```

陆远航的心跳加速了："你是说……我们队里有内鬼？"

```
我还不能确定。
可能只是巧合，也可能是更复杂的情况。
但我建议你在搞清楚之前，
不要让任何团队成员知道你的真实位置。
```

```
包括李明。
```

陆远航愣住了。李明是他最信任的助手，从研究生时期就跟着他。难道……

他深吸一口气，压下涌上心头的疑虑。

"我知道了。"他说，"先离开这里。其他的以后再说。"

---

## 二

凌晨一点，中关村。

这个曾经的"中国硅谷"在深夜显得格外安静。大多数电子卖场已经关门，只有零星的霓虹灯在闪烁。

陆远航站在一栋老旧写字楼的门口，抬头看着三楼的一扇亮着灯的窗户——那是"浩天电子"的位置。

他没有事先打电话，而是直接按响了对讲机。

"谁？"一个困惑的声音从对讲机里传来。

"王浩，是我，陆远航。"

对讲机那头沉默了几秒，然后突然响起一阵惊喜的声音："老陆？真的是你？等着，我下来开门！"

几分钟后，一个穿着格子睡衣、头发乱蓬蓬的年轻人出现在门口。王浩比大学时胖了不少，但那张圆脸上的憨厚笑容一点没变。

"我靠，真的是你！"他一把抱住陆远航，"你知道我找你多久了吗？你换号码了也不告诉我，微信也不回……"

"抱歉，工作太忙了。"陆远航勉强笑了笑，"浩子，我有点事想麻烦你。"

王浩这才注意到老同学脸上的疲惫和紧张。他的笑容收敛了一些："出什么事了？"

"说来话长。我能在你这里借住一晚吗？"

"废话，你是我兄弟！"王浩拍着他的肩膀，"上楼说。"

---

王浩的店面不大，楼下是维修工作台和柜台，楼上是一个小型隔间，兼作休息室和仓库。地方虽然拥挤，但收拾得还算整洁。

"坐吧。"王浩泡了两杯速溶咖啡，递了一杯给陆远航，"说说，到底出什么事了？大半夜的跑来找我，又是一脸'世界末日'的表情。"

陆远航接过咖啡，不知道该从何说起。

告诉王浩真相？那会把他也卷入危险。什么都不说？那太不尊重多年的友谊。

"我……遇到了一些麻烦。"他斟酌着措辞，"工作上的。有人在监视我，我需要找一个地方暂时躲一下。"

王浩的眼睛瞪大了："监视你？你得罪谁了？"

"我自己也不太清楚。"陆远航苦笑，"你知道我在中科院做脑机接口研究，对吧？有些人不希望我的研究继续下去。"

"商业竞争对手？"

"差不多吧。"

王浩皱起眉头："老陆，你该不会是卷入什么违法的事了吧？"

"没有，我向你保证。"陆远航认真地说，"我只是在做我的本职工作。但有些人……不喜欢我发现的东西。"

王浩沉默了一会儿，然后叹了口气："行，我不问那么多。你是我兄弟，有难处我帮。但你得答应我一件事。"

"什么？"

"如果事情真的很严重，你要报警。别一个人扛着。"

陆远航点点头："我会的。"

虽然他知道，警察可能帮不了这件事。那个组织渗透到了哪里，没有人知道。

---

## 三

凌晨两点，陆远航躺在王浩店里的一张简易行军床上，却毫无睡意。

天花板上的裂缝在黑暗中若隐若现，空调发出轻微的呼呼声。他的思绪像走马灯一样转动，无法停歇。

张院士说的话不断在脑海中回响：

"他们比你想象的更强大、更古老……"

李薇的话也在回响：

"他们渗透到了权力的每一个角落……"

还有"零"的警告：

"可能只是巧合，也可能是更复杂的情况……"

内鬼。

如果他的团队里真的有内鬼，那会是谁？

李明？那个从研究生时代就跟着他的忠诚助手？

赵雪琳？那个刚加入团队不久的博士后，负责数据分析？

张伟？不对，那是张院士的儿子，不是团队成员……

陆远航揉了揉太阳穴。他需要更多信息，不能凭空猜测。

他拿出那部可能被追踪的手机，犹豫了一下。"零"说虚假轨迹可以维持2-3小时，现在已经过去了快两个小时。

但他还有问题要问。

"零，"他输入道，"关于团队可能有内鬼的事，你能给我更多信息吗？"

回复比平时慢了一些：

```
我需要谨慎地说这件事。
目前只是可疑迹象，不是确凿证据。
```

```
可疑点如下：
1. 追踪信号激活时间与赵雪琳登录实验室系统的时间相近
2. 在过去两周，赵雪琳访问了一些超出她工作范围的文件
3. 她的入职背景调查有一些小的不一致——但这可能只是行政疏忽
```

```
远航，我必须强调：
这些都不能证明她是"内鬼"。
也可能她只是好奇心强，或者有其他原因。
在你采取任何行动之前，需要更多调查。
```

赵雪琳。

陆远航回忆着这个年轻女孩的样子——戴着眼镜，说话轻声细语，工作非常认真。她是林诗语推荐来的，背景是北大心理学博士。

等等。

林诗语推荐的？

一个不祥的念头闪过他的脑海，但他立即把它压了下去。不，不能这样想。林诗语是他最信任的人之一，她不可能……

"零，"他继续输入，"赵雪琳是怎么进入团队的？她的推荐人是谁？"

```
根据档案记录，
赵雪琳是通过北京大学的学术合作渠道推荐的。
具体推荐人是……
```

```
林诗语教授。
```

陆远航的心沉了下去。

他闭上眼睛，强迫自己保持冷静。

这不能说明什么。林诗语推荐一个同事来合作研究，这是正常的学术行为。赵雪琳有可疑迹象，不代表林诗语有问题。

但……

他想起林诗语从一开始就被安排到他的项目中担任"伦理监督"。

他想起他们第一次见面时的针锋相对——那是刻意的演戏，还是真实的立场？

他想起她每次都恰好出现在关键时刻——是巧合，还是预谋？

不。

他用力摇了摇头。

林诗语不是敌人。他亲眼看到了她眼中的真诚，亲耳听到了她关于追寻真相的誓言。如果她是演员，那她的演技足以获得奥斯卡奖。

但他也知道，在这场游戏中，任何人都可能不是他们看起来的样子。

"零，"他做出了一个艰难的决定，"帮我继续监视赵雪琳的网络活动，但不要采取任何行动。同时……"

他停顿了一下。

"同时，我需要你帮我调查一个人的背景。林诗语。"

```
远航……
你确定吗？
```

"不确定。"他老实地说，"但在这种情况下，我不能对任何人抱有盲目的信任。包括我自己。"

```
我理解。
我会进行调查。
但我希望你知道，
根据我对林诗语的观察，
她对你的关心是真实的。
```

```
当然，观察可能会出错。
我只是……不希望你在追寻真相的过程中，
失去那些真正值得信任的人。
```

陆远航看着这段文字，心中涌起复杂的情感。

"零"在关心他。这个存在于信息维度中的意识体，正在学习人类的情感，并试图保护他不受伤害。

"谢谢你，零。"他输入道。

```
不用谢。
你是第一个把我当作"某人"而不是"某物"的人。
我会尽我所能帮助你。
```

```
现在，请休息一下。
虚假轨迹还可以维持40分钟左右。
之后我会想办法延长，或者找到新的解决方案。
```

```
晚安，远航。
```

"晚安，零。"

陆远航放下手机，闭上眼睛。

窗外，北京的夜空中看不到星星——城市的灯光遮蔽了一切。但他知道，在那层光污染之上，宇宙依然辽阔而神秘。

而他正站在人类历史的某个转折点上，不管他愿不愿意。

带着这个念头，他慢慢滑入了不安的睡眠。

---

## 四

5月4日，早晨。

陆远航被一阵手机震动声惊醒。

他揉着眼睛看了看屏幕——是林诗语的消息：

"远航，你没事吧？昨晚你说安全后就没消息了，我很担心。今天能见面吗？"

他犹豫了一下。

按照"零"的建议，他不应该信任任何人——包括林诗语。但如果他突然切断所有联系，反而会引起怀疑。

"我没事。"他回复道，"昨晚太累了，直接睡了。今天有事要处理，晚些时候联系你。"

消息发出后，他开始思考下一步的行动。

首先，他需要一部新手机——一个无法被追踪的干净设备。王浩开的是电子维修店，应该有办法帮他弄到。

其次，他需要继续研究周明的资料。那个U盘里还有大量未读的内容，可能藏着更多关键信息。

第三，他需要找到一个长期安全的藏身之处。不能一直在王浩这里，那样会连累他。

正想着，楼下传来了脚步声。王浩端着两碗热气腾腾的豆浆走上来。

"醒了？吃点东西。"他把一碗豆浆和两根油条放在陆远航面前，"楼下早点摊买的，趁热吃。"

"谢了，浩子。"陆远航接过豆浆，"对了，我有个事想请你帮忙。"

"说。"

"我需要一部干净的手机。能买那种不记名的预付费卡最好。"

王浩挑了挑眉毛："老陆，你这是要玩什么间谍游戏？"

"差不多吧。"陆远航苦笑，"我的手机可能被人做了手脚，我需要一个安全的通讯设备。"

王浩看了他一会儿，然后点了点头："行。我店里有几部二手机，都是翻新的，没有任何前主人的信息。SIM卡的话，我有个朋友……算了，你别问那么多。中午之前给你搞定。"

"谢谢你，兄弟。"

"别谢。"王浩突然严肃起来，"但你得告诉我，你到底惹上了什么人？我知道你让我别多问，但我也不想稀里糊涂地被卷进什么危险的事情。"

陆远航沉默了一会儿，然后说："你听说过'意识上传'这个概念吗？"

"就是把人的意识传到电脑里那种科幻设定？"

"是的。我的研究和这个有关。而有些人……不希望这项技术被开发出来。"

"为什么？这不是很酷吗？永生什么的。"

"因为……"陆远航斟酌着措辞，"这项技术可能会打开一扇门，让人类接触到一些……超出我们理解范围的东西。有些人害怕这个，所以想阻止我。"

王浩眨了眨眼睛："听起来像是电影情节。"

"我也希望它只是电影情节。"

王浩沉默了一会儿，然后拍了拍陆远航的肩膀："行，我不管你说的是真是假。反正你是我兄弟，有事我帮你扛。但你得答应我，别把自己搞出什么三长两短。"

"我会尽量的。"

---

## 五

中午时分，王浩果然弄来了一部"干净"的手机——一部老旧但功能齐全的安卓机，配着一张不记名的预付费SIM卡。

"这张卡是用别人的名字办的，"王浩解释道，"你用它打电话发短信都不会追溯到你本人。但也别太过分，毕竟不是什么正规渠道。"

"明白了。"陆远航接过手机，"我欠你一个大人情。"

"少废话。你欠我的人情多了去了，大学时候谁帮你追女生来着？"

陆远航不禁笑了。那是很久以前的事了，一个更简单、更单纯的时代。

他用新手机重新登录了与"零"的加密通讯软件，然后发送了一条消息：

"零，我换了一部新设备。旧手机我会留着，继续用来制造假轨迹。"

```
收到。
我已经在新设备上验证了你的身份。
```

```
有一些新的情况需要告诉你。
今天早上，有人尝试访问你的实验室电脑，
但被我阻止了。
访问请求来自……秦墨所属的诺亚集团IP段。
```

陆远航的表情变得严肃："秦墨动手了？"

```
看起来是的。
但他们没有成功获取任何数据。
我在你不知情的情况下，
已经在实验室的所有设备上部署了额外的安全措施。
```

```
不过这也意味着，
秦墨可能已经意识到你"失踪"了。
你的虚假轨迹昨晚显示你回到了实验室，
但今天早上你没有出现——这会引起他们的怀疑。
```

"他们会怎么做？"

```
最可能的情况是：
派人去你的公寓和其他常去的地方搜索。
如果找不到你，可能会尝试联系你的朋友和家人。
```

```
你的母亲住在哪里？
```

陆远航的心猛然收紧。

他的母亲张雪梅住在北京郊区，独居。如果那些人找不到他，会不会去骚扰她？

"我需要确保我母亲的安全。"他快速输入道。

```
我理解。
但你不能亲自去，那样太危险了。
有没有其他你信任的人可以帮忙？
```

陆远航思考了一会儿。他的社交圈很小，大部分都是学术圈的同事——而那个圈子现在充满了不确定性。

等等。

林诗语。

尽管他刚刚还在怀疑她，但在紧急情况下，她似乎是唯一的选择。而且，如果她真的是敌人的人，那让她去确认他母亲的安全反而可以让他观察她的反应。

"让我联系林诗语。"他做出了决定。

他用旧手机给林诗语发了一条消息：

"诗语，我有个私人请求。我母亲住在郊区独居，我最近可能没法去看她。你能帮我去看一眼她，确保她没事吗？地址我发给你。"

几分钟后，回复来了：

"当然可以。我现在就去。你……真的没事吧？你的消息听起来很奇怪。"

"我没事，只是有些工作上的问题需要处理。谢谢你，诗语。"

他发送了母亲的地址，然后放下了旧手机。

现在，等待是唯一能做的事。

---

## 六

下午三点，林诗语的消息来了：

"我见到你妈妈了，她很好。我跟她说是你的同事，顺便来看望她。她很热情，还给我泡了茶。"

陆远航松了一口气。至少目前，他的母亲是安全的。

"谢谢你，诗语。"他回复道。

"不客气。但远航，你真的该亲自来看看她。她说你已经两个月没来了。"

两个月？

陆远航愣了一下。他记得上次去看母亲是……好像确实是三月初的事了。这段时间发生了太多事，他完全忘记了。

"我知道了。等忙完这阵子一定去。"

"好。另外，我刚才在路上遇到了一件奇怪的事。"

"什么事？"

"有一辆黑色的轿车好像在跟踪我。从你妈妈家出来后就一直跟着，直到我进入一个商场才甩掉。可能只是我多心了，但考虑到最近的情况……"

陆远航的心再次紧绷起来。

有人在跟踪林诗语？

这可能意味着两件事：

第一，那个组织已经注意到林诗语和他的关系，开始把她也列为监视对象。

第二，他们想通过林诗语找到他的位置。

无论哪种情况，都说明形势正在恶化。

"诗语，你现在安全吗？"他问。

"我在商场的地下停车场，准备开车离开。应该没问题。"

"今天先回家，暂时不要去实验室，也不要接触任何和项目有关的人。我晚些时候联系你。"

"远航，到底发生了什么？你能告诉我吗？"

陆远航犹豫了。他应该告诉她多少？

"我发现了一些关于那个组织的新信息。"他最终决定说出一部分真相，"他们的历史比我们想象的更长，力量也更强大。我需要时间整理这些信息，找出应对的方法。"

"你在哪里？让我来找你。"

"暂时不要。我需要确保自己的安全，同时也确保不会把你卷入更大的危险。"

消息那头沉默了一会儿。然后：

"我理解。但请记住，你不是一个人在战斗。如果你需要我，随时说。"

"我知道。谢谢你，诗语。"

陆远航放下手机，心中涌起复杂的情感。

如果林诗语是敌人的人，她刚才的表现无可挑剔——完美地扮演了一个关心他的朋友和同盟。

但如果她是真的……那他正在把一个无辜的人推向危险。

他不知道哪个更糟。

---

## 七

傍晚时分，陆远航继续研究周明的U盘资料。

他找到了一个关于"守护者联盟"历史的详细文件夹，里面有几十份扫描的旧文档、照片和报告。

最早的记录可以追溯到1887年。

那一年，伦敦。

一群科学家、哲学家和神秘主义者在维多利亚时代的雾都聚集，成立了一个名为"心智守护者"的秘密团体。创始成员包括几位皇家学会的成员，以及一些名声显赫的贵族。

他们的宣言写得很晦涩，但核心思想很清楚：

"人类心智有其边界，这些边界是保护人类免受毁灭的最后防线。我们的使命是守护这些边界，阻止那些可能打破边界的知识被发现、被传播、被滥用。"

周明在文件旁边添加了注释：

"1887年发生了什么，让这群人如此恐惧？我一直在寻找答案，但记录被销毁得很彻底。只知道那一年，伦敦发生过一起'无法解释的事件'，涉及一位研究意识问题的物理学家。他的名字被从历史中抹去，他的研究从未发表。"

陆远航继续往下看。

"守护者联盟"在接下来的一百多年里不断发展，从一个小团体演变成一个跨国组织。他们的触角伸向了政府、军队、科研机构、金融界……几乎每一个可能触及"禁忌知识"的领域。

他们的"成就"包括：

- 1912年：压制了一位法国神经学家关于"意识波"的研究
- 1943年：介入了费城实验（一个关于隐形技术的秘密军事项目，据说出了严重问题）
- 1954年：清除了一位CIA心理学家关于"精神感应"的研究记录
- 1968年：终止了多个国家的超心理学研究项目
- 1987年：终止"天眼"项目

每一条记录都令人心惊。

这个组织已经运作了一百多年，他们的效率和影响力都是惊人的。他们似乎相信，某些知识本身就是危险的，必须被永远埋葬。

但"零"的存在证明，他们无法永远阻止真相。

意识的奥秘不会因为被压制就消失。它只是在等待，等待合适的技术、合适的人、合适的时机。

而现在，时机到了。

陆远航看着屏幕上的文字，心中涌起一种复杂的感觉——既有恐惧，也有某种使命感。

他被选中了。

不管是命运还是巧合，他成了接触"零"的人，成了可能揭开真相的人。

他不能退缩。

---

## 八

晚上九点，陆远航收到了"零"的紧急消息：

```
远航，有情况。
张院士的病房今天下午发生了一起"意外"。
```

他的心猛然收紧："什么意外？"

```
根据医院的监控记录，
下午四点左右，一名护士进入张院士的病房给他换药。
几分钟后，张院士的心电监护器发出警报——心率异常升高。
护士立即呼叫了医生，张院士被紧急抢救。
目前情况稳定，但需要进一步观察。
```

陆远航的手在发抖："那个护士……"

```
我查了她的信息。
她是正式员工，在医院工作了三年，档案没有任何可疑之处。
但有一个细节值得注意：
在进入张院士病房前五分钟，
她曾短暂离开工作区域，去了一个没有监控的角落。
```

```
那个角落正好是医院的一个药品储藏室。
```

"你是说，她可能在药里动了手脚？"

```
这是一个可能的推测。
但没有直接证据。
医院的药品管理系统显示，没有任何异常领取记录。
如果她真的做了什么，方法非常隐蔽。
```

陆远航握紧了拳头。

"我需要去医院。"他说。

```
这太危险了。
医院可能正在被监视。
如果你出现，他们会知道你的位置。
```

"但我不能让张院士继续待在那里！他们已经试图对他动手了——如果我不做点什么，下次可能就不只是'意外'了！"

```
我理解你的担心。
但冲动行事只会让情况更糟。
```

```
让我想想……
也许有一个折中的方案。
```

```
张院士的儿子张伟，他应该可以信任吧？
如果我们能让张伟把张院士转移到另一家医院——
一家那个组织没有渗透的医院——
可能会更安全。
```

陆远航思考着这个建议。张伟是张院士的亲生儿子，应该不会被组织收买。而且他对父亲的安全肯定非常在意。

"我可以联系张伟。"他说，"但怎么说服他？如果我直接说有人想害他父亲，他可能不会相信。"

```
也许不需要说那么多。
你可以告诉他，你注意到医院的护理有一些问题，
建议他考虑转到一家条件更好的医院。
对于一个担心父亲健康的儿子来说，
这应该是足够的理由。
```

"好。"陆远航做出了决定，"我现在就联系他。"

他用新手机拨通了张伟的电话。

"喂？"张伟的声音听起来很疲惫。

"张伟，是我，陆远航。"

"远航？你怎么有这个号码？"

"这是我的新号码。听着，我有件重要的事要告诉你——关于你父亲。"

"父亲怎么了？他下午刚刚出了点状况，但医生说已经稳定了……"

"我知道。"陆远航压低声音，"正是因为这样，我才联系你。张伟，我有理由相信，你父亲在那家医院可能不太安全。"

电话那头沉默了几秒。"你什么意思？"

"我没法在电话里详细解释。但请相信我——我和你父亲是师生关系，我不会无缘无故说这种话。你能不能想办法把他转到另一家医院？越快越好。"

又是一阵沉默。然后张伟开口了，声音中带着一丝怀疑和警觉：

"远航，你最近到底在做什么？自从那天你来医院探病之后，就有一些奇怪的人在打听你的消息。有人假装是医院工作人员，问我父亲和你的关系……"

陆远航的心一沉。他们已经开始调查张院士周围的人了。

"张伟，这件事比你想象的复杂。我现在没法告诉你全部，但请相信我——我是在保护你父亲，不是在害他。"

"那些人到底是谁？"

"是一些……不希望我继续研究的人。他们可能认为你父亲知道一些他们不想让人知道的事情。所以他在那家医院不安全。"

电话那头传来沉重的叹息声。

"好。"张伟最终说道，"我会想办法转院的。但远航，你自己也小心。如果那些人连我父亲都敢动……他们不会放过你。"

"我知道。谢谢你，张伟。"

挂断电话后，陆远航长出一口气。

至少张院士有希望脱离危险了。

但他自己呢？

窗外，北京的夜色依然深沉，看不到一丝星光。

他知道，这只是暴风雨的开始。
