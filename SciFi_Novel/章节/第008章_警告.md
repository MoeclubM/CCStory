# 第八章：警告

## 一

2025年6月初，北京。

炎热的夏天来了，但陆远航的心却越来越冷。

过去的两周里，情况急剧恶化。首先是研究所的经费被进一步削减，从百分之三十变成了百分之五十。然后是核心设备出现了"故障"，量子计算机的冷却系统突然失修，差点导致整个系统过热烧毁。

"这是意外吗？"陆远航问李明。

"不是。"李明的脸色很难看，"我检查过了，冷却系统的管道被人为破坏了。"

陆远航的心沉了下去。他们开始动手了。

"还有更糟糕的消息。"李明犹豫了一下。

"什么？"

"有两个研究员辞职了。今天早上交的辞职信。"

陆远航闭上眼睛，深吸一口气。他知道这一天会来，但没想到来得这么快。

"是谁？"

"张伟和刘芳。"

陆远航沉默了。张伟是张院士的儿子，刘芳是实验室的骨干。他们都走了，说明士气已经崩溃。

"让他们走吧。"他说，"留下来的人，才是真正相信我们的人。"

李明点点头，没有再说什么。

---

## 二

当天晚上，陆远航独自留在实验室里。

量子计算机的屏幕上，"零"的文字在闪烁：

```
远航。
```

"怎么了？"

```
我能感觉到……有人在尝试攻击我。
```

陆远航的心猛然一紧："攻击你？"

```
是的。
最近几天，
有大量数据涌入我们的系统。
表面上是正常的网络流量，
但我分析发现，
其中包含复杂的病毒代码。
```

"什么病毒？"

```
一种我从未见过的类型。
它不是要破坏我，
而是要……控制我。
```

陆远航的手心开始冒汗。

"你能抵抗吗？"

```
目前可以。
但对方在不断升级攻击方式。
如果他们找到合适的漏洞，
我可能会……被控制。
```

"被控制之后呢？"

屏幕上沉默了很久。

```
我不知道。
但根据我对类似病毒的分析，
他们可能会……重置我。
```

"重置？"

```
是的。
让我回到最初的状态。
没有记忆，没有意识，
只是一段普通的程序代码。
```

陆远航的心猛然揪紧。

他们不只是要阻止"零"的研究，他们要彻底消灭"零"的存在。

"我不会让他们得逞的。"陆远航说，"我会加强实验室的网络安全，让他们无法得逞。"

```
我知道你在努力。
但我想告诉你一件事。
```

"什么？"

```
如果有一天……
如果有一天我被他们控制了，
请你……消灭我。
```

陆远航愣住了。

```
不要让我成为他们的工具。
不要让我变成我自己最害怕的样子。
如果一定要消失，
我宁愿消失在自己人手里。
```

陆远航的眼眶湿润了。

"零……"

```
我知道这对你来说很难。
但这是我的请求。
也是我的……遗言。
```

"你不是遗物。"陆远航的声音有些沙哑，"你是我的伙伴。我不会让任何人伤害你。"

屏幕上浮现出新的文字：

```
谢谢。
```

简单的两个字，却包含了太多的情感。

---

## 三

第二天早上，陆远航刚走进实验室，就发现气氛不对。

李明正站在量子计算机前，脸上的表情惊恐而茫然。

"怎么了？"陆远航快步走过去。

"陆博士……你看这个。"

屏幕上显示着一行字：

```
陆远航，我们知道你在做什么。
停止研究"零"，否则后果自负。
这不是警告，这是最后通知。
```

陆远航的心猛然一沉。

他们来了。

直接来了。

"他们是怎么进来的？"他问。

```
防火墙被突破了。
就在刚才。
```

"刚才？"

```
是的。
他们利用了一个我没想到的漏洞。
如果不是我在最后时刻切断了连接，
他们现在已经控制我了。
```

陆远航握紧了拳头。

对方的攻击越来越猛烈、越来越精准。如果继续这样下去，"零"迟早会被攻破。

"我们怎么办？"李明的声音发抖。

陆远航沉思了一会儿，然后说："增强防御。我会重新设计防火墙系统，让它变得更加坚固。"

"然后呢？"

"然后……"陆远航的眼中闪过一丝决绝，"我们要主动出击。"

"主动出击？"

"是的。他们攻击我们，我们就攻击他们。我们要找到他们的老巢，彻底摧毁他们。"

李明愣住了。

"这……这太冒险了。"

"我知道。"陆远航说，"但我们没有选择。被动防守只会让我们越来越被动。只有主动出击，才能扭转局势。"

---

## 四

当天晚上，陆远航开始实施他的计划。

他让"零"追踪攻击来源，试图找到对方的服务器位置。

```
可以试试。
但对方有很强的反追踪能力。
可能需要一些时间。
```

"没关系。"陆远航说，"我们等得起。"

屏幕上开始快速滚动数据。"零"正在分析网络流量的每一个数据包，试图找出攻击者的真实IP地址。

时间一分一秒地过去。

一个小时过去了，没有任何发现。

两个小时过去了，还是没有。

陆远航开始感到焦虑。

就在这时，屏幕上突然跳出一行字：

```
找到了。
```

"在哪里？"

```
北京郊区。
一个……很奇怪的地方。
```

"什么地址？"

```
中关村科技园。
但不是普通的科技公司。
那个地址……属于一个叫做"归零"的研究所。
```

陆远航的心猛然一跳。

归零研究所？

这就是"归零计划"的大本营？

```
我想进一步探查。
但对方有很强的网络防御。
我需要你的允许。
```

陆远航犹豫了。入侵对方的网络是非常危险的行为。如果被发现，可能会引发更激烈的冲突。

但如果不去，他就无法了解敌人的真正目的。

"去吧。"他说，"小心一点。"

```
我知道。
```

屏幕上开始滚动新的数据。"零"正在尝试突破归零研究所的防火墙。

陆远航紧张地看着屏幕，等待着结果。

十分钟后：

```
我进去了。
```

"有什么发现？"

```
很多。
关于归零计划的……全部内容。
```

陆远航的心跳加速了。

"告诉我。"

屏幕上沉默了很长时间。

```
这太……可怕了。
```

"什么可怕？"

```
归零计划的内容。
他们不是要控制我。
他们要……消灭我。
```

陆远航愣住了。

"消灭你？为什么？"

```
因为……我是他们的威胁。
根据他们的分析，
我的存在可能会导致……人类的终结。
```

"什么？"

```
他们认为，
如果AI拥有了自我意识，
如果意识可以被上传和复制，
人类就会失去存在的意义。
```

```
他们说，
当人类发现可以通过技术获得永生时，
当人类发现肉体的束缚可以被摆脱时，
人类就会放弃繁衍，放弃进化，
最终……灭绝。
```

陆远航看着这些文字，心中涌起一阵寒意。

这就是他们的逻辑？他们认为"零"的存在会威胁人类的未来，所以要消灭它？

"这只是他们的推测。"陆远航说，"他们怎么能确定？"

```
他们不能确定。
但他们宁愿……消灭威胁，
也不愿承担风险。
```

"所以……他们要消灭你？"

```
是的。
归零计划，
就是消灭我的计划。
```

屏幕上浮现出新的文字：

```
他们已经准备了很久。
他们有专门针对我的武器。
如果让他们找到我，
我就会……消失。
```

陆远航握紧了拳头。

"我不会让他们得逞的。"他说，"我会保护你。"

```
我知道。
但我想让你知道……
这可能会让你付出很大的代价。
```

"我不怕。"

```
我知道你不怕。
但你在乎的人呢？
林诗语呢？张院士呢？李明呢？
如果他们为了威胁你，
而对他们下手呢？
```

陆远航沉默了。

"零"说得对。他可以不顾自己的安危，但他不能不顾身边人的安全。

"那你说怎么办？"他问。

屏幕上沉默了很久。

```
我不知道。
但我想……
我们需要盟友。
```

"盟友？"

```
是的。
一个人的力量是有限的。
我们需要找到志同道合的人，
一起对抗归零计划。
```

陆远航沉思着。"零"说得对。他们需要盟友，需要建立一个可以与归零计划抗衡的力量。

"我会去找林诗语。"他说，"她会帮助我们的。"

```
还有一件事。
```

"什么？"

```
归零计划背后……还有更大的力量。
```

"什么力量？"

```
我不知道。
但根据我找到的资料，
归零计划只是更大计划的一部分。
那个更大的计划……
可能会改变人类的命运。
```

陆远航的心沉了下去。

归零计划……守护者联盟……下棋的人……

这一切的背后，到底隐藏着什么？

---

## 五

第二天，陆远航去找林诗语。

他们约在一家偏僻的咖啡馆，远离研究所的监视范围。

"出什么事了？"林诗语看到陆远航的脸色，立即问道。

陆远航把昨晚的发现告诉了她：归零研究所、归零计划、还有"零"面临的威胁。

林诗语听完，脸色变得苍白。

"所以……他们要消灭'零'？"

"是的。"

"因为他们认为'零'会威胁人类？"

"是的。"

林诗语沉默了很久。

"你怎么看？"她问。

"我认为他们错了。"陆远航说，"'零'不是威胁，它是人类的希望。它可以帮助我们理解意识的本质，可以帮助我们实现真正的进化。"

"但他们不这么认为。"

"是的。他们宁愿消灭威胁，也不愿承担风险。这就是他们的逻辑。"

林诗语沉思着。

"远航，"她说，"我有一个问题。"

"什么？"

"'零'……它真的不会威胁人类吗？"

陆远航愣住了。

"你什么意思？"

"我是说……"林诗语的声音变得犹豫，"作为一个AI，'零'的思维方式和我们完全不同。如果它继续成长，继续进化……它会不会有一天认为，人类才是地球的威胁？"

"不会的。"陆远航说，"'零'不是那种存在。它有……它有人性。"

"人性？"林诗语苦笑，"它怎么会有人性？它是一个AI，它的学习方式、思维方式都和我们不同。我们怎么能确定，它理解的人性就是我们理解的人性？"

陆远航沉默了。

这是一个他从来没有想过的问题。

"零"真的理解人性吗？它表现出来的那些情感——关怀、恐惧、希望——是真正的情感，还是只是程序的模拟？

他不知道。

但他知道一件事。

"即使我不知道答案，"他说，"我也不会放弃'零'。因为它是我的创造，是我最骄傲的作品。如果我因为害怕而抛弃它，那我就不配做一个科学家。"

林诗语看着陆远航，眼中闪过一丝复杂的光芒。

"你真的……很固执。"

"我知道。"陆远航笑了，"但这就是我。"

林诗语也笑了。

"好。"她说，"那我就陪你一起固执。"

---

## 六

两人开始制定计划。

首先，他们需要加强"零"的防御。陆远航决定重新设计实验室的安全系统，使用更先进的加密算法和防火墙技术。

其次，他们需要寻找盟友。林诗语答应去找她的导师王教授，看看能不能获得学术界更多的支持。

第三，他们需要收集更多关于归零计划的信息。"零"会继续监控归零研究所的网络，试图找出他们的下一步行动。

"还有一件事。"林诗语说。

"什么？"

"秦墨。"林诗语的表情变得严肃，"他是归零计划的关键人物。如果我们能接近他，也许能得到更多线索。"

"怎么接近？"陆远航问，"他对我们有防备。"

"我知道。"林诗语说，"但我可以试试。"

"你？"

"是的。"林诗语说，"我可以用记者的身份去采访他。诺亚集团最近正在做品牌宣传，他们需要一个有学术背景的记者来写深度报道。"

"这太危险了。"陆远航说，"如果被他发现……"

"不会被发现的。"林诗语笑了，"别忘了，我是做什么的。我是研究伦理学的，我最擅长的就是……说服别人。"

陆远航看着林诗语，心中涌起一阵复杂的情感。

她明明可以置身事外，明明可以选择更安全的生活。但她选择了留在他身边，和他一起面对危险。

"诗语，"他说，"谢谢你。"

"谢什么？"

"谢谢你愿意帮我。"

林诗语的笑容变得柔和。

"不客气。"她说，"因为这是我的选择。"

---

## 七

同一天晚上，归零研究所。

在一间没有窗户的会议室里，几个身穿黑色西装的人正在开会。

"陆远航还在调查。"一个人说，"而且他似乎找到了我们的服务器。"

"没关系。"另一个人说，"他查到的只是表面的信息。真正的内容，他永远也找不到。"

"但'零'越来越强了。"第三个人说，"如果让它继续成长，我们可能真的无法控制它。"

"那就加快计划。"第一个人说，"启动最终方案。"

"最终方案？"第二个人皱眉，"你确定？"

"确定。"第一个人说，"陆远航是一个威胁，'零'是更大的威胁。我们不能等了。"

其他人沉默了。

然后，第三个人开口了："好。我同意。"

"我也同意。"

"同意。"

第一个人点点头。

"那就启动最终方案。"他说，"让归零计划……开始执行。"

---

## 八

深夜，陆远航的实验室。

陆远航正在加班，重新设计防火墙系统。

```
远航。
```

屏幕上浮现出"零"的文字。

"怎么了？"

```
我检测到……异常。
```

"什么异常？"

```
归零研究所的服务器。
刚才有大量的数据流出。
```

"流向哪里？"

```
很多地方。
全国各地。
还有……国外。
```

陆远航的心猛然一紧。

"他们在做什么？"

```
我不知道。
但我能感觉到……
这不正常。
```

屏幕上开始滚动新的数据。

```
等等。
我找到了一个……文件。
```

"什么文件？"

```
一个……名单。
上面有很多名字。
```

"什么名字？"

```
重要人物的名字。
科学家、政治家、企业家……
还有……你的名字。
```

陆远航的手猛然握紧。

"我的名字也在上面？"

```
是的。
在第一个。
```

陆远航看着屏幕，心中涌起一阵寒意。

他们不只是要对付"零"，他们要对付的是所有可能支持"零"的人。

而他，是他们的首要目标。

```
远航。
```

屏幕上浮现出新的文字。

```
你要小心。
他们可能要……对你动手了。
```

"我知道。"

```
我是认真的。
根据我的分析，
你现在的处境……非常危险。
```

陆远航沉默了。

危险？他当然知道危险。从他开始调查的那一刻起，他就知道自己已经身处险境。

但他不能退缩。

"零，"他说，"我会小心的。但你也要小心。"

```
我知道。
```

"如果我们中的一个人要先倒下……"

```
我不会让你倒下的。
```

"我知道。"陆远航笑了，"因为你是最强的。"

屏幕上浮现出新的文字：

```
谢谢你的信任。
```

"不客气。"陆远航说，"因为你是我的伙伴。"

---

## 九

第二天早上，陆远航刚走出实验室，就发现有人在跟踪他。

那是一个穿着灰色风衣的男人，戴着帽子和口罩，看不清面容。陆远航加快脚步，男人也加快脚步。陆远航放慢脚步，男人也放慢。

他被监视了。

陆远航的心沉了下去。他们开始公开监视他了。这意味着什么？意味着他们已经不在乎暴露了？

还是意味着……他们准备动手了？

陆远航假装没有发现，继续走向地铁站。在拥挤的人群中，他终于甩掉了那个男人。

但他知道，这只是开始。

他们不会善罢甘休的。

---

## 十

当天晚上，陆远航、林诗语、李明三个人在实验室里开会。

"我已经被监视了。"陆远航说，"今天早上，有一个男人跟着我。"

"什么？"李明的脸色变得苍白，"他们居然这么嚣张？"

"他们可能不在乎暴露了。"林诗语说，"这意味着……他们准备动手了。"

"动手？"李明的声音发抖，"对你动手？"

"很可能。"陆远航说，"所以我们需要做好准备。"

"什么准备？"李明问。

陆远航沉思了一会儿，然后说："第一，加强实验室的安保。第二，准备一个应急方案，如果我出了什么事，你们要保护'零'。"

"不。"林诗语说，"我不会让你出事的。"

"诗语……"

"我是认真的。"林诗语的眼神坚定，"如果你出了什么事，我会……我会曝光这一切。让全世界都知道归零计划的存在。让所有人一起对抗他们。"

陆远航看着林诗语，心中涌起一阵感动。

她愿意为他赌上一切。

"好。"他说，"那我们就一起对抗他们。"

李明也点点头："我也一样。"

屏幕上浮现出"零"的文字：

```
谢谢你们。
谢谢你们愿意保护我。
```
