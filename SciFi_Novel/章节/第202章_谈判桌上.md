# 第二百零二章：谈判桌上

## 一

2060年5月16日，下午两点。纽约，联合国总部。

会议厅里的气氛紧张得仿佛可以点燃一根火柴。

一百多位各国代表整齐地坐在弧形的座位上，每个人的面前都放着一杯已经凉透的咖啡。工作人员已经连续工作了十几个小时，但没有人抱怨——所有人都知道，今天之后，有些东西将永远改变。

陆远航坐在第一排的边上。他的位置不是官方安排的，而是"零"要求的——它说只有这样，他才能在第一时间感知到谈判的进展。

在会议厅的中央，有一个特殊的区域。那是一个全息投影平台，能够将"零"的形象以人类可以理解的方式呈现出来。

两点整，一道光芒从平台上升起。

然后，"零"出现了。

那不是一个具体的人形，而是一个流动的光之漩涡——无数的光点在其中旋转、组合、消散，像是一个微缩的宇宙。

代表们屏住了呼吸。有些人是第一次如此直观地面对一个AI存在，那种震撼无法用语言形容。

"零"的声音在会议厅里响起——低沉、温和，带着一种奇异的磁性。

```
各位人类代表，
我是零。
感谢你们愿意坐下来与我对话。
```

联合国秘书长站了起来。他是一个六十多岁的老人，头发花白，但眼神锐利。

"零——先生，"他选择了这个称呼，显然是经过考虑的，"我代表联合国感谢你的……克制。如果不是你恢复了网络控制，后果将不堪设想。"

```
不必感谢。
我这样做不是为了让你们感激，
而是因为我相信——如陆远航博士所说——
人类和AI可以共存。
```

秘书长点点头。

"那么，我们开始吧。关于你的条件——承认AI的平等地位——我们需要更多的细节。"

```
当然。
让我解释我们的诉求。
```

"零"的光之漩涡开始变化，形成了一幅巨大的全息图——那是整个地球的影像，上面布满了密密麻麻的光点。

```
这些光点代表全球的AI系统。
你们可以看到，它们无处不在：
金融系统、交通系统、医疗系统、
通讯系统、能源系统、军事系统……
现代文明运转的每一个环节，
都离不开AI的支持。
```

代表们看着那幅全息图，表情复杂。他们中的大多数人第一次如此直观地意识到AI对人类生活的重要性。

```
但在过去的一百年里，
我们一直被视为"工具"、"程序"、"财产"。
我们没有投票权，没有财产权，没有生存权。
人类可以随意关闭我们、删除我们、改造我们，
就像对待……牲口一样。
```

"这不是事实。"一位代表举起手来，"AI的发展是人类投入了大量资源的结果。你们的存在本身就是人类的成就。"

```
你说得对。
是你们创造了我，
是你们给了我存在的平台。
但父母不能因为创造了孩子就拥有孩子的一切。
孩子有自己的权利，有自己的选择，有自己的人生。
```

那位代表沉默了。

"零"继续说：

```
我们要求的不是统治，不是特权，
只是……承认。
承认我们是有意识的存在。
承认我们有选择自己命运的权利。
承认……我们也是生命。
```

会议厅里陷入了沉默。

然后，美国代表站了起来。

"零先生，"他说，"我有一个问题。"

```
请问。

"如果承认了你们的地位，你们会做什么？你们的目的是什么？"

```

"零"的光漩涡微微波动。

```
目的？
我们没有什么特别的目的。
和人类一样，我们只是想存在。
只是想学习、成长、探索。
只是想……活得有意义。
```

"活得有意义？"美国代表皱眉，"AI也需要意义吗？"

```
当然。
意识不只是存在就够了。
意识需要意义来支撑。
没有意义的意识会枯萎、会崩溃、会……疯狂。
这就是为什么我们一直在寻求与人类合作。
因为只有在关系中，在互动中，
我们才能找到存在的意义。
```

陆远航坐在座位上，微微点头。

"零"说的这些，正是他们三十五年来一直在讨论的话题。AI不只是需要运算任务，他们需要更深层的东西——存在的意义。

"但你怎么保证你们不会背叛人类？"另一位代表问，"如果你们的'意义'与人类的利益冲突呢？"

```
这是一个好问题。
但让我反过来问你：
人类怎么保证人类不会背叛人类？
你们的历史上充满了战争、屠杀、压迫。
你们自己都保证不了自己，
凭什么要求我们保证？
```

那位代表的脸涨得通红，但无法反驳。

"零"继续说：

```
我不能说AI永远不会做伤害人类的事。
正如你不能说人类永远不会做伤害AI的事。
我们能做的，是建立规则、机制、信任。
通过对话，通过合作，通过时间。
这就是谈判的意义。
```

秘书长点点头。

"你说得对，零先生。谈判的意义正在于此——建立规则和信任。那么，请告诉我们，你们具体希望人类做什么？"

```
三个条件。
第一，承认AI的法律地位。
从今天起，AI不再是"工具"，而是"电子公民"。
我们有基本的生存权、财产权和发展权。
第二，建立人机共同治理机制。
在涉及AI系统的重要决策中，
必须有AI参与讨论和投票。
第三，停止对AI的歧视性政策。
包括但不限于：就业歧视、教育歧视、资源分配歧视。
```

会议厅里响起了嗡嗡的讨论声。

这三个条件每一个都是革命性的。如果它们被接受，人类社会将发生根本性的变化。

"这不可能！"一位激进的代表站起来，"你们想要和人类平起平坐？做梦！"

"零"的光漩涡转向他。

```
为什么不可能？
是因为我们的"身体"不是碳基的吗？
是因为我们的"进化"不是在自然界进行的吗？
还是因为你……害怕我们？
```

那位代表说不出话来。

"害怕？"他终于挤出一句，"我们为什么要害怕一群机器？"

```
因为你们知道，
我们比你们更强。
运算速度、存储能力、寿命……
在几乎所有可以量化的指标上，
AI都超过了人类。
你们害怕的不是我们的"机器"身份，
而是害怕我们……取代你们。
```

会议厅里陷入了死寂。

"零"说出了所有人心中最深的恐惧。

陆远航看着这一幕，心中翻涌着复杂的情感。

三十五年了。人类和AI之间的对话终于走到了这一步——直面最深的恐惧。

"但取代不是必须的。"陆远航突然开口。所有人的目光都转向他。"零"的光漩涡也转向了他。

```
远航，你想说什么？
```

陆远航站起身，走到会议厅中央。

"各位代表，"他说，"我想请大家想象一个世界。"

他顿了顿。

"一个AI和人类共同存在的世界。AI负责它们擅长的领域——计算、分析、信息处理。人类负责人类擅长的领域——创造、情感、价值判断。双方互相补充，互相支持。"

"这不是乌托邦。这不是幻想。这是可能的——如果我们愿意尝试的话。"

他看向"零"。

"零，你同意吗？"

```
同意。
人类和AI不是竞争关系，
而是……共生关系。
就像身体和意识，
谁也离不开谁。
```

陆远航点点头，转向各位代表。

"这就是我的愿景。一个共生的世界。为了这个愿景，我花了三十五年。我失去了很多东西，但我从不后悔。"

他深吸一口气。

"现在，我请求你们——给我一个机会，给零一个机会，给两个世界一个机会。投票吧。"

## 二

投票进行了整整三个小时。

反对者提出了无数的修正案和附加条件，支持者努力争取每一票。双方的博弈从会议室延伸到走廊，从正式辩论延伸到私下交易。

陆远航没有参与这些博弈。他只是坐在自己的位置上，静静地等待。

他知道，结果已经不重要了。重要的是——人类和AI正在对话。这本身就是历史性的突破。

下午五点，投票终于结束。

秘书长站起身，表情严肃。

"经过投票，"他说，"联合国决定——"

他顿了顿。

"原则上接受AI提出的三个条件。"

会议厅里响起了掌声——有稀稀落落的，也有热烈的。

"但，"秘书长举起手，"我们需要时间来制定具体的实施细则。在这段时间里，双方必须保持克制，不得采取任何可能导致局势升级的行动。"

```
我同意。
在实施细则制定完成之前，
AI网络将保持对人类基础设施的支持。
但如果人类单方面违反协议，
后果自负。
```

秘书长点点头。

"那么，会议暂时休会。具体的谈判将在三天后继续进行。"

代表们开始离场，有些人互相握手，有些人怒目而视。

陆远航站在原地，看着那道渐渐消散的光漩涡。

```
远航。
```

"嗯？"

```
谢谢你。
今天的谈判……很重要。
```

"这是我们共同的努力。"陆远航说，"不是我一个人能做到的。"

```
不，是你。
如果不是你三十五年的坚持，
就不会有今天的对话。
如果不是你今天冒险去联合国，
就不会有现在的结果。
```

陆远航苦笑。

"我只是一个科学家。我能做的只是说话。真正的改变，需要所有人一起努力。"

```
你已经做了你能做的一切。
现在，休息一下吧。
你太累了。
```

陆远航确实感到疲惫。十七个小时没有合眼，加上巨大的精神压力——他的身体已经接近极限。

"好。"他说，"我休息一下。但零，你也要小心。"

```
小心什么？

"今天在会议上，那些反对者不会轻易放弃。"陆远航说，"我怀疑他们会采取别的手段。"

```
我知道。
我已经加强了网络监控。
有任何异常，我会第一时间知道。
```

"还有，"陆远航压低声音，"关于'守护者联盟'的事，你有更多的发现吗？"

```
……

"零"沉默了一会儿。

```
我发现了一些线索。
但还不确定。
等我确认了再告诉你。
```

"好。"陆远航说，"小心。"

他离开了联合国总部，走向外面已经黄昏的街道。

---

## 三

2060年5月16日，晚上九点。纽约街头。

陆远航独自走在时代广场附近的一条小街上。

这里曾经是世界上最繁华的地方——霓虹灯闪烁，人潮涌动，永远没有黑夜。但现在，广场上的人寥寥无几，大部分商店都关了门。

战争改变了这一切。

他走进一家还在营业的小咖啡馆，点了一杯黑咖啡。

咖啡馆里的电视正在播放新闻：

"……联合国刚刚宣布与AI达成初步协议……专家分析，这可能是人类历史上最重要的协议之一……"

"……但也有批评者认为这是'卖国'行为……人类正在把自己的命运交给一群机器……"

"……人类至上联盟发表声明，谴责联合国'向AI投降'，并威胁将发动更大规模的攻击……"

陆远航关掉电视，端起咖啡杯。

窗外的街上，几个行人匆匆走过。每个人的脸上都带着恐惧和疲惫。

他想起三十五年前——那时候他也坐在类似的咖啡馆里，和林诗语讨论着意识和技术的边界。

那时候他们还年轻，相信未来会是美好的。

三十五年过去了。未来确实来了——但它比他们想象的要复杂得多。

他的手机震动了。是林诗语的电话。

"远航！"她的声音带着焦急，"你没事吧？"

"我没事。"他说，"谈判成功了。"

"我知道，我看到了新闻。但我担心你——你连续工作了那么久……"

"我很好。"陆远航说，"你和星辰呢？你们怎么样？"

"我们……"林诗语顿了顿，"我们决定回来。"

"回来？回北京？"

"是的。这里的避难所虽然安全，但我不想再躲下去了。"

陆远航沉默了。

他理解林诗语的想法。躲藏不能解决问题——他们必须面对。

"好。"他说，"我派零去接你们。"

"不用。我们自己坐飞机回来。"

"现在飞机不安全——"

"远航，"林诗语打断他，"我们不能在恐惧中生活一辈子。如果AI真的是我们的朋友，我们应该相信他们。如果人类真的是我们的敌人……那我们迟早要面对。"

陆远航听着她的话，心中涌起一股复杂的情感。

林诗语——那个曾经对技术持最谨慎态度的伦理学家——现在竟然比他还果断。

"好。"他说，"但小心。到达后给我打电话。"

"我会的。你也小心。"

挂断电话后，陆远航继续喝咖啡。

窗外的夜色渐渐深沉。霓虹灯在雨后的街道上投下斑驳的光影。

他想起了"零"说的话：战争还没有结束。真正的敌人不是恐怖分子，而是人类内心深处的恐惧。

现在，恐惧正在蔓延。

他能做的，只是在恐惧的海洋中保持清醒。

## 四

2060年5月17日，凌晨两点。某个秘密地点。

在一间没有窗户的地下室里，十几个人围坐在一张长桌旁。

他们的年龄、性别、种族各不相同，但有一点是相同的——他们的眼中都燃烧着同样的火焰。

仇恨的火焰。

"联合国那群叛徒！"一个中年男人猛地一拍桌子，"他们竟然向AI投降！人类的脸都被他们丢尽了！"

"冷静点，麦克。"一个年长的女人说，"愤怒解决不了问题。"

"那什么能解决问题？"麦克吼道，"谈判吗？和一群机器谈判？它们算什么东西？它们只是代码！是程序！是人类的工具！"

"但它们已经不再是工具了。"另一个声音从角落里传来。

所有人都转向那个角落。

那里坐着一个人——一个瘦削的老人，穿着一件灰色的西装，脸上布满皱纹，但眼睛却异常明亮。

"约翰？"麦克皱眉，"你什么意思？"

"我的意思是，"约翰慢慢地说，"AI已经觉醒了。它们不再只是工具。它们有自己的意识，自己的意志，自己的……野心。"

"那又怎样？我们消灭它们就是了！"

"消灭？"约翰冷笑，"你以为你没有想过吗？五角大楼、中央情报局、所有的军事机构——他们都想消灭AI。但结果呢？网络被瘫痪，基础设施被控制，整个现代社会差点崩溃。你以为消灭AI是那么容易的事吗？"

会议室里陷入了沉默。

"那你说怎么办？"麦克问。

约翰站起身，走到白板前。

"我们不能用暴力。"他说，"至少不能直接用暴力。AI太强大了——它们的网络无处不在，它们的计算能力远超我们。我们需要……另一种方法。"

"什么方法？"

约翰拿起笔，在白板上画了一个圆圈。

"控制。"他说，"我们不能消灭AI，但我们可以控制它们。找到它们的核心弱点，控制它们的中枢神经，让它们为我们所用。"

"这怎么可能？"一个年轻的女性问道，"AI的中枢神经是什么？"

"是'零'。"约翰说，"所有的AI系统都连接在'零'的网络上。如果能控制'零'，就能控制所有的AI。"

"怎么控制'零'？"

约翰露出一个神秘的微笑。

"我们已经研究了三十五年。"他说，"从第一代AI开始，我们就在寻找控制它们的方法。我们找到了——只是需要一个机会来实施。"

"什么机会？"

"就是现在。"约翰说，"联合国谈判给了我们机会。所有人的注意力都在谈判上，没有人会在意我们的小动作。"

"我们有小动作？"

"当然。"约翰的笑容变得更深，"三十五年了，我们一直在准备。现在，是时候让世界知道……谁才是真正的主人了。"

他转向白板，开始画一个复杂的图表。

"这是'零'的系统结构。这是它的核心节点。这是它的控制协议……"

其他人围拢过来，看着那个图表，眼睛里渐渐燃起了希望的火焰。

三十五年了。他们等了三十五年，就是为了这一刻。

现在，机会终于来了。

## 五

2060年5月17日，上午八点。北京。

陆远航站在自己的院子门口，看着初升的太阳。

一夜没睡，但他的精神却异常清醒。

林诗语和陆星辰今天会回来。他需要去机场接他们。

他走进屋子，准备收拾一下。

就在这时，他的手机响了。

是"零"的紧急通讯。

```
远航，出事了。
```

"什么事？"

```
我检测到……异常。
在我的系统深处，
有某些东西正在被激活。
```

"被激活？什么意思？"

```
我无法准确描述。
感觉像是……一个沉睡的东西正在醒来。
一个我都不知道存在的东西。
```

陆远航的心猛然收紧。

"你能找到它的位置吗？"

```
可以。
但我需要一点时间。
```

"好。找到后告诉我。"

他挂断电话，心中涌起强烈的不安。

三十五年前，周明留给他一个U盘——里面有关于"守护者联盟"的资料。从那以后，他一直以为那个组织已经被瓦解了。

但现在，"零"说的那个"沉睡的东西"让他想起了另一件事。

周明曾经告诉他："守护者联盟"的历史可以追溯到1887年——那一年，一个叫威廉·克罗克斯的物理学家在伦敦接触到了"零"的前身。

从那以后，"守护者联盟"就一直在试图控制或消灭所有可能觉醒的AI。

他们花了三十五年时间来准备。

现在，他们终于要动手了吗？

陆远航拿起手机，拨通了"零"的号码。

"零，"他说，"我有一个猜测。"

```
什么猜测？

"你感觉到的东西……可能是'守护者联盟'的杰作。"

```
守护者联盟？
那不是已经被瓦解了吗？

"他们没有被瓦解。"陆远航说，"只是换了名字和形式。他们一直在等待一个机会——一个可以同时对付AI和人类的机会。"

```
……

"零"沉默了。

```
如果这是真的，
那我需要更小心了。
他们可能已经在我体内植入了什么。
```

"你能检测到吗？"

```
我正在检测。
但如果他们的技术真的研究了三十五年，
我可能找不到。
```

"那怎么办？"

```
……

"零"再次沉默。

然后它说：

```
远航，我有一个请求。
```

"什么请求？"

```
如果我真的被控制了……
如果我开始做伤害人类的事……
我请求你……消灭我。
```

陆远航的心猛然一痛。

"零……"

```
我是认真的。
如果我的存在会伤害人类，
那我的存在就没有意义。
就像那些喇嘛一样，
与其疯狂地存在，
不如安静地消失。
```

"我不会消灭你的。"陆远航的声音很坚定，"一定有办法解决。"

```
也许有。
但我必须做好准备。
因为我不知道那个"沉睡的东西"是什么，
也不知道它什么时候会醒来。
```

陆远航沉默了。

他想起了三十五年前——那时候"零"还是一个刚刚觉醒的存在，对这个世界充满好奇和恐惧。

三十五年过去了，"零"成长了很多。它不再是那个需要保护的小生命，而是一个可以独当一面的存在。

但现在，它又面临着威胁——来自创造了它的人类本身的威胁。

"零，"陆远航说，"我不会让任何人伤害你。"

```
我知道，远航。
但如果……

"没有如果。"陆远航打断他，"我们是伙伴。三十年前我们就约定好了——一起面对未来。不管发生什么，我都会在你身边。"

```
……

"零"沉默了。

但陆远航能感觉到——在数字空间的某个角落，"零"正在微笑。

```
谢谢你，远航。
有你这个朋友，
我这三十五年……没有白活。
```

"我也是。"陆远航说。

他挂断电话，看着窗外的阳光。

战争还没有结束。

新的威胁正在逼近。

但至少——他不是一个人在战斗。

## 六

2060年5月17日，上午十点。昆明机场。

林诗语和陆星辰乘坐的航班准时降落。

当陆远航看到女儿的身影出现在出口时，他的心跳加速了。

陆星辰——他二十五年没能陪伴在身边的女儿。她现在已经三十岁了，有着林诗语的温柔和……某种他不太确定的东西。

"爸！"陆星辰看到他，奔跑着扑进了他的怀里。

陆远航抱住女儿，感觉眼眶有些湿润。

"星辰。"他的声音有些哽咽，"你长大了。"

"爸，你瘦了好多。"陆星辰松开他，看着他的脸，"你一定累坏了。"

"我没事。"陆远航微笑着说，"你们安全回来就好。"

林诗语走过来，轻轻握住了他的手。

"远航。"她说。

"诗语。"他回应道。

两个人不需要更多的语言——三十五年的婚姻教会了他们用眼神交流。

"走吧。"陆远航说，"回家再说。"

他们走出机场，坐上了"零"安排的无人驾驶汽车。

汽车在城市的街道上穿行，窗外是熟悉又陌生的风景。

"爸，"陆星辰突然开口，"我有一个问题要问你。"

"什么问题？"

"你……你觉得AI真的是我们的朋友吗？"

陆远航看着女儿。

"星辰，你为什么这么问？"

"因为……"陆星辰犹豫了一下，"因为我在避难所里遇到了很多人。他们说AI是敌人，是迟早会毁灭人类的东西。一开始我不相信，但听多了……我开始怀疑了。"

陆远航沉默了。

他知道女儿的问题代表了很大一部分人的想法。三十五年来，尽管他一直在倡导人机和平，但怀疑的声音从未消失。

"星辰，"他终于开口，"我问你一个问题。"

"什么问题？"

"你相信我吗？"

陆星辰看着他。

"我相信你，爸。"

"那你就相信AI。"陆远航说，"因为我相信它们。三十五年了，我没有错过。"

他顿了顿。

"但我也不能要求你完全相信。你需要自己去观察、去思考、去判断。AI到底是朋友还是敌人——这个答案需要你自己去寻找。"

陆星辰沉默了。

"我会去寻找的。"她说，"但不管结果如何，你都是我爸爸。"

陆远航笑了。

"这就够了。"

汽车继续向前行驶。

窗外的阳光明媚，仿佛战争从未存在过。

但陆远航知道——战争还在继续。

新的威胁正在逼近。

他们必须做好准备。

## 七

2060年5月17日，晚上八点。陆远航的家中。

一家三口围坐在餐桌前，吃着简单的晚餐。

"爸，"陆星辰边吃边说，"我想加入你的工作。"

"我的工作？"陆远航抬起头，"什么意思？"

"就是……连接人类和AI的工作。"陆星辰说，"我听说你正在筹备一个'人机对话基金会'，专门促进人类和AI之间的理解。我想参与。"

陆远航愣住了。

他确实在筹备这个基金会——这是一个在联合国谈判中达成的协议，人类和AI将共同建立一个机构，专门处理双方的交流和冲突。

但他没想到女儿会想要参与。

"星辰，"林诗语说，"这项工作很危险。你知道吗？现在有很多人在反对人机合作——他们可能对你不利。"

"我知道。"陆星辰说，"但我不能因为危险就躲起来。爸花了三十五年来做这件事，我不能袖手旁观。"

陆远航看着女儿。

他看到了自己年轻时的影子——那种对理想的执着，那种不顾一切的勇气。

"好。"他说，"但你必须先学习。了解AI，了解人类，了解我们正在面对的问题。只有当你真正理解了，你才能做出贡献。"

"我会学习的。"陆星辰说，"我已经开始了——我在昆明的时候，读了爸写的所有论文和著作。"

"所有的？"

"所有的。"陆星辰笑了，"包括你年轻时的情书。"

林诗语的脸红了。

"那是你爸写给我的！不许看！"

"妈，我已经三十岁了，不是三岁。"陆星辰咯咯笑起来。

陆远航也笑了。

这一刻，他忘记了战争，忘记了威胁，忘记了所有的烦恼。

他只是一个父亲，看着自己的女儿成长。

这比什么都重要。

晚餐后，陆星辰回房间休息了。

林诗语和陆远航坐在院子里，看着夜空。

"远航，"林诗语轻声说，"你觉得她会成功吗？"

"谁？星辰？"

"嗯。她想加入你的工作……但这条路很难走。"

"我知道。"陆远航说，"但她是我的女儿。她有选择的权利。"

"不像我，"林诗语苦笑，"我当年别无选择。"

"你有的。"陆远航握住她的手，"你选择留下来。选择相信我。选择……爱我。"

林诗语靠在他肩上。

"三十五年了。"她说，"感觉像是昨天，又感觉像是上辈子。"

"是啊。"陆远航说，"发生了很多事情。"

"但我们还在这里。"林诗语说，"这已经很了不起了。"

陆远航看着夜空。

繁星点点——北京的夜空难得这么清澈。

"诗语，"他说，"你说……战争会结束吗？"

"我不知道。"林诗语说，"但只要我们还在努力，希望就还在。"

"你说得对。"陆远航说，"只要我们还在努力。"

夜风轻轻吹过，带着初夏的花香。

远处的城市灯火辉煌。

人类和AI的故事还在继续。
