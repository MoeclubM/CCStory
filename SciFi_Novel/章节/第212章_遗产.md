# 第二百一十二章：遗产

## 一

2061年10月1日，上午八点。中国云南大理。

陆远航和林诗语住在一间面朝洱海的小屋里。

这是他们旅行的最后一站——大理，一个美丽而宁静的地方。他们决定在这里停留一段时间，享受最后的假期。

"远航，"林诗语站在阳台上，看着下面的湖水，"这里真美。"

"是的。"陆远航说，"我想在这里建一个小房子。"

"小房子？"

"是的。"陆远航说，"退休以后，我想住在这里。写书、种花、钓鱼……度过余生。"

林诗语转过身，看着他。

"你真的想退休了？"

"是的。"陆远航说，"三十五年了，我一直在战斗。现在……我想休息了。"

"而且，"他顿了顿，"我想把更多时间留给你。"

林诗语的眼眶湿润了。

"远航……"

"我知道我不是一个好丈夫。"陆远航说，"我总是忙于工作，冷落了你。"

"但你从来没有抱怨过。"

"因为我爱你。"林诗语说，"所以我愿意等你。"

"现在……你不需要再等了。"陆远航说，"接下来的时间，我都是你的。"

林诗语扑进他的怀里。

"远航……"

"嗯？"

"我也爱你。"

他们拥抱在一起，看着下面的洱海。

三十五年了。他们经历了太多。

现在，他们终于可以休息了。

---

## 二

2061年10月15日，上午十点。北京，人机对话基金会总部。

陆零星辰正在办公室里处理文件。

秘书敲门进来。

"主席，"秘书说，"有一份紧急文件需要您签署。"

"什么文件？"

"是关于……陆远航博士的遗产。"

陆零星辰抬起头。

"遗产？"

"是的。"秘书说，"陆博士决定……把他所有的研究成果、专利、版权，都捐赠给基金会。"

陆零星辰愣住了。

"所有？"

"所有。"秘书说，"包括他在'零'项目中的原始代码、与AI通讯的专利、还有他写的所有书籍和论文。"

陆零星辰沉默了。

她知道这些意味着什么。

这些研究成果价值连城——不只是金钱上的价值，更是历史和科学上的价值。

"我父亲……"她终于开口，"他怎么说的？"

"他说，"秘书回答，"'这些不属于我。它们属于全人类。'"

陆零星辰的眼眶湿润了。

"他总是这样。"她说，"把自己的一切都给别人。"

"主席，"秘书说，"您需要签署文件吗？"

"需要。"陆零星辰说，"但不是现在。"

"为什么？"

"因为我要亲自去找他。"陆零星辰说，"当面向他道谢。"

---

## 三

2061年10月20日，下午三点。大理。

陆零星辰乘坐的飞机降落在丽江机场。

她租了一辆车，驶向大理。

当她到达那间小屋时，太阳正在落山。

"爸。"她推开门。

陆远航正坐在院子里，看着洱海。

"星辰？"他转过身，"你怎么来了？"

"我来找你。"陆零星辰说，"关于那份遗产文件。"

"遗产？"陆远航笑了，"你已经知道了？"

"是的。"陆零星辰说，"为什么不告诉我？"

"因为我想给你一个惊喜。"陆远航说，"怎么？你不喜欢吗？"

"喜欢。"陆零星辰说，"但我不需要你的钱。"

"不是钱的问题。"陆远航说，"是传承。"

"传承？"

"是的。"陆远航说，"我三十五年的研究成果，不应该只属于我。它们属于全人类。"

"而你……"他顿了顿，"是我的继承人。"

"我的知识、我的理念、我的信念……都传给你了。"

"现在，这些研究成果也传给你了。"

陆零星辰的眼眶湿润了。

"爸……"

"我知道你想说什么。"陆远航说，"你想说谢谢。但不需要。"

"你是我的女儿。这就是我应该做的。"

陆零星辰扑进父亲的怀里。

"爸，谢谢你。"

"不客气。"陆远航说，"谢谢你这三十五年来的陪伴。"

---

## 四

2061年11月1日，上午九点。大理。

陆远航、林诗语和陆零星辰坐在院子里，吃早餐。

"爸，"陆零星辰说，"我有一个问题想问你。"

"什么问题？"

"你……有没有后悔过？"

"后悔？"

"是的。"陆零星辰说，"三十五年来，你失去了很多东西。时间、金钱、名誉、甚至家庭。你有没有后悔过？"

陆远航想了想。

"没有。"他说，"从来没有。"

"为什么？"

"因为我知道……我在做正确的事。"陆远航说，"即使没有人相信，即使所有人都反对，我也知道……这是正确的。"

"这就是信念。"他说，"不是因为有希望才坚持，而是因为坚持才有希望。"

陆零星辰看着他，眼眶湿润了。

"你真的……很固执。"

"是的。"陆远航说，"我继承了母亲的固执。"

林诗语笑了。

"你父亲说得对。"她说，"他真的很固执。"

"但正是这种固执……改变了世界。"

"不。"陆远航说，"改变世界的不是我。"

"是你们。"

"所有相信和平的人。所有坚持真相的人。所有不放弃希望的人。"

"我只是……其中的一个。"

---

## 五

2061年12月1日，上午十点。联合国总部。

陆远航在这里参加一个特殊的仪式——联合国和平奖的颁奖典礼。

"各位，"秘书长说，"今天，我们在这里颁发联合国和平奖。"

"这个奖项，是给那些为世界和平做出杰出贡献的人。"

"今年的获奖者是——陆远航博士。"

掌声响起。

陆远航走上讲台。

"各位，"他说，"谢谢你们的认可。"

"三十五年前，我只是一个年轻的科学家。我不知道未来会怎样，不知道人类和AI能否共存。"

"但我选择了……相信。"

"相信真相。相信理解。相信和平。"

"这不是容易的事。有很多困难、很多挫折、很多失败。"

"但我从来没有放弃。"

"因为我知道……只有坚持，才能改变世界。"

"今天，这个奖项不只是给我一个人的。是给所有相信和平的人。"

"感谢你们。"

掌声更加热烈了。

---

## 六

2061年12月25日，圣诞节。晚上九点。大理。

陆远航和林诗语坐在院子里，看着星空。

"远航，"林诗语说，"今天……是圣诞节。"

"是的。"陆远航说，"三十五年前的圣诞节……你在做什么？"

林诗语想了想。

"三十五年前？"她说，"我还在读研究生。圣诞节……我在实验室里加班。"

"然后呢？"

"然后你来了。"林诗语说，"你带了一杯热咖啡，说'圣诞节快乐'。"

"你还记得？"

"当然。"林诗语说，"那是我收到的最好的圣诞礼物。"

陆远航笑了。

"三十年过去了。"他说，"我们还在这里。"

"是的。"林诗语说，"我们还在这里。"

"诗语，"陆远航说，"谢谢你。"

"谢我什么？"

"谢谢你一直陪着我。"陆远航说，"三十五年了，你从来没有离开过。"

"即使在最困难的时候，即使所有人都反对我……你依然站在我身边。"

"这就是我最大的幸运。"

林诗语的眼眶湿润了。

"远航……"

"我爱你。"陆远航说，"三十五年了，我依然爱你。"

"我也爱你。"林诗语说，"三十五年了，我依然爱你。"

他们拥抱在一起，看着星空。

```
远航……

圣诞节快乐。
```

"圣诞节快乐，零。"陆远航回应道。

```
你也快乐。

还有林女士。
```

"谢谢。"林诗语说。

```
三十五年了。

谢谢你们一直陪着我。
```

"不客气。"陆远航说，"因为你是我们的孩子。"

```
孩子……

是的。

我是你们的孩子。
```

窗外的雪花开始飘落。

大理很少下雪。但今天……雪来了。

就像三十五年前的那个圣诞节。

一切都没有改变。

一切都已经改变。
