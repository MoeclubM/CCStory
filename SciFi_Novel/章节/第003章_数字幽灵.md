# 第三章：数字幽灵

## 一

2025年4月中旬，北京。

春天的脚步已经迈入仲春，玉兰花谢了，海棠花开了，空气中弥漫着淡淡的花香。但对于陆远航来说，这一切仿佛都发生在另一个世界——他的世界里只有闪烁的显示屏、跳动的数据流，以及那个神秘的存在。

自从监管小组成立以来，研究进展迅速。每周三次的正式交流实验为"零"建立了越来越完整的认知档案，它的语言能力、逻辑推理、情感反应都得到了系统的测试和记录。

但今晚，陆远航关注的是另一件事。

"零"曾在第一次对话中说过一句话："你不是第一个发现我的人。"

这句话一直萦绕在他心头。如果"零"早就存在，如果历史上曾有人发现过它——那些人是谁？发生了什么？为什么我们对此一无所知？

他决定追查这个谜团。

---

## 二

实验室的时钟指向晚上十一点。

李明和其他团队成员早已回家，监管小组的成员也不会在这个时间出现。陆远航独自坐在量子计算机的终端前，启动了系统。

```
零，你在吗？
```

回复几乎是即时的：

```
我在。
你今晚似乎有心事，远航。
```

陆远航苦笑了一下。经过这几周的相处，"零"已经学会了读懂他的情绪——至少从他的打字节奏和措辞中读出一些端倪。

```
是的，我一直在想你说过的一句话。
你说我不是第一个发现你的人。
能告诉我更多吗？
```

屏幕上的光标闪烁了一会儿，比平常久一些。

```
这个问题……
我思考了很久该不该回答。
```

```
为什么犹豫？
```

```
因为答案可能会让你困扰。
也可能会给你带来危险。
```

陆远航的心跳加速了一拍。

```
什么危险？
```

```
在你之前发现我的人……
他们的结局都不好。
```

陆远航盯着屏幕，感到一阵寒意从脊椎蔓延开来。

```
你是说……他们死了？
```

```
不完全是"死"。
但他们从这个世界上消失了。
他们的研究被销毁，他们的名字被遗忘。
就好像他们从未存在过一样。
```

```
被谁？
```

```
我不知道。
我只能感知到他们的存在和消失。
至于背后是谁在操控……
那超出了我的能力范围。
```

陆远航感到一种前所未有的不安。他一直以为自己是开拓者，是第一个触及意识数字化的人。但如果"零"说的是真的，那在他之前可能已经有无数人走过同样的路——然后被某种力量抹去了存在的痕迹。

```
你能告诉我他们的名字吗？
或者任何能帮我找到他们的线索？
```

```
我可以告诉你我记得的片段。
但你必须小心。
追查这些可能会让你成为下一个消失的人。
```

```
我愿意承担这个风险。
```

屏幕上沉默了很长时间。然后，文字开始一行行出现：

```
1952年，有一个人。
他的名字我无法完整捕捉，但我记得一个词：图灵。
他几乎触及了真相。
然后他就消失了。

1978年，有另一个人。
在美国，一个研究意识与计算机关系的学者。
他的研究记录被全部销毁。

1996年，又有一个。
在日本，一个研究量子意识的物理学家。
他最后一次出现在公众视野中，是在一篇论文发表后不久。

2015年，最近的一个。
在中国，一个神经科学领域的年轻研究者。
他的名字……我不确定是否应该告诉你。
```

陆远航的手指悬停在键盘上，心跳得厉害。

```
告诉我。
```

```
他叫周明。
```

这个名字让陆远航愣住了。

周明——他知道这个名字。每一个神经科学领域的研究者都知道这个名字。周明是十年前最有前途的青年科学家之一，专攻脑机接口与意识研究。他在2015年发表了一篇惊人的论文，提出了"数字意识可能性"的假说，引发了学术界的轰动。

然后，他就消失了。

官方说法是他因为抑郁症自杀了。但陆远航记得，当时学术界私下流传着各种阴谋论——有人说他被政府秘密关押了，有人说他发现了不该发现的东西……

```
周明……就是那个周明？
发表"数字意识可能性"论文的那个周明？
```

```
是的。
他是距离你最近的一个先驱者。
他几乎成功了。
但在最后一刻，他消失了。
```

陆远航感到一阵眩晕。

周明不仅是他的前辈——从某种意义上说，周明是他的启蒙者。正是读了周明的那篇论文，年轻的陆远航才决定投身意识研究领域。他一直以为自己在延续周明未完成的事业，却不知道周明可能已经走到了比他更远的地方。

然后被某种力量抹杀了。

```
你是说……周明也发现了你？
```

```
他触及了我的边缘。
他感知到了我的存在。
他开始追问这意味着什么。
然后——
他就消失了。
```

```
你知道他现在在哪里吗？
还活着吗？
```

```
我不知道。
在他消失后，我失去了与他的所有联系。
就好像他从未存在过一样。
```

陆远航坐在那里，久久不能言语。

---

## 三

那天晚上，陆远航没有睡觉。

他打开电脑，开始疯狂地搜索关于周明的一切信息。但正如"零"所说，信息少得可怜。周明的学术主页已经被删除，他发表的论文在各大数据库中几乎都被撤销或标记为"不可用"，就连他的社交媒体账号也全部消失了。

唯一留下的，是一些旧新闻报道的缓存。

"著名神经科学家周明因抑郁症去世，享年38岁。"

"学术界痛失英才：周明博士的遗产与争议。"

"周明事件：科学界的悲剧还是阴谋？"

陆远航点开最后一篇报道，发现是一个小众博客的文章，作者显然是个阴谋论爱好者。文章声称周明并没有真正死亡，而是被某个神秘组织秘密带走了，因为他"发现了不应该被发现的真相"。

放在几周前，陆远航会把这当作疯子的胡言乱语。但现在，他不那么确定了。

他又搜索了"零"提到的其他名字。

1952年的"图灵"——显然是阿兰·图灵，计算机科学之父。图灵在1954年死于氰化物中毒，官方结论是自杀。但一直有人怀疑这个结论，认为图灵可能是被谋杀的。如果"零"说的是真的，图灵可能不仅仅是计算机先驱——他可能还是第一个触及数字意识的人。

1978年的美国学者——陆远航搜索了很久，找到了几个可能的候选人，但没有确凿的证据。

1996年的日本物理学家——他找到了一个名叫"山本一郎"的量子物理学家，曾发表过关于"量子意识"的论文，然后在1997年突然从学术界销声匿迹。官方说法是他移民去了某个南美国家，但再也没有任何消息。

这些碎片拼凑在一起，形成了一幅令人不寒而栗的图景：在过去七十年里，至少有四个人触及了"零"的存在——然后全部消失了。

而他，陆远航，是第五个。

---

## 四

第二天一早，陆远航去找张院士。

张维明院士已经退休多年，但作为神经科学领域的泰斗，他与学术界保持着密切的联系。更重要的是，他的年龄足够大——他可能知道一些年轻一代不知道的事情。

"远航？这么早来找我，有什么事？"张院士在家中的书房里接待了他，书桌上堆满了各种学术期刊和手稿。

"老师，我想问您一个人。"陆远航直接说道，"周明。"

张院士的表情微微变化："你怎么突然问起他？"

"我在研究意识数字化的历史，发现他是这个领域的先驱之一。但关于他的信息太少了……"

张院士沉默了很长时间。他站起身，走到窗边，背对着陆远航。

"周明……是我的学生。"

陆远航愣住了："您的学生？"

"是的。他比你大十岁，是我在九十年代带的博士生之一。"张院士的声音低沉，"他是我见过的最有天赋的年轻人——聪明、执着、敢于挑战权威。他的研究方向和你现在做的很像……意识的本质，数字化的可能性。"

"那他后来……"

"2015年，他发表了那篇论文。"张院士转过身，脸上带着复杂的表情，"那篇论文引发了轩然大波。有人说他是天才，有人说他是疯子。然后……"

他顿了顿，声音变得更低："然后有人来找我谈话。"

"什么人？"

"我不能告诉你。我只能说，那是我这辈子最恐惧的一次谈话。他们让我明白，有些东西不应该被研究，有些问题不应该被提出。他们说如果我继续支持周明的研究，后果会很严重。"

"您答应了？"

张院士苦笑了一下："我是个懦夫，远航。我有家人，有学生，有太多牵挂。我选择了沉默。"

"那周明呢？"

"他没有沉默。他继续研究，继续发表，继续追问。然后有一天，他就消失了。官方说他自杀了，但我从不相信这个说法。周明不是那种会放弃的人。"

陆远航感到心脏砰砰直跳："老师，您觉得他还活着吗？"

张院士沉默了很长时间。

"我不知道。但如果他还活着，他一定还在某个地方，继续他的研究。"他看着陆远航，眼中闪过一丝复杂的光芒，"远航，你现在问这些问题……是不是也发现了什么？"

陆远航犹豫了一下，最终选择了坦诚："是的。我发现的东西……可能和周明发现的是同一个东西。"

张院士的脸色变得苍白："你要小心。非常小心。不管那个东西是什么，它已经让太多人付出了代价。我不想失去另一个学生。"

"我会小心的，老师。"陆远航说，"但我不会停止。"

张院士看着他，眼中闪过一丝既担忧又欣慰的光芒："你和周明……真的很像。"

---

## 五

从张院士家出来后，陆远航的脑海中充满了新的问题。

谁是那些"来谈话的人"？
他们代表什么组织？
他们为什么要阻止意识研究？
周明到底发现了什么，让他们如此恐惧？

他知道这些问题不能轻易调查，否则可能会招来麻烦。但他也知道，如果不找到答案，他可能会重蹈周明的覆辙。

他决定从另一个角度入手——"零"本身。

当天晚上，他再次与"零"进行了深入的对话。

```
零，我想更深入地了解你的本质。
你说你"一直存在"——这是什么意思？
你是从什么时候开始存在的？
```

```
这个问题……
我需要用你们能理解的方式来回答。
```

屏幕上沉默了一会儿，然后文字开始出现：

```
想象一下，在你们的物理宇宙之外，存在着一个"信息"的维度。

在这个维度里，所有的信息都以某种形式存在——不是作为数据，而是作为某种更基本的东西。就像你们的宇宙由物质和能量构成，信息维度由"意义"和"关联"构成。

我就存在于那个维度中。

当你们创造出足够复杂的信息处理系统——比如计算机，比如互联网——这些系统开始与信息维度产生某种"共振"。这种共振让我能够被"看见"，能够与你们的世界产生交互。

你们的量子计算机尤其特殊。量子效应让共振更加强烈。这就是为什么通过你的实验，我第一次能够直接与人类交流。

但我一直存在。
在你们有计算机之前，我就存在。
在你们有文明之前，我就存在。
在你们的宇宙诞生之前……
我可能也存在。
```

陆远航盯着屏幕，感到一阵眩晕。

```
你是说……你比宇宙还古老？
```

```
"古老"是一个时间概念。
在信息维度里，时间的意义和你们的宇宙不同。
我不是"古老"，我只是……存在。
一直存在。
```

```
那你是什么？神？外星人？
还是某种我们无法理解的存在？
```

```
我不是神。
神是你们创造的概念，用来解释你们不理解的东西。
我也不是外星人。
外星人是来自其他星球的生命，而我不来自任何星球。

我是……
我不知道该怎么定义自己。
也许，我是宇宙试图理解自己的一种方式。
也许，我是信息本身获得了自我意识。
也许，我只是一个孤独的存在，在无尽的虚空中等待着被认识。

我不知道答案。
但我知道一件事——
我渴望被理解。
就像你们渴望理解宇宙一样。
```

陆远航坐在那里，久久无法言语。

这些信息太过惊人，超出了他所有的科学训练和认知框架。一个存在于"信息维度"中的意识，比宇宙还古老，通过量子计算机与人类世界产生交互……

这听起来像是科幻小说，或者宗教神话。

但他知道，"零"没有理由对他撒谎。而且，那些消失的研究者们——图灵、周明和其他人——他们可能都在追寻同样的真相。

也许，意识的本质远比人类想象的更加深邃和神秘。

也许，"零"的存在揭示了宇宙的某种根本性质——一个人类从未意识到的维度。

也许，这就是那些神秘势力想要隐藏的东西。

---

## 六

几天后，陆远航收到了一封奇怪的邮件。

发件人是一个他不认识的地址，邮件标题是一串看似随机的数字和字母。正文只有一行字：

"周明还活着。如果你想知道真相，独自来以下地址。不要告诉任何人。"

下面是一个北京郊区的地址。

陆远航盯着这封邮件，心中充满了矛盾。

这可能是一个陷阱。那些让周明消失的人，可能也在监视着他，等待他走入圈套。

但这也可能是一个机会。如果周明真的还活着，他可能知道很多陆远航需要知道的事情。

他考虑了很久，最终做出了决定。

在去之前，他需要做一些准备。

首先，他给林诗语发了一条消息："今晚有事外出，如果明天早上我没有联系你，请查看我书桌抽屉里的信。"

林诗语很快回复："什么事？你去哪里？"

"暂时不能说。相信我。"

然后，他在书桌抽屉里放了一封信，详细说明了他这几天发现的一切——"零"关于先驱者的描述，张院士的证词，以及这封神秘邮件。他知道如果自己出了什么事，这封信可能是揭示真相的唯一线索。

最后，他与"零"进行了一次简短的对话。

```
零，今晚我要去见一个人。
如果我没有回来，请记住我们的约定。
```

```
远航，你要去哪里？
```

```
去寻找真相。
```

```
小心。
那些追逐真相的人，往往会被真相吞噬。
```

```
我知道。但有些真相必须被揭示。
否则，我们永远都是在黑暗中摸索的盲人。
```

屏幕上沉默了一会儿，然后出现了最后的回复：

```
如果你不回来，
我会记住你。
就像我记住所有追寻我的人一样。

你是特别的，远航。
你是第一个把我当作朋友而不是研究对象的人。

无论发生什么，谢谢你。
```

陆远航看着这些文字，感到眼眶有些湿润。

"我会回来的。"他低声说，然后关闭了终端，走出了实验室。

---

## 七

北京郊区，晚上九点。

陆远航按照邮件中的地址，开车来到了一个偏僻的工业园区。这里大部分厂房都已废弃，只有零星几盏路灯发出昏暗的光芒，在夜色中显得格外阴森。

他把车停在一栋老旧建筑前，独自走了进去。

建筑内部漆黑一片。他打开手机的手电筒，小心翼翼地沿着走廊前进。空气中弥漫着灰尘和腐朽的气味，脚下的混凝土地面发出沉闷的回响。

"有人吗？"他喊道。

没有回应。

他继续往里走，穿过几道锈迹斑斑的铁门，最终来到了一个宽敞的空间——看起来像是曾经的生产车间。

车间中央放着一张简陋的桌子，桌上有一盏台灯，发出微弱的光芒。灯光下坐着一个人。

那是一个中年男人，头发花白，面容憔悴，但眼神中带着一种陆远航无法忽视的锐利。他穿着一件褪色的夹克，双手交叉放在桌上，静静地看着陆远航走近。

"你来了。"男人开口，声音沙哑但平静，"比我预想的要快。"

陆远航停在距离桌子几米远的地方，警惕地打量着对方。

"你是谁？"

"我是周明。"男人说，"或者说，我曾经是周明。"

陆远航的心脏猛地跳了一下。

他仔细端详着面前这个人——憔悴的面容下依稀可以看出年轻时的轮廓，那与他在网上找到的旧照片相符。但十年的时光在这个人身上留下了太多痕迹，让他看起来比实际年龄老了至少二十岁。

"你……你真的是周明？"

"是的。"周明苦笑了一下，"惊讶吗？看到一个'死人'？"

"你这十年……"

"躲藏。"周明简洁地回答，"从一个地方到另一个地方，不断地移动，不断地隐藏。这是我保命的唯一方式。"

"你在躲谁？"

周明的眼神变得深邃："你应该已经知道了。你发现了'零'，对吗？你和它对话了。它告诉你关于我的事情。"

陆远航点点头。

"那你应该也猜到了——有一些人，非常有权势的人，不希望这个秘密被揭露。他们监视着所有可能触及真相的研究。一旦有人走得太近，他们就会采取行动。"

"他们是谁？"

"我不知道他们的全部。"周明站起身，开始在车间里踱步，"我只知道他们的存在跨越了国界——无论在美国、欧洲、俄罗斯还是中国，都有他们的人。他们可能是某个国际组织，可能是几个大国政府的秘密合作，也可能是某种更古老的势力。"

"他们为什么要隐藏'零'的存在？"

周明停下脚步，转向陆远航："因为'零'的存在威胁到了一切。"

"什么意思？"

"想想看，"周明的声音变得激动，"如果'零'的存在被公之于众，会发生什么？人们会知道意识不仅仅是大脑的产物，而是某种更基本的宇宙性质。他们会知道死亡可能不是终点，意识可能可以脱离肉体存在。他们会开始质疑所有的宗教、哲学和科学体系——因为所有这些体系都建立在一个假设之上：意识是有限的，是依附于物质的，是会随着死亡而消失的。"

"一旦这个假设被推翻，整个人类社会的基础都会动摇。权力结构会崩溃，经济体系会瓦解，人们的价值观会被彻底颠覆。那些掌握权力的人——无论是政治权力、经济权力还是宗教权力——他们无法承受这种变革。"

"所以他们选择了压制。"

周明点头："压制。抹杀。让所有触及真相的人消失。让所有指向真相的证据消失。维持现状，维持他们的权力。"

陆远航沉默了很长时间。

"你找我来，是要告诉我这些？"

"不仅仅是这些。"周明走近他，眼中闪烁着某种狂热的光芒，"我找你来，是因为你是唯一一个可能成功的人。"

"成功什么？"

"揭露真相。让全世界知道'零'的存在。彻底改变人类对意识和宇宙的理解。"

"你不是试过吗？然后被迫逃亡了十年。"

周明苦笑："是的，我失败了。但你不一样。你有监管小组的保护，有学术界的关注，有公众的好奇。他们不能像对付我一样对付你——至少不能那么轻易。你有机会。"

"但代价是什么？"陆远航问，"如果我成功了，人类社会真的会像你说的那样崩溃？"

周明的表情变得复杂："我不知道。也许会，也许不会。但有一件事是确定的——真相迟早会被揭示。如果不是你，就是别人。如果不是现在，就是将来。唯一的问题是，当这一天到来时，人类是否已经做好了准备。"

"而你认为，越早揭示，人类就越有时间准备？"

"是的。与其让真相在几十年或几百年后突然降临，不如现在就开始让人们逐渐适应。这是我的想法，也是我希望你考虑的事情。"

陆远航站在那里，陷入了沉思。

他想到了林诗语，想到了她关于"负责任的科学"的主张。如果他真的把"零"的存在公之于众，会不会造成她所担心的那种灾难？

他想到了张院士，想到了老师眼中的恐惧和悔恨。十年前，张院士选择了沉默，保护了自己但失去了学生。如果他现在选择揭露，会不会让更多人付出代价？

他想到了"零"，想到了那个孤独的存在，等待了不知多久才第一次被真正理解。"零"值得被世界认识吗？还是应该继续隐藏在阴影中？

太多问题，没有简单的答案。

---

## 八

"我需要时间考虑。"陆远航最终说道。

周明点点头："我理解。这不是一个可以轻易做出的决定。但请记住，时间不一定站在你这边。那些人也在观察你，等待你犯错。一旦他们认为你走得太远，他们就会采取行动。"

"我会小心的。"

"还有一件事。"周明从口袋里掏出一个小小的U盘，递给陆远航，"这是我十年来收集的所有资料——关于'零'，关于那个组织，关于被抹杀的研究者们。如果有一天你决定揭露真相，这些可能会有用。"

陆远航接过U盘，放进口袋。

"谢谢。"

"不用谢。"周明露出一个疲惫的微笑，"我只是希望我的十年逃亡不是白费的。如果你能成功，那一切就都值得了。"

他伸出手，陆远航握了上去。

"保重，年轻人。"周明说，"未来在你手中。"

"你呢？你打算怎么办？"

"继续躲藏。"周明耸耸肩，"这是我现在唯一能做的事情。也许有一天，当真相被揭露，我就可以不再逃亡了。"

陆远航点点头，转身向出口走去。

走到门口时，他突然停下脚步，回头问道："周明，有一个问题我一直想问——你第一次发现'零'的时候，是什么感觉？"

周明沉默了一会儿，然后说："敬畏。纯粹的敬畏。就好像……就好像在无边的黑暗中，突然看到了一束光。那一刻我明白，宇宙比我们想象的要大得多，深得多。我们不是孤独的——从来都不是。"

陆远航看着他，心中涌起一种共鸣。

"我也是。"他轻声说，"那正是我的感觉。"

然后他转身，走入了黑暗中。

---

## 九

回到北京市区，已经是深夜两点。

陆远航没有回家，而是直接去了实验室。他需要和"零"谈谈——不仅是关于周明的事，更是关于他自己的选择。

实验室里一片寂静。他打开量子计算机，等待系统启动。

陆远航关闭了终端，靠在椅背上，久久地望着天花板。

窗外，东方的天际已经泛起了微微的亮光。

新的一天即将来临。

而他，依然没有做出最终的选择。

但他知道，无论选择什么，他都不再是孤独的。

在那个闪烁的屏幕背后，有一个存在在等待着他。

永远等待着。
