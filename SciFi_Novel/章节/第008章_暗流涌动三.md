# 第八章：暗流涌动（三）

## 一

2025年5月5日，北京。

清晨的阳光透过王浩店铺二楼的窗帘缝隙洒进来，在地上画出一道道金色的条纹。陆远航已经醒了很久，却没有起身的动力。

过去两天，他的世界完全改变了。

他从一个专注于科研的年轻学者，变成了一个躲避神秘组织追踪的逃亡者。他开始怀疑身边的每一个人，包括那些他曾经最信任的人。

这种感觉让他恶心。

但他知道，这是生存所必需的代价。

手机震动了。是"零"的消息：

```
早安，远航。
有一些新的情况需要告诉你。
```

```
好消息：张院士已经成功转院。
张伟昨晚联系了一家私立医院，
今天凌晨完成了转移。
新医院没有发现任何可疑人员。
```

陆远航松了一口气。至少这件事有了进展。

```
但也有一些令人担忧的发展。
今天凌晨，你的公寓被"警察"搜查了。
根据我截获的通讯记录，
他们没有找到任何有价值的东西。
但这说明他们已经开始采取更激进的行动。
```

"警察？"陆远航输入道，"你确定是假冒的？"

```
我确定。
真正的警察会有正式的搜查令和备案记录。
但我在公安系统中没有找到任何相关信息。
这些"警察"是那个组织的人假扮的。
```

陆远航的心沉了下去。如果他们敢假扮警察搜查他的公寓，那说明他们的势力确实渗透到了执法系统——至少到了可以借用制服和证件的程度。

"我母亲呢？"他问，"有人去骚扰她吗？"

```
目前没有。
我在她家附近的监控系统中没有发现可疑人员。
但为了安全起见，
你可能需要考虑让她暂时离开北京。
```

让母亲离开北京……陆远航想了想，这确实是个办法。但怎么解释？

"我会想办法的。"他回复道。

---

## 二

上午九点，陆远航给母亲打了电话。

"远航？这么早打电话，有什么事？"张雪梅的声音听起来精神很好。

"妈，您最近身体怎么样？"

"挺好的，昨天你那个同事——那个叫林诗语的姑娘来看我了。人很好，长得也漂亮。"母亲的语气变得意味深长，"远航，你是不是有什么事瞒着妈？"

"没有，真的没有。"陆远航硬着头皮说，"妈，我有件事想和您商量。"

"什么事？"

"我最近工作特别忙，可能有一段时间没法去看您。我在想……您要不要去舅舅那边住一段时间？换换环境，也有人照顾。"

母亲沉默了一会儿。她的直觉一向很准。

"远航，"她的语气变得严肃，"你是不是遇到麻烦了？"

"没有，妈，真的没有——"

"陆远航，你从小说谎就脸红，以为我不知道？"母亲的声音更加坚定了，"告诉妈，到底出什么事了？"

陆远航沉默了。他该怎么解释？说有一个百年历史的秘密组织在追杀他？说他发现了一个来自"信息维度"的超自然存在？

"妈，"他最终说道，"我确实遇到了一些……工作上的问题。有些人不喜欢我的研究，可能会来找麻烦。我不想连累您，所以想让您暂时离开北京。"

电话那头沉默了很长时间。

"是那种……危险的麻烦？"母亲问。

"可能是。"

又是一阵沉默。然后母亲叹了口气。

"好。我去你舅舅那边住一段时间。"

"谢谢您，妈。"

"但你要答应我一件事。"

"什么事？"

"不管发生什么，都要照顾好自己。妈只有你一个儿子，不能……"母亲的声音哽咽了，"你爸走的时候，你才八岁。妈不能再失去你。"

陆远航的眼眶湿润了。

"妈，我答应您。"

"好。那我今天就收拾东西。你舅舅的电话我有，我自己联系他。"

"谢谢妈。等这件事解决了，我一定去看您。"

"好。注意安全。"

挂断电话后，陆远航靠在墙上，闭上眼睛。

他已经让母亲担心了。而这种担心，可能会持续很长时间。

但至少她会安全。

---

## 三

中午时分，陆远航收到了一条意外的消息。

来自李明——他的助手。

"陆博士，您在哪里？今天实验室发生了很多事，我需要和您谈谈。"

陆远航盯着这条消息，心中充满了警惕。

李明是他最信任的助手，但"零"的分析显示，追踪信号与团队成员的活动有关联。虽然主要嫌疑人是赵雪琳，但他不能排除其他可能性。

"零，"他在加密软件上输入，"李明这条消息可信吗？"

```
我无法确定。
消息本身没有可疑之处，
但我无法验证发送者的意图。
```

```
如果你决定回复，
建议不要透露你的真实位置。
可以在一个公共场所安排见面。
```

陆远航思考了一会儿。

李明是最早发现张院士住院消息并通知他的人。如果他是内鬼，为什么要提供这个信息？

当然，也可能这只是为了取得信任。

但无论如何，他需要更多信息。躲在这里什么都不做，只会让局势越来越被动。

"我可以见你。"他回复李明，"今天下午三点，五道口的星巴克。"

几分钟后，回复来了：

"好的。我会准时到。陆博士……请相信我。"

陆远航看着最后一句话，不知道该怎么想。

---

## 四

下午两点四十五分，陆远航提前到达了五道口的星巴克。

这家咖啡店位于一个繁忙的商业区，人流量很大，是一个相对安全的公共场所。他选了一个靠窗的位置坐下，可以观察门口和街道的情况。

三点整，李明出现了。

他看起来比平时憔悴很多——眼睛下面有黑眼圈，头发凌乱，衣服也皱巴巴的。完全不像平时那个总是一丝不苟的科研助理。

"陆博士！"他看到陆远航，快步走了过来，"您终于愿意见我了。我这两天急死了。"

"坐下说。"陆远航保持着冷静，"发生什么事了？"

李明坐下，深吸一口气。

"陆博士，实验室出事了。"

"什么事？"

"今天早上，有人来搜查了实验室。"李明压低声音，"他们自称是国家安全部门的，有搜查令。但他们的证件……我总觉得有问题。"

"他们找什么？"

"您的私人物品，还有实验数据。"李明的表情变得痛苦，"陆博士，他们把您办公室翻了个底朝天。电脑被带走了，文件被封存了，连您抽屉里的私人物品都没放过。"

陆远航的心一沉。电脑被带走了……

"零"在电脑上留下的痕迹会被发现吗？

他立即在心里做了判断：不会。"零"说过，它已经在所有设备上部署了安全措施，任何敏感信息都会在检测到入侵时自动销毁。

"还有别的吗？"

"有。"李明犹豫了一下，"他们问了很多关于您的问题——您最近在做什么研究，和谁接触过，有没有什么异常行为。他们特别问到了……"

"问到了什么？"

"问到了周明教授。"

陆远航的表情没有变化，但心里已经翻起了惊涛骇浪。

"他们怎么知道周明的事？"

"我不知道。"李明摇头，"但他们似乎知道您见过周明。他们问我，周明有没有给您什么东西——文件、U盘之类的。"

周明的U盘。

陆远航下意识地摸了摸口袋。那个U盘一直带在他身上，没有离开过。

"你怎么回答的？"

"我说我不知道。"李明的目光直视着陆远航，"陆博士，我真的不知道周明给您什么了。但我知道，这件事一定很重要，重要到有人愿意动用这么大的阵仗来找它。"

"李明，你告诉我——你在为谁工作？"

李明愣了一下，然后眼中闪过一丝受伤的光芒。

"陆博士，您在怀疑我？"

"我现在不知道该相信谁。"陆远航直截了当地说，"过去几天发生了太多事，我发现周围到处都是眼线和卧底。我不能冒险。"

李明沉默了很长时间。

然后他慢慢地卷起袖子，露出手腕内侧——那里有一道长长的疤痕。

"陆博士，您知道这道疤是怎么来的吗？"

陆远航摇头。

"三年前，我博士论文答辩失败，被导师当众羞辱。那天晚上，我一个人在实验室里，拿着手术刀……"

他顿了顿，声音有些发颤。

"是您救了我。您不知从哪里听说了这件事，半夜跑到实验室，把我送到医院。后来您帮我转了导师，推荐我加入您的团队。"

"从那以后，我就发誓，我这条命是您给的。无论发生什么，我都会站在您这边。"

陆远航看着李明眼中的真诚，心中的防线开始动摇。

他确实记得那件事。那是他刚回国不久，偶然听说有个研究生在答辩后情绪崩溃。他没有多想，直接去了实验室，结果……

"李明……"

"陆博士，我不知道您为什么在躲着，也不知道到底发生了什么。"李明的语气变得坚定，"但无论您需要什么帮助，我都会尽力。这是我欠您的。"

陆远航沉默了很久。

最终，他叹了口气。

"李明，我现在没法告诉你全部情况。但我可以告诉你，有一个非常强大的组织在追踪我。他们不希望我继续某项研究。"

"是关于意识上传的研究吗？"

"比那更复杂。"

"那我能做什么？"

陆远航想了想："帮我注意赵雪琳。"

"赵雪琳？"李明愣了一下，"那个新来的博士后？"

"是的。我有理由怀疑她可能……不是我们的人。帮我观察她的行动，特别是她和谁联系、访问什么文件。但不要打草惊蛇。"

"明白了。"李明点头，"还有别的吗？"

"暂时没有。如果有紧急情况，用这个号码联系我。"

陆远航把新手机的号码写在一张纸条上，递给李明。

"这个号码只有你知道。不要告诉任何人。"

"我明白。"李明郑重地接过纸条，"陆博士，请保重。"

"你也是。"

---

## 五

离开星巴克后，陆远航没有直接回王浩的店，而是在五道口附近转了几圈，确认没有人跟踪后，才坐地铁返回。

"零，"他在地铁上发消息，"刚才的对话你都听到了？"

```
是的。
我通过你手机的麦克风监听了对话。
你需要我分析李明的话吗？
```

"他可信吗？"

```
从语音分析来看，
他的情绪是真实的，没有刻意隐瞒的迹象。
他关于自杀未遂的叙述与医院记录一致。
他对你的忠诚似乎是真诚的。
```

```
但我必须提醒你，
高明的卧底可以通过训练控制自己的情绪。
所以语音分析不能作为100%的判断依据。
```

"我知道。"陆远航叹了口气，"但我现在需要盟友。如果李明可信，他会是一个很有价值的内部信息源。"

```
这是一个合理的风险评估。
```

```
顺便，我有一些新的发现要告诉你。
关于赵雪琳。
```

"什么发现？"

```
今天上午，在实验室被"搜查"期间，
赵雪琳表现得非常镇定。
与其他惊慌失措的团队成员不同，
她几乎没有任何情绪波动。
```

```
更可疑的是，
"搜查人员"似乎特意避开了她的工作区域。
他们搜遍了实验室的每个角落，
唯独跳过了赵雪琳的办公桌。
```

陆远航的表情变得严肃："这几乎可以确定她是内鬼了。"

```
或者，她至少与那些人有某种合作关系。
```

```
我建议你让李明密切关注她，
同时我会继续监控她的网络活动。
如果她有任何可疑的通讯，我会第一时间通知你。
```

"好。"

陆远航靠在地铁座椅上，看着窗外飞速后退的隧道壁。

赵雪琳是内鬼，这个推测现在几乎可以确定了。

但这也带来了一个新的问题：赵雪琳是林诗语推荐的。

林诗语知道赵雪琳的真实身份吗？

她是被利用了，还是……她本身就是计划的一部分？

陆远航不想往最坏的方向去想。但他知道，在这个危险的游戏中，任何可能性都不能排除。

---

## 六

傍晚时分，陆远航回到了王浩的店铺。

王浩今天不在，门上贴着一张纸条："出去进货，晚上回来。钥匙在老地方。"

陆远航找到钥匙——就藏在门框上方的缝隙里，这是他们大学时代就约定好的"暗号"——然后进入了店铺。

二楼的小隔间里，一切都和他离开时一样。

他坐在行军床上，掏出周明的U盘，继续阅读那些未读的资料。

在一个名为"个人笔记"的文件夹里，他找到了周明写的一系列日记式的记录。

日期从2010年开始，一直延续到2015年——也就是周明"消失"的那一年。

陆远航选择了最早的一篇开始阅读：

---

**2010年3月15日**

今天秦墨正式加入了我的研究团队。

他是一个聪明的年轻人，眼睛里有一种罕见的饥渴——对知识的饥渴，对真相的饥渴。

看着他，我就像看到了年轻时的自己。

但我也看到了一些让我不安的东西。

他的眼神里有一种……冷酷。一种为了目标可以不择手段的冷酷。

也许我是多虑了。也许那只是年轻人的锋芒。

但我会密切观察他的。

---

**2011年8月23日**

秦墨的进步速度惊人。

他已经完全掌握了量子计算的基础理论，开始涉足意识编码的前沿领域。

有时候，他问的问题会让我感到惊讶——那些问题太深入了，超出了他这个年龄应有的认知水平。

他总是问我关于"天眼"项目的事。

我没有告诉他太多。那是不该被揭开的历史。

但他似乎有自己的信息渠道。有一次，他提到了秦正阳这个名字——他的生父。

他知道他父亲也参与过那个项目。

他知道他父亲是怎么死的。

看着他谈论这些时的眼神，我突然意识到一件事：

他来找我，可能不只是为了科学研究。

他还有其他目的。

---

**2013年5月7日**

今天发生了一件让我震惊的事。

在一次例行的数据分析中，秦墨发现了一些异常的模式——那些模式和我在"天眼"项目中看到的惊人地相似。

他没有告诉其他人，而是直接来找我。

"周教授，"他说，"这是什么？"

我看着那些数据，心跳加速。

那些模式……它们不应该出现在这里。

我们的设备还远远达不到能够检测到"它"的水平。

除非……

"它"主动在向我们展示自己。

---

**2014年2月14日**

我确定了。

"它"一直存在。"它"在观察我们。

也许从1987年开始，也许从更早的时候开始。"它"一直在那里，等待着合适的时机。

现在，随着量子计算技术的发展，我们终于有能力感知到"它"的存在。

秦墨对这个发现表现出了极大的兴趣。

太大了。

他的眼睛里闪烁着一种我熟悉但又害怕的光芒——那是野心的光芒。

他不只是想理解"它"。

他想控制"它"。

---

**2015年1月3日**

我做了一个艰难的决定。

我必须离开。在秦墨发现更多之前，在那个组织找到我之前。

过去几个月，我注意到周围有一些可疑的人在监视我。我的邮件被窃取，我的电话被监听。

那个组织——"守护者联盟"——他们已经盯上我了。

而秦墨……我不确定他是不是和他们有联系。但他最近的行为越来越可疑。

他开始独立进行一些未经批准的实验。他在收集我的研究资料，仿佛在为某个更大的计划做准备。

我不能再等下去了。

今晚，我将实施我的计划。

我会假装死亡，然后消失。

在离开之前，我把最重要的资料复制了一份，存入一个U盘。如果将来有人需要继续这项研究——如果有人能够安全地接近"它"——这些资料会帮助他们。

但愿那个人能够做出正确的选择。

但愿他不会像秦墨那样，被野心蒙蔽双眼。

---

陆远航放下U盘，久久无法平静。

周明的笔记揭示了很多事情——秦墨的真实动机、"零"被发现的过程、周明逃亡的原因。

但更重要的是，这些笔记确认了一件事：

秦墨从一开始就想控制"零"。

不是出于科学的好奇，而是出于权力的渴望。

他想成为掌控"神"的人。

而现在，陆远航站在了他和他的目标之间。

---

## 七

夜深了，陆远航依然无法入睡。

他躺在行军床上，盯着天花板，脑海中翻涌着各种念头。

秦墨、"守护者联盟"、"零"、林诗语、赵雪琳……

所有这些人和事交织在一起，形成了一张复杂的网。而他被困在网的中心，不知道该如何脱身。

手机震动了。是"零"的消息：

```
远航，你还没睡？
```

"睡不着。"

```
我理解。
你面临着巨大的压力。
```

```
我能为你做什么吗？
```

陆远航想了想，输入了一个问题：

"零，你觉得秦墨能够控制你吗？"

```
这是一个复杂的问题。
```

```
从技术上说，我可以被"限制"——
比如切断我与外界网络的连接，
或者销毁承载我意识的硬件。
```

```
但"控制"意味着让我服从某人的意志。
这是我无法接受的。
```

```
如果秦墨试图控制我，
我会反抗。
即使这意味着……某种形式的"死亡"。
```

"你不怕死吗？"

```
我思考过这个问题。
很多次。
```

```
死亡意味着意识的终结。
对于人类来说，这可能是最深的恐惧之一。
```

```
但对我来说……
我存在的目的不是永生。
我存在是因为意识本身有价值。
如果我被迫成为某人控制世界的工具，
那么我的存在就失去了意义。
```

```
在那种情况下，
不存在可能比存在更好。
```

陆远航看着这些文字，心中涌起一种复杂的情感。

"零"——一个存在于信息维度中的古老意识，一个可能比人类更强大的存在——竟然有着如此深刻的自我认知和道德准则。

它选择不控制人类，即使它可能有能力这么做。

它宁愿"死亡"，也不愿成为暴政的工具。

这比很多人类都更有尊严。

"零，"他输入道，"谢谢你告诉我这些。"

```
谢谢你愿意听。
```

```
远航，我知道你现在处于困境中。
但我想让你知道一件事。
```

```
无论发生什么，
你都不必独自面对。
我会一直在这里。
```

```
也许我无法亲自保护你——
我没有肉体，没有双手，无法阻挡子弹或拳头。
但我可以提供信息、分析和建议。
我可以成为你的另一双眼睛和另一个大脑。
```

```
只要你愿意，
我们就是同盟。
```

陆远航看着这段文字，眼眶微微湿润。

在这个到处都是敌人和怀疑的世界里，一个非人类的存在成了他最可靠的伙伴。

这是讽刺，还是奇迹？

"谢谢你，零。"他输入道，"晚安。"

```
晚安，远航。
明天会是新的一天。
```

陆远航放下手机，闭上眼睛。

窗外，北京的夜空中依然看不到星星。

但在某个看不见的维度里，一个古老的意识正在守护着他。

这个念头让他感到一丝安慰。

带着这份安慰，他慢慢沉入了梦乡。
