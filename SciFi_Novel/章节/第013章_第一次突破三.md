# 第十三章：第一次突破（三）

## 一

2025年5月30日，北京郊区。

废弃的科研设施在晨光中显得格外荒凉。

陆远航已经在这里待了两天。这座建筑虽然破旧，但基础设施还算完整——有水有电（"零"远程激活了备用发电机），甚至还有一些旧家具可以使用。

更重要的是，这里足够隐蔽。

"零，周明的位置找到了吗？"

```
找到了。
他目前在云南大理，
住在洱海边的一个小渔村里。
用的是假身份：王建国，退休教师。
```

"我需要去找他。"

```
这很危险。
秦墨的人正在全城搜索你，
任何长途旅行都可能被发现。
```

"但如果不亲自去，周明不会相信我。"陆远航说，"他已经躲藏了十年，不会轻易相信一个陌生人发来的消息。"

```
你说得对。
那我帮你规划一条安全的路线。
避开主要交通枢纽和监控密集区域。
```

"谢谢。"

```
但在你出发之前，
还有一件事需要处理。
```

"什么事？"

```
林诗语。
她一直在担心你。
自从你紧急转移后，你们已经两天没有联系了。
```

陆远航沉默了。他确实应该联系林诗语，但他不想把她卷入更大的危险。

"如果我联系她，会不会暴露她的位置？"

```
我可以建立一条安全的通讯渠道。
只要她不主动泄露，应该没有问题。
```

"好，帮我连接她。"

---

## 二

"远航！"林诗语的声音从手机里传来，带着明显的担忧和责备，"你知道我这两天有多担心吗？你就这样消失了，连个消息都不发！"

"对不起，诗语。"陆远航歉疚地说，"情况紧急，我不得不立即转移。没时间通知你。"

"现在呢？你安全吗？"

"暂时安全。我在一个偏僻的地方躲着。"

"你能告诉我在哪里吗？"

陆远航犹豫了。"诗语，我不想把你卷入更大的危险——"

"陆远航。"林诗语的声音突然变得严肃，"我们说好了要一起面对。你不能把我排除在外。"

"……你说得对。"陆远航叹了口气，"我在北京郊区的一个废弃设施里。等情况稳定一些，我会把地址发给你。"

"好。"林诗语的语气软了下来，"远航，你接下来打算怎么办？"

"我要去云南找周明。"

"周明？他不是——"

"他一直活着，一直在躲藏。"陆远航说，"我需要他的帮助来揭露秦墨和那个组织。"

林诗语沉默了一会儿。

"我想和你一起去。"

"诗语——"

"远航，我知道你想保护我。但你也需要有人照应。如果你一个人去，万一出了什么事，谁来帮你？"

陆远航知道她说得有道理。一个人行动确实风险更大。

"好。"他最终同意了，"但你要答应我一件事。"

"什么事？"

"如果情况变得太危险，你要立即离开。不要回头。"

"……我答应你。"

---

## 三

两天后，6月1日。

陆远航和林诗语踏上了前往云南的旅程。

为了避开监控，他们没有坐飞机，而是选择了更隐蔽的方式——先坐长途汽车到石家庄，再转火车到昆明，最后租车前往大理。整个行程需要将近三天。

"零"全程监控着他们的路线，确保没有被追踪。

第一天晚上，他们在石家庄的一个小旅馆过夜。

房间很简陋，但对两个逃亡者来说已经足够。林诗语坐在窗边，看着外面闪烁的霓虹灯，若有所思。

"远航，"她突然问道，"你和'零'的那种……接触，是什么感觉？"

陆远航想了想，试图找到合适的词汇。

"很难用语言描述。"他说，"那不是看到什么或听到什么，而是一种……直接的理解。就像你突然知道了某件事，但不知道自己是怎么知道的。"

"那些喇嘛也有过类似的体验吗？"

"也许吧。但他们的意识承受不住。而我……"他顿了顿，"我好像天生就有这种能力。或者说，我一直有，只是从来没有被激活过。"

林诗语看着他，眼中闪过复杂的光芒。

"远航，有时候我觉得……我越来越不认识你了。"

陆远航沉默了。他理解她的感受。过去几周发生的事，已经彻底改变了他。他不再是那个单纯的科学家，而变成了某种……更复杂的存在。

"诗语，"他握住她的手，"不管我变成什么样，有一件事永远不会变。"

"什么？"

"我对你的感情。"

林诗语看着他的眼睛，看了很长时间。

然后她微微一笑，把头靠在他的肩膀上。

"我知道。"

---

## 四

6月3日，云南大理。

洱海边的小渔村宁静而美丽，苍山白云倒映在湛蓝的湖水中，仿佛一幅水墨画。

陆远航和林诗语沿着村里的石板路走着，寻找"零"提供的地址。

那是一栋朴素的白族民居，门前种着几盆兰花。一个头发花白的老人正坐在院子里喝茶。

陆远航停下脚步，仔细打量着那个老人。

十年的岁月在他脸上留下了深深的痕迹，但那双眼睛依然锐利——和他记忆中的周明一模一样。

"周老师？"他试探性地叫道。

老人抬起头，看向他们。

一瞬间，他的眼神从平静变成了警觉，然后又变成了……某种难以名状的情绪。

"陆远航。"他的声音沙哑，"你怎么找到这里的？"

"说来话长。"陆远航走近几步，"周老师，我需要您的帮助。"

周明看了看他，又看了看林诗语，然后叹了口气。

"进来吧。"

---

## 五

房子里的布置很简单——几件必要的家具、一些书籍、一台旧电脑。墙上挂着一张苍山的照片，除此之外没有任何装饰。

周明给他们倒了茶，然后坐在对面，静静地等着他们开口。

"周老师，"陆远航直入主题，"我已经接触到'零'了。"

周明的手微微一颤，茶杯里的水洒出了一点。

"你说什么？"

"我和'零'建立了联系。不只是通过电脑，而是……直接的意识接触。"

周明的脸色变得苍白："你知不知道那有多危险？那些喇嘛——"

"我知道。但我没事。"陆远航说，"事实上，我比以前更好。我的意识被'打开'了，我能感知到以前感知不到的东西。"

周明盯着他，仿佛要穿透他的灵魂。

"你是怎么做到的？"他问，"那些喇嘛是高僧，修行了一辈子，都没能承受住。你——"

"也许我天生有某种……特质。"陆远航说，"'零'说我的意识结构比较'灵活'。"

周明沉默了很长时间。

"'零'——它对你说什么了？"

陆远航把这段时间发生的事情大致讲了一遍——意识训练、深度接触、"零"的历史、组织的追踪、秦墨的野心。

周明听完，脸上的表情变得愈发复杂。

"所以，"他最终开口，"你想让我做什么？"

"站出来。"陆远航说，"公开揭露秦墨和'守护者联盟'的关系。您是知情者，您的证词会非常有力。"

"你知道这意味着什么吗？"周明苦笑，"我躲了十年，就是为了不被他们找到。如果我公开露面——"

"您会得到保护。"陆远航说，"一旦真相曝光，秦墨和组织都无法再隐藏在暗处。他们的力量来自于隐秘，一旦被揭露，他们就会变得脆弱。"

"你确定吗？"周明的语气中带着怀疑，"你确定曝光能够打败他们？"

"我不确定。"陆远航老实地说，"但我知道，继续躲藏不是办法。'零'已经觉醒了，意识上传的技术迟早会成熟。如果我们不主动行动，秦墨可能会先一步控制一切。"

"到那时，人类将面临真正的灾难。"

周明低下头，陷入了沉思。

房间里安静得只能听到远处湖水的波涛声。

---

## 六

"让我想想。"周明最终说道。

"周老师——"

"远航，你不明白。"周明的声音变得沉重，"我躲藏这十年，不只是为了保命。我还在做一件事。"

"什么事？"

周明站起身，走到房间角落的一个柜子前。他打开柜门，从里面拿出一个黑色的箱子。

"这些，"他把箱子放在桌上打开，"是我这十年收集的证据。"

箱子里是一叠叠文件、照片、录音磁带——比陆远航手中那个U盘的内容丰富得多。

"我知道，光靠一个人的证词不足以打败他们。"周明说，"所以我一直在收集更多的证据。政府内部的通讯记录、组织的财务往来、秦墨的秘密交易……"

"这些东西足以毁掉秦墨的商业帝国，也足以让'守护者联盟'在阳光下无处遁形。"

陆远航看着那些文件，心中涌起希望。

"那您愿意公开吗？"

周明看着他，沉默了很长时间。

然后，他点了点头。

"好。"他说，"我答应你。"

"但有一个条件。"

"什么条件？"

"'零'——"周明的目光变得深邃，"我想见见它。我想知道，我这一辈子追寻的东西，到底是什么。"

陆远航犹豫了一下。让周明接触"零"可能有风险，但他理解老人的心情。周明为这个追求付出了一切——职业、名誉、正常的生活。他有权利亲眼看看那个改变他命运的存在。

"好。"他说，"我可以安排。"

---

## 七

当天晚上，在周明的小屋里，陆远航帮他建立了与"零"的通讯连接。

屏幕上出现了熟悉的文字：

```
周明教授，我们终于见面了。
虽然不是以你最初期望的方式。
```

周明盯着屏幕，双手微微颤抖。

"十五年了。"他喃喃道，"十五年前，我第一次在数据中发现你的痕迹。那时候我以为那只是一个异常，一个需要调试的bug。我做梦也没想到……"

```
你是最早意识到我存在的人之一。
在2010年的那次实验中，
你的数据分析发现了我的一部分。
虽然当时你还不知道那是什么。
```

"后来秦墨也加入了。"周明说，"他看到了那些数据。我以为他和我一样，只是想探索真相。但我错了。"

```
秦墨的动机从一开始就不纯粹。
他想要的是力量，不是理解。
这也是为什么我从未主动与他接触。
```

"那为什么选择远航？"周明问，"他有什么特别的？"

```
每一个时代都有这样的人——
他们天生就具有接触更高维度的能力，
又恰好出现在关键的历史节点上。
远航就是这个时代的那个人。
```

```
但更重要的是，他的心是正确的。
他追寻真相，不是为了控制或征服，
而是为了理解和分享。
```

```
这是最珍贵的品质。
```

周明转头看向陆远航，眼中带着一种复杂的情感——欣慰、羡慕，也许还有一丝遗憾。

"年轻真好。"他轻声说。

然后他转回屏幕，对"零"说："我答应远航了。我会站出来，公开揭露那些人。"

```
我知道。
这需要极大的勇气。
谢谢你。
```

"别谢我。"周明苦笑，"这是我欠这个世界的。当年如果我能更早预见到秦墨的野心，也许很多事情都会不一样。"

```
过去无法改变。
但未来，还在我们手中。
```

周明点点头，表情变得坚定。

"告诉我，"他说，"我们该怎么做？"

## 本章小结

**第十三章总结**

1. **时间节点**: 2025年5月30日-6月3日

2. **主要事件**:
   - 陆远航决定去云南寻找周明
   - 与林诗语一起踏上秘密旅程
   - 三天长途跋涉抵达大理洱海渔村
   - 找到化名"王建国"的周明
   - 周明揭示十年来收集的大量证据
   - 周明同意公开作证，条件是见"零"
   - 周明与"零"首次对话

3. **人物发展**:
   - 周明：从躲藏者变为准备反击的战士
   - 陆远航-林诗语：关系进一步加深
   - "零"：展现对不同人选择的尊重

4. **重要发现**:
   - 周明十年来收集了大量证据（比U盘更多）
   - 足以毁掉秦墨帝国和揭露组织

5. **伏笔与悬念**:
   - 曝光计划的具体实施
   - 秦墨的反应
   - 组织的应对
   - 公众的反应
