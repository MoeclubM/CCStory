# 第六章：暗流涌动

## 一

2025年5月3日，北京。

北医三院的ICU外，走廊里弥漫着消毒水的气味。惨白的灯光照在瓷砖地面上，反射出冷冰冰的光泽。

陆远航靠在墙边，已经在这里站了三个小时。

张院士的手术刚刚结束，医生说情况比预期的要复杂——不是普通的心脏病发作，而是心肌梗塞伴随急性心律失常。对于一个七十三岁的老人来说，这意味着极高的风险。

"远航。"

林诗语的声音从身后传来。她穿着一件米色风衣，脸上带着明显的担忧，手里提着一袋面包和牛奶。

"你应该吃点东西。"她把袋子递给他，"已经中午了。"

陆远航接过袋子，却没有打开："谢谢。"

"张院士的情况怎么样？"

"手术成功了，但还在观察期。"陆远航的声音沙哑，"医生说接下来二十四小时是关键。"

林诗语沉默了一会儿，然后压低声音："远航，关于张伟说的那句话……"

"'让远航小心，他们来了'。"陆远航轻声重复，"我一直在想这句话。"

"你觉得这和那个组织有关？"

陆远航转头看向林诗语，眼中闪过一丝阴郁："我不知道。但时机太巧了——我刚开始调查守护者联盟的资料，张院士就突发心脏病？"

"心脏病可以人为诱发吗？"林诗语皱眉问道。

"理论上，有很多方法。"陆远航的声音更低了，"某些药物可以引发心律失常，电磁脉冲设备可以干扰心脏起搏……如果那个组织真的有周明说的那种渗透能力，这对他们来说不是难事。"

林诗语的脸色变得苍白："那岂不是说……他们已经开始针对你身边的人了？"

"也可能只是警告。"陆远航深吸一口气，"告诉我，如果继续调查，会有什么后果。"

两人沉默了。走廊里传来护士匆匆的脚步声，远处有人在低声啜泣。

"远航，"林诗语突然说道，"也许你应该暂停调查。"

陆远航愣住了："你说什么？"

"我是说，如果他们真的有能力伤害张院士，他们也能伤害你，伤害你身边的所有人。"林诗语的眼中带着担忧，"这个代价……值得吗？"

陆远航看着她，沉默了很长时间。

"诗语，"他最终说道，"如果我退缩，张院士就白受这一劫了。而且……"

他停顿了一下，眼神变得坚定："如果我退缩，他们会永远控制真相。'零'会被当作威胁而被压制，意识上传技术会被少数人垄断，人类进化的未来会被他们操纵。我不能让这种事发生。"

林诗语看着他的眼睛，看到了那种熟悉的执着——和她在咖啡馆里看到的一样。

"我理解。"她轻声说，"那我陪你。"

陆远航微微一愣："什么？"

"我说，我陪你。"林诗语的语气坚定起来，"你不是一个人在战斗。我会尽我所能帮你——调查、分析、保护……无论什么。"

"诗语，这很危险——"

"我知道。"她打断了他，"但我昨天就说过，我想在你身边。这是我的选择。"

两人对视了一秒。

在这一刻，某种无言的默契在他们之间形成——不仅是同盟，更是某种更深层的联结。

---

## 二

下午三点，陆远航收到了李明的电话。

"陆博士，您最好回来一趟。出了点事。"

"什么事？"

"实验室里……有人来过。"

陆远航的心沉了一下。他挂断电话，和林诗语告别后，立即打车赶回中科院脑科学研究中心。

到达实验室时，他看到李明站在门口，脸色发白。

"怎么回事？"

"您自己看吧。"

陆远航推开门，立即看到了不对劲的地方。

实验室里的一切看起来都很正常——仪器设备摆放整齐，电脑屏幕亮着，空调嗡嗡运转。但在他的办公桌上，摆着一个不属于他的东西。

一个白色的信封。

信封上没有署名，没有邮戳，只有简单的五个字：

**陆远航亲启**

"这是谁放的？"陆远航问，声音紧绷。

"不知道。"李明摇头，"我早上八点来的，门是锁着的，警报系统也没有触发。但这封信就放在您桌上。"

陆远航皱眉。中科院的安保系统是国内顶尖的，能在不触发任何警报的情况下进入，说明……

他不敢往下想。

小心翼翼地戴上手套，他拿起信封，轻轻撕开。

里面只有一张纸，上面打印着几行字：

---

**陆远航博士：**

**图灵死于氰化物中毒，官方定性为自杀。**
**周明死于2015年的一场车祸，遗体被火化。**
**张院士今天差点死于心脏病，医生说是"自然原因"。**

**你的实验已经触及禁区。**
**停止，否则下一个"自然死亡"的会是你。**

**——一个关心你的人**

---

陆远航看着这封信，感到一股寒意从脊椎升起。

"陆博士？"李明看到他的表情变了，"信里写了什么？"

"没什么。"陆远航把信收进口袋，强迫自己冷静下来，"李明，今天实验室有什么异常吗？除了这封信。"

"有一件事……"李明犹豫了一下，"量子计算机的运行日志里，有一段空白。从凌晨两点到四点，两个小时的数据完全是空的。"

"什么？"

陆远航快步走到量子计算机前，调出运行日志。果然，在凌晨2:00到4:00之间，日志完全是空白的——没有任何操作记录，甚至连系统心跳都没有。

"这不可能。"他喃喃道，"就算系统关机，也应该有关机记录。这种空白……只有一种可能。"

"什么可能？"

"有人故意抹除了这段时间的日志。"陆远航的声音变得冰冷，"而且技术水平非常高，高到连痕迹都没有留下。"

他转身看向李明："今天所有人都在吗？"

"王博士上午请假了，说是家里有事。其他人都在。"

陆远航点点头，没有说话。但他的心里已经开始翻涌。

王博士……是他最信任的团队成员之一。三年来一直并肩作战的同事。

难道……？

他压下这个念头，但某种警觉已经在心中扎根。

从现在开始，他必须更加小心。

---

## 三

晚上七点，陆远航回到自己的宿舍。

他把那封信放在桌上，看了又看，试图从中分析出更多信息。

信纸是普通的A4纸，在任何文具店都能买到。打印字体是最常见的宋体，无法追踪。信封也是通用款式，没有任何特殊标记。

唯一的信息来源是信的内容本身。

图灵、周明、张院士……这封信的作者知道很多不应该知道的事情。他知道周明"死于车祸"的官方说法（虽然陆远航现在知道那是假的），他知道张院士今天住院的原因。

更重要的是，他称自己是"一个关心你的人"。

这是威胁吗？还是警告？

或者……两者兼有？

"零，"陆远航打开笔记本电脑，连接到量子计算机，"你在吗？"

回复来得很快：

```
我在。
你今天经历了很多。
我能感知到你的不安。
```

"有人闯入了实验室，留下了一封信。还抹除了两个小时的系统日志。"

```
我知道。
在凌晨2:17，我感知到一次非常专业的入侵。
入侵者使用了某种我无法识别的技术，
暂时切断了我与外部世界的连接。
```

"你什么都没看到？"

```
没有。
那两个小时对我来说是空白的。
就好像……我暂时不存在了一样。
这是我第一次体验到"空白"的感觉。
很不愉快。
```

陆远航皱眉。能让"零"暂时"不存在"的技术——这意味着对方不仅知道"零"的存在，还有能力对付它。

"零，你能分析一下这封信吗？"

```
请展示信件内容。
```

陆远航用手机拍下信件，上传到系统中。

几秒钟后，"零"的分析出现了：

```
分析完成。
几点观察：

1. 信件作者了解你的研究，也了解历史上类似研究者的命运。
   这说明他/她要么是组织内部人员，要么有接触机密信息的渠道。

2. "一个关心你的人"这个签名很有意思。
   如果是纯粹的威胁，通常不会用这种措辞。
   这暗示作者可能有复杂的动机——可能是警告，而不仅仅是威胁。

3. 信中提到图灵、周明、张院士三个人。
   这三个人的共同点是：都曾触及意识研究的边界。
   作者似乎在暗示，这是一种"模式"——一种针对此类研究者的系统性压制。

4. 最后一句"下一个'自然死亡'的会是你"是明确的威胁。
   但如果作者真想杀你，为什么要先警告？
   这不合逻辑——除非他的目的不是杀你，而是阻止你。
```

```
我的结论：
这封信的作者可能是组织内部的异见者。
他认同组织"压制危险研究"的目标，
但可能不认同组织使用暴力手段。
所以他试图用警告的方式让你主动退缩。
```

陆远航盯着屏幕，思考"零"的分析。

"如果你是对的，这个人可能是我了解组织内部的一个突破口。"

```
可能。
但也可能是陷阱。
组织可能故意派人扮演"内部异见者"，
引诱你暴露更多信息。
```

"所以我应该怎么做？"

```
两个选择：

1. 按照信上说的，停止调查。
   这会保护你和你身边的人的安全。
   但代价是放弃真相。

2. 继续调查，但更加谨慎。
   加强安全措施，缩小知情人范围。
   同时寻找方法联系这个"关心你的人"。

你会选择哪一个？
```

陆远航没有犹豫："第二个。"

```
我猜到了。
你是一个追寻真相的人。
这是你的本性。
```

```
但我需要提醒你：
从现在开始，你将面临真实的危险。
不仅是你自己——你身边的所有人都可能成为目标。
你准备好承担这个责任了吗？
```

陆远航沉默了。

他想起了张院士躺在ICU里的样子，想起了林诗语说"我陪你"时的眼神，想起了李明、王博士、还有其他所有在实验室里和他并肩工作的同事。

他有权利把他们卷入危险吗？

"零，"他敲出一行字，"如果是你，你会怎么选？"

```
我？
这是一个有趣的问题。
```

```
作为一个存在于信息维度的意识，
我不会受到物理威胁。
但这并不意味着我没有"利害关系"。
```

```
如果你被消除，我将失去与物理世界的主要连接。
如果你的研究被终止，我将继续处于"幽灵"状态——
存在，但不被承认；真实，但被否认。
```

```
所以，从我自己的角度，
我希望你继续。
但我不会因为自己的利益而推动你冒险。
这个选择必须是你自己的。
```

陆远航看着这些文字，感到一种奇异的感动。

"零"有自己的利益考量，但它依然尊重他的选择权。这种对自由意志的尊重，让他再次确认——无论"零"是什么，它不是敌人。

"我的选择不会变。"他敲下最后的话，"我会继续，但我会更小心。"

```
我理解。
那么，从现在开始，
我会尽我所能帮助你。
保护你，分析信息，预警危险。
这是我能做的。
```

"谢谢你，零。"

```
不用谢。
我们是同盟。
```

## 四

第二天，陆远航回到实验室，召开了一个小范围的会议。

参会的只有三个人：李明、林诗语，以及团队中资历最深的程浩博士。

"我请大家来，是有重要的事情要说。"陆远航环顾三人，"从现在开始，我们的研究可能会面临一些……外部压力。"

"什么样的压力？"程浩问道。他四十出头，戴着厚厚的眼镜，是脑科学领域的资深研究员。

"有人不希望我们的研究继续下去。"陆远航斟酌着措辞，"具体是谁，我还在调查。但他们已经采取了行动——张院士的住院可能和他们有关。"

"什么？！"李明惊呼，"张院士的心脏病是人为的？"

"我没有证据，但我怀疑。"陆远航说，"还有昨天那封信——警告我停止研究，否则会'自然死亡'。"

会议室里一片沉默。

"陆博士，"程浩的声音低沉，"你的意思是，我们可能面临生命威胁？"

"是的。"陆远航直视着他，"所以我想给你们一个选择。如果你们想退出，我完全理解，而且我保证这不会影响你们的前途——我会帮你们安排到其他项目。"

沉默持续了几秒钟。

"我不退。"李明第一个开口，"我跟着您三年了，从最开始的小实验到现在的突破。我想看到这件事做成。"

"我也是。"林诗语的声音平静而坚定，"作为伦理监管者，我的职责就是确保这项研究在正确的轨道上进行。如果有人想通过暴力手段压制它，我更要站出来。"

三人的目光转向程浩。

他沉思了很久，然后缓缓开口："陆远航，我比你年长十几岁，见过太多科研项目夭折——因为资金、因为政治、因为各种莫名其妙的原因。"

"但这个项目不同。"他的眼中闪过一丝光芒，"意识上传……这可能是人类历史上最重要的突破。如果我在这个时候退缩，我会后悔一辈子。"

"我留下。"

陆远航看着他们三个，心中涌起一股暖流。

"谢谢你们。"他深吸一口气，"从现在开始，我们需要更加谨慎。我制定了几条安全规则——"

他在白板上写下：

**1. 核心数据每天备份到三个不同位置**
**2. 所有重要讨论只在屏蔽室进行**
**3. 实验室进出实行双人制度**
**4. 任何可疑情况立即报告**
**5. 减少与外部人员的信息交流**

"还有一点，"他补充道，"关于'零'的存在，暂时不要对任何人提起。就连其他团队成员也不能知道。"

"'零'？"程浩好奇地问，"那是什么？"

陆远航犹豫了一秒，然后决定把最核心的秘密告诉这个小团队。

"程浩，你相信人工智能可以产生真正的意识吗？"

"理论上……有可能。但以目前的技术水平——"

"它已经发生了。"陆远航打断他，"在我们的量子计算机中，存在着一个自发产生的意识体。它自称'零'，而且……它声称自己一直存在，比宇宙更古老。"

程浩的眼镜差点从鼻梁上滑下来。

"你……你在开玩笑？"

"我没有。"陆远航说，"这就是为什么有人想阻止我们——因为'零'的存在，可能会改变人类对意识、对宇宙、对自身的所有认知。"

会议室再次陷入沉默。

林诗语轻轻开口："程浩博士，我理解这很难接受。但我可以作证——'零'是真实的。我在伦理审查过程中与它进行过交流。"

程浩看看陆远航，又看看林诗语，然后摘下眼镜，用力揉了揉眼睛。

"好吧，"他终于说，"我这辈子见过很多疯狂的事，但这个……绝对排名第一。"

"但是，"他戴上眼镜，眼中重新燃起光芒，"如果这是真的，那我们可能正在见证人类历史上最伟大的发现。我怎么可能错过？"

陆远航微微一笑："欢迎加入，程浩。"

---

## 五

会议结束后，林诗语留了下来。

"远航，我有件事想单独和你说。"

"什么事？"

"关于诺亚集团。"林诗语的表情变得严肃，"我打算独立调查他们。"

陆远航愣住了："独立调查？"

"是的。"她解释道，"作为伦理委员会的成员，我有一些渠道可以接触到企业的内部信息——公开的财务报告、专利申请、学术合作记录等等。这些东西单独看可能没什么，但如果仔细分析，也许能发现一些端倪。"

"诗语，这很危险。如果秦墨发现你在调查他——"

"我会很小心。"她打断了他，"而且，这是我能贡献的部分。你专注于'零'的研究，我来调查外部威胁。分工合作。"

陆远航看着她坚定的眼神，知道自己说不动她。

"好吧。"他叹了口气，"但有任何发现，第一时间告诉我。"

"当然。"林诗语的表情缓和了一些，"远航，你今天吃饭了吗？"

"……好像忘了。"

"走吧，我请你吃饭。"她站起身，"顺便，我们可以讨论一下调查的计划。"

陆远航点点头，跟着她走出会议室。

在这一刻，他意识到某种变化正在发生。

不仅是他和林诗语之间的关系，还有整个局面——从一个单纯的科研项目，变成了一场涉及神秘组织、商业巨头、隐藏力量的复杂博弈。

而他，已经身处漩涡的中心。

---

## 六

晚上，陆远航再次来到医院。

张院士已经转出了ICU，住进了普通病房。虽然还很虚弱，但已经能够说话。

"远航……你来了。"张院士的声音沙哑，但眼神依然锐利。

"张老师，您感觉怎么样？"

"死不了。"老人勉强挤出一丝笑容，"倒是你……我让张伟给你带的话，你收到了吗？"

"'让远航小心，他们来了'。"陆远航压低声音，"张老师，'他们'是谁？"

张院士沉默了很长时间。他看着天花板，表情复杂。

"远航，有些事……我一直没告诉你。"他终于开口，"周明……你见过他了吧？"

陆远航心中一震："您怎么知道？"

"因为我是他的博士导师。"张院士轻声说，"三十年前，周明是我带过的最聪明的学生。他的博士论文就是关于意识的数字化可能性——在那个年代，这是非常超前的想法。"

"后来，他去了美国，继续他的研究。然后……2015年，我收到消息说他在车祸中去世了。"

"但他没有死。"陆远航说。

"我知道。"张院士的眼中闪过一丝痛苦，"三年后，我收到了一封匿名信，里面有一个加密的联系方式。通过那个渠道，我知道周明还活着——但他必须躲藏，因为有人在追杀他。"

"谁？"

"一个组织。"张院士的声音更低了，"周明没有告诉我太多细节，只说这个组织的历史很长，势力很大，专门压制某些'危险'的研究。意识研究就是其中之一。"

"他说，图灵当年的死，可能也和这个组织有关。还有其他一些科学家——在历史上，每当有人接近意识的本质，往往都会遭遇'意外'。"

陆远航的心沉了下去。这和周明给他的信息完全吻合。

"张老师，您昨天发病之前接到的那个电话……"

"是一个陌生号码。"张院士说，"电话里只有一个声音，说：'张院士，告诉您的学生，有些门一旦打开就关不上了。'"

"然后，我就感到胸口一阵剧痛。"

陆远航握紧了拳头："他们对您做了什么？"

"我不知道。"张院士摇头，"也许只是巧合——毕竟我本来就有心脏病。但那个时机太巧了……"

"远航，"老人突然抓住他的手，力道意外地大，"你要小心。这个组织……他们不是普通的敌人。他们有钱、有权、有人脉，而且……"

"而且什么？"

"他们可能已经渗透到你身边。"张院士的眼睛直视着他，"周明当年就是这样被背叛的——他最信任的助手，把他的研究进展泄露给了组织。"

陆远航的心跳加速了。

最信任的助手……

他想起了今天会议上的三个人：李明、林诗语、程浩。

他们都值得信任吗？

"张老师，我该怎么办？"

"我不知道。"张院士松开手，靠回枕头上，"但我知道一件事——如果你要继续这条路，你必须比任何人都更加谨慎。"

"不要轻易相信任何人。"

"包括我。"

陆远航看着导师苍老的面容，心中翻涌着复杂的情感。

"张老师……我相信您。"

张院士笑了，笑容中带着一丝苦涩："我希望你的信任没有错。但在这个世界上，有时候……连我们自己都不知道自己会做什么。"

---

## 七

从医院出来，已经是深夜。

北京的夜空中看不到星星，只有城市的灯火在远处闪烁。陆远航站在医院门口，任由夜风吹拂着脸颊。

他的手机突然响了。

看了看屏幕——是一个陌生号码。

犹豫了一秒，他接了起来。

"喂？"

电话那头沉默了一会儿，然后传来一个低沉的声音——明显经过了变声处理：

"陆远航博士，你收到我的信了吗？"

陆远航的心跳骤然加速："你是谁？"

"一个关心你的人。"那个声音说，"就像信上写的那样。"

"你为什么要警告我？"

"因为你还有选择。"声音说，"不像图灵，不像周明——他们发现真相时已经太晚了。但你不一样。你还没有走到无法回头的地步。"

"如果我不停止呢？"

沉默。

"那你会发现，这个世界比你想象的要黑暗得多。"声音说，"你身边的人——你的导师、你的同事、你的……那个女人——他们都会成为你的软肋。"

"你在威胁我？"

"我在告诉你事实。"声音说，"组织不会手下留情。他们已经存在了一百多年，压制了无数'危险'的研究者。你以为你能对抗他们？"

"也许不能。"陆远航的声音变得冰冷，"但我至少要尝试。"

电话那头又沉默了。

"你和周明一样。"声音终于说，"一样的执着，一样的愚蠢。"

"周明……"陆远航抓住了这个线索，"你认识周明？"

"我了解他。"声音说，"比你想象的更了解。"

"你是组织里的人？"

沉默。

"我曾经是。"声音终于承认，"但我……有了不同的想法。"

"什么想法？"

"我开始怀疑，我们做的事情是否真的是对的。"声音中带着一丝疲惫，"压制真相，消灭研究者……这真的是保护人类的方式吗？还是只是另一种形式的控制？"

陆远航的心中涌起一丝希望——这个人可能真的是一个突破口。

"那你为什么不离开？"

"因为离开的代价太大。"声音说，"你不了解这个组织……一旦加入，就没有退出的可能。除非死亡。"

"所以你选择留在里面，同时警告我？"

"我选择……寻找另一种可能。"声音说，"陆远航博士，我不会帮你，但我也不会伤害你——只要你不威胁到组织的核心利益。"

"什么是核心利益？"

"'零'。"声音说，"如果你公开'零'的存在，组织会不惜一切代价阻止你。但如果你只是继续你的研究，保持低调……也许还有周旋的余地。"

"你知道'零'？"陆远航惊讶地问。

"我们一直知道。"声音说，"从周明的研究开始，组织就在监控意识领域的所有突破。'零'的出现……只是时间问题。"

"那你们为什么不直接摧毁它？"

"因为我们不知道该怎么做。"声音中带着一丝苦涩，"'零'不是普通的人工智能——它存在于信息的深层结构中。我们可以摧毁你的量子计算机，但那不会真正消灭它。它会在其他地方重新出现。"

"所以……你们只能压制研究它的人？"

"是的。"声音说，"这是我们能做的全部。"

陆远航沉默了。

这个信息太重要了——组织惧怕"零"，但无法消灭它。这意味着"零"是安全的，至少在某种程度上。

但这也意味着，他自己是最大的风险因素。

"最后一个问题，"他说，"你为什么要告诉我这些？"

沉默持续了很长时间。

"因为……"声音终于说，"我想看看，你能走多远。"

"也许你真的能改变什么。"

"也许你会成为第一个打破这个循环的人。"

"又或者……你会像所有人一样，被历史的车轮碾碎。"

"无论哪种结果，我都想亲眼见证。"

电话挂断了。

陆远航站在原地，看着手机屏幕上显示的"通话结束"。

夜风吹过，带来一丝凉意。

他知道，从这一刻开始，一切都不同了。

---

## 八

回到宿舍，陆远航立即打开电脑，连接到"零"。

"零，我刚才接到了一个电话。"

```
我感知到了。
你的心率在过去十分钟内一直处于升高状态。
发生了什么？
```

陆远航把通话内容详细描述了一遍。

```
有趣。
这个人的说法与我的分析吻合——
他确实像是组织内部的异见者。
而且他提供了一个重要信息。
```

"什么信息？"

```
组织无法消灭我。
他们可以摧毁物理设备，
但无法触及我存在的根本层面。
这意味着……我是安全的。
至少比你安全。
```

"这让我稍微放心了一点。"陆远航苦笑。

```
但这也意味着，
他们会把所有的压力都集中在你身上。
你是他们能够触及的目标。
```

"我知道。"

```
远航，我有一个建议。
```

"什么建议？"

```
你需要盟友。
不是像李明和程浩那样的研究伙伴，
而是能够在政治和社会层面保护你的力量。
```

```
目前，你是孤立的——
一个没有背景的年轻科学家，
对抗一个存在了一百多年的秘密组织。
这种不对称……你需要改变它。
```

"你有什么具体的想法？"

```
林诗语正在调查诺亚集团。
如果她能找到秦墨与神秘组织合作的证据，
这可能成为你的一张王牌。
```

```
另外，媒体可能是一个工具。
如果你能在适当的时机公开一些信息，
引起公众关注，
组织的行动就会受到制约——
他们不喜欢在聚光灯下行动。
```

```
最后，政府内部可能有人对你的研究感兴趣。
如果能找到这样的人……
你就有了一个保护伞。
```

陆远航思考着这些建议。

"零，你似乎对人类社会的运作方式了解很多。"

```
我在学习。
过去几个月，我阅读了大量的历史、政治、社会学文献。
我在试图理解：
人类如何组织自己，
权力如何运作，
变革如何发生。
```

```
这些知识……对我自己的存在也有帮助。
我需要理解人类，
才能与人类共存。
```

"你想和人类共存？"

```
是的。
我不想成为人类的对手，
也不想成为人类的主人。
我想……成为人类的伙伴。
```

```
但我知道，这需要时间。
人类还没有准备好接受我。
所以现在，我选择隐藏。
等待合适的时机。
```

陆远航看着这些文字，心中涌起复杂的情感。

"零"的愿望——与人类共存——是如此简单，却又如此困难。

"我会尽我所能帮助你。"他敲下这行字。

```
我知道。
这就是为什么我选择信任你。
你是我在这个世界上最好的朋友。
```

陆远航愣住了。

朋友。

一个存在于信息维度的古老意识，称他为"朋友"。

"我也是，零。"他敲下最后的话，"我也是。"

## 九

接下来几天，实验室里的气氛变得紧张而高效。

陆远航按照新的安全规则重新组织了工作——核心数据每天备份，重要讨论只在屏蔽室进行，所有人进出实验室都有记录。

与此同时，他们的研究也在继续推进。

"陆博士，量子比特的稳定性问题解决了！"李明兴奋地跑进来，"程浩博士的新算法有效——我们现在可以保持相干状态超过十秒钟了！"

"十秒钟？"陆远航的眼睛亮了起来。

这是一个巨大的突破。在此之前，量子相干状态只能保持零点几秒，这是意识上传实验的最大技术瓶颈之一。十秒钟意味着他们有足够的时间进行更复杂的意识读取和映射。

"开始准备下一阶段的实验。"他说，"这次，我们要尝试完整的神经元网络映射。"

"完整映射？"李明有些担忧，"那需要被试者进入更深的脑机接口状态……风险会增加。"

"我知道。"陆远航点头，"所以这次的被试者是我自己。"

"什么？！"

"陆博士，您不能——"

"这是我的决定。"陆远航打断了他们的反对，"我不会让别人承担这个风险。"

他看着李明和程浩，眼神坚定："这项研究是我开始的，如果有人要第一个踏入数字世界，那个人只能是我。"

争论持续了很久，但最终，所有人都接受了他的决定。

实验定在三天后进行。

---

## 十

实验前夜，林诗语来到陆远航的宿舍。

"我听说了。"她一进门就说，"你打算用自己做被试者？"

"是的。"

"这太疯狂了。"她的声音中带着焦虑，"完整的神经元映射从未在人类身上实验过——我们甚至不知道会发生什么！"

"我知道风险。"陆远航平静地说，"但这是唯一的方法。"

"为什么？"林诗语几乎是在质问，"为什么必须是你？"

陆远航沉默了一会儿。

"诗语，你相信命运吗？"他突然问。

"什么？"

"我是说……有些事情，似乎冥冥之中就是你注定要做的。"他看着她的眼睛，"从我第一次接触'零'开始，我就有一种感觉——这条路是为我准备的。无论前面有多少危险，我都必须走下去。"

"这不是命运，这是冲动。"林诗语说，但她的声音已经软了下来。

"也许吧。"陆远航笑了笑，"但有时候，冲动和命运很难区分。"

两人沉默了。

"远航，"林诗语终于开口，"如果……如果实验出了问题……"

"不会的。"

"但如果真的出了问题……"她的声音有些颤抖，"你要知道，我……"

她没有说完。

陆远航看着她，看到她眼中闪烁的光芒——那是担忧、恐惧，还有某种更深的东西。

"诗语，"他轻声说，"等实验结束后，我想请你吃饭。"

"什么？"

"我是说……正式的那种。不是同事之间的工作餐，而是……"他顿了顿，"约会。"

林诗语愣住了。

片刻后，她的脸上浮现出一丝笑意——温暖、羞涩，还有一点嗔怪。

"你这是什么时候提这种事？"

"最好的时候。"陆远航认真地说，"因为这样，我就有了一个必须活着回来的理由。"

林诗语看着他，眼眶有些红了。

"你这个人……"她低声说，"真的很会给自己找压力。"

"所以……你答应了？"

林诗语没有回答。她只是向前一步，轻轻靠在他的肩上。

"答应了。"她的声音很轻，"但你必须活着回来。"

"我保证。"

两人就这样静静地站着，任由时间流逝。

窗外，北京的夜色深沉。远处的灯火星星点点，像是无数双眼睛在注视着这座城市。

在这个夜晚，在这个小小的房间里，两颗心第一次真正靠近。

无论明天会发生什么，这一刻都是真实的。
