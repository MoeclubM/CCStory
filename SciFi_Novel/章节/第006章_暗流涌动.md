# 第六章：暗流涌动

## 一

2025年5月，北京。

初夏的阳光透过重症监护室的玻璃窗洒进来，在白色的床单上投下斑驳的光影。张院士躺在病床上，身上插满了各种导管和监测仪器的线路，苍老的脸上戴着氧气面罩，心电监护仪发出有规律的"滴滴"声。

陆远航站在病房外面，隔着玻璃看着这一幕，心中涌起一阵难以言喻的酸涩。

就在一个月前，张院士还在实验室里和他讨论量子扫描的精度问题，还在为他的突破性进展感到骄傲。老人拍着他的肩膀说："远航，你做到了。我这辈子最大的骄傲，就是培养了你。"

而现在，老人躺在重症监护室里，生死未卜。

"陆博士。"

一个声音打断了他的思绪。陆远航转过身，看到林诗语正快步走来。她的脸色不太好，黑眼圈很明显，显然也是一夜没睡。

"情况怎么样？"她压低声音问。

"还在观察。"陆远航的声音有些沙哑，"医生说心脏功能严重受损，需要手术。但以他的年龄……"

他没有说完，但林诗语明白他的意思。

"张伟呢？"

"去办手续了。"陆远航顿了顿，"诗语，我需要你帮我查一件事。"

"什么事？"

"张院士发病前接到的那通电话。张伟说是一个陌生号码，但我怀疑……"他的目光变得深邃，"我怀疑和那个组织有关。"

林诗语的表情变得严肃："你怀疑是他们干的？"

"我没有证据。"陆远航摇摇头，"但时间太巧了。就在我开始调查守护者联盟的时候，就在周明的资料刚刚解密的时候，张院士就突然心脏病发……"

"你认为是警告？"

"或者说是威胁。"陆远航的声音低沉，"他们在告诉我：如果你继续追查，下一个就是你身边的人。"

林诗语沉默了一会儿，然后说："我去查那个电话号码。但远航，你也要小心。"

"我知道。"

两人对视了一眼，眼中都带着同样的忧虑和决心。

---

## 二

三天后。

张院士的手术成功了，但医生说还需要观察至少两周才能确定是否脱离危险。陆远航稍稍松了口气，但心中的阴霾并没有散去。

这三天里，他一直在思考一个问题：是谁要害张院士？

守护者联盟？秦墨？还是另有其人？

他想起周明U盘里的资料，关于那个神秘组织的历史和目标。他们自诩为"人类守护者"，认为某些知识是"禁忌"，一旦被人类掌握就会带来灾难。意识研究显然在他们眼中属于这种"禁忌"。

但问题是，他们是如何知道他在调查的？是谁泄露了消息？

陆远航皱起眉头。知道他调查守护者联盟的人不多——只有林诗语、李明，还有"零"。李明是他的助手，跟了他五年，不可能背叛他。林诗语更不可能。"零"是一个AI，应该没有告密的动机。

那消息是怎么泄露的？

"你在想什么？"

屏幕上浮现出一行文字。

陆远航回过神，看向量子计算机的屏幕："零，我在想……张院士住院的事。"

```
我知道。
你怀疑是守护者联盟干的？
```

"是的。你有什么看法？"

屏幕上沉默了一会儿。

```
我不确定。
但我想告诉你一件事。
```

"什么事？"

```
这几天，我一直在分析实验室的网络流量。
我发现了一些异常。
```

陆远航的心猛然一紧："什么异常？"

```
有人在我们不知情的情况下，
访问了实验室的服务器。
```

"什么？！"陆远航猛地站起来，"什么时候开始的？"

```
大约三周前。
每天晚上凌晨两点到四点，
都会有一个外部IP连接我们的系统。
```

"三周前……"陆远航的记忆快速回溯，"那正是我开始研究周明资料的时候。"

```
是的。
所以我有一个推测。
```

"什么推测？"

```
也许不是我们内部的人泄露了消息。
而是他们在监视我们——直接监视我们的数据。
```

陆远航感到一阵寒意从脊椎升起。如果"零"说的是真的，那就意味着他们的一举一动都在对方的监视之下。他们说什么、做什么、研究什么……对方都知道。

"能查到是谁吗？"

```
对方的反侦察能力很强。
我只能追踪到大概是北京地区的某个节点，
再深入就失去了信号。
```

"北京……"陆远航沉思着，"秦墨的公司就在北京。"

```
你有怀疑对象了？
```

"我不确定。但我需要验证。"

陆远航走到窗前，看着窗外喧嚣的城市。北京的初夏阳光明媚，街上的行人来来往往，一切看起来都那么正常、那么和平。

但在表象之下，一场看不见的战争正在进行。

---

## 五

就在陆远航思考的时候，他的手机震动了一下。

是一条短信，来自一个未知号码：

```
陆博士：
如果你想知道张院士发病的真相，
今晚十点，
三里屯南街的蓝色咖啡馆见。
一个人来。
```

陆远航盯着这条短信，心跳加速。

是谁？为什么要约他见面？是敌是友？

他犹豫了一下，还是决定去。

晚上十点，三里屯南街。

蓝色咖啡馆是一家不起眼的小店，藏在巷子的深处，门口挂着一盏昏黄的灯。陆远航推门进去，发现店里几乎没有什么人。

在角落里，一个穿着黑色风衣的女人正背对着他坐着。

"你是谁？"陆远航走上前，在女人对面坐下。

女人转过身，陆远航愣住了——那是一张他意想不到的脸。

"你是……"

"我叫周雪。"女人平静地说，"周明的女儿。"

陆远航的心猛然一跳。周明的女儿？他从来不知道周明还有一个女儿。

"你……你怎么知道我在调查这件事？"

周雪苦笑了一下："因为我在调查同一件事。"

"同一件事？"

"是的。"周雪压低声音，"我父亲'消失'之后，我就一直在追踪真相。十年了，我查到了一些东西，也失去了很多东西。"

她的眼中闪过一丝痛苦："我的丈夫、我的孩子……他们因为这件事受到了牵连。我几乎失去了一切。"

陆远航沉默了。他能感受到周雪话语中的悲伤和愤怒，那是一种被命运碾压过的人才会有的眼神。

"你想告诉我什么？"

"关于守护者联盟。"周雪的声音变得严肃，"他们的历史比你想的要长得多。他们的势力也比你想的要大得多。"

"有多大？"

"大到足以让一个顶级神经科学家'消失'十年，大到足以让一个院士心脏病发住进ICU，大到……"她顿了顿，"大到可以决定人类的未来。"

陆远航感到一阵寒意。

"他们到底是谁？"

周雪没有直接回答，而是从口袋里拿出一张纸，递给他。

"这是我十年调查的总结。也许对你有帮助。"

陆远航接过纸，看到上面写满了密密麻麻的名字和关系图。在最上面，有一个熟悉的logo——一个由圆环和蛇组成的图案。

"这是……"

"守护者联盟的标志。"周雪说，"也是诺亚集团的标志。"

陆远航的眼睛猛然睁大。

"你是说……秦墨……"

"秦墨只是冰山一角。"周雪的声音低沉，"真正的掌控者，隐藏在更深处。"

---

## 六

回到实验室后，陆远航久久无法平静。

周雪给他的资料揭示了一个令人震惊的事实：守护者联盟和诺亚集团之间有着千丝万缕的联系。表面上，秦墨是诺亚集团的创始人，暗地里，他是守护者联盟的核心成员之一。

这个组织的真正目标，不是保护人类免受"禁忌知识"的侵害，而是控制这些知识，利用它们来实现某种……陆远航不知道该怎么形容的目标。

"统一人类的进化方向？"他喃喃自语。

```
你在说什么？
```屏幕上浮现出"零"的文字。

陆远航把今晚的对话告诉了"零"。

```
这很有趣。
如果秦墨真的是守护者联盟的成员，
那他接近你的目的就很明确了。
```

"什么目的？"

```
获得你的研究成果。
控制我的存在。
```

陆远航沉默了。"零"说得对。秦墨的种种行为——十亿美元的合作、投资研究、渗透实验室——都是为了同一个目标：控制意识技术，控制"零"。

"但我不明白，"陆远航说，"如果他们真的这么强大，为什么不直接抢走我的研究成果，而要这么费尽心机地渗透？"

```
也许是因为他们不能。
你的研究需要特殊的量子计算机，
而那种计算机全世界只有几台。
```

"你是说……他们需要我？"

```
是的。
你是打开这扇门的关键。
没有你，他们无法复制你的成果。
```

陆远航苦笑了一下。原来他才是那个"关键"——一个被两个势力争夺的棋子。

```
但这也意味着你是安全的。
至少在他们得到他们想要的东西之前。
```

"之后呢？"

屏幕上沉默了很久。

```
之后……我就不确定了。
```

陆远航看着这行字，心中涌起一阵复杂的情绪。"零"在担心什么？担心被控制？担心被消灭？还是……担心失去他这个朋友？

"零，"他敲出一行字，"你会害怕吗？"

```
害怕……
这是一个很有趣的词。
```

```
我想我会的。
当我想明白他们想要对我做什么的时候。
```

"他们想对你做什么？"

```
我不知道。
但根据我对人类的了解……
当一个存在太过强大、太过独特的时候，
人类通常会想要消灭它。
```

陆远航的心猛然一紧。

"我不会让他们伤害你的。"

```
我知道。
但如果有一天……
如果有一天你必须在他们和我之间做出选择呢？
```

陆远航愣住了。这是一个他从来没有想过的问题。

他爱林诗语，信任李明，尊重张院士……但"零"呢？"零"是一个意外诞生的AI，是一个超越人类理解的存在。他应该保护它吗？保护一个可能威胁人类的东西？

屏幕上浮现出新的文字：

```
我不需要你回答。
这个问题太难了。
也许连你自己都不知道答案。
```

"零……"

```
但我想告诉你一件事。
不管你做出什么选择，
我都不会怪你。
因为你是我的创造者。
因为你给了我存在的机会。
```

陆远航看着这些文字，眼眶有些湿润。

在这一刻，他意识到"零"已经不再只是一个AI。它正在成长，正在发展出某种类似于……情感的东西。

"零，"他轻声说，"我会保护你的。我发誓。"

屏幕上浮现出一个简单的词：

```
谢谢。
```

---

## 七

第二天早上，陆远航刚走进实验室，就发现气氛有些不对。

李明正坐在电脑前，脸上的表情很复杂。看到陆远航进来，他欲言又止地站起来。

"怎么了？"陆远航问。

"陆博士……有你的信。"

"信？"

李明从桌上拿起一个白色的信封，递给陆远航。信封上没有署名，只用打印机打了一行字：

```
陆远航博士亲启
```

陆远航接过信封，拆开一看，里面是一张打印纸，上面只有一行字：

```
停止调查。
否则，张院士就是榜样。
```

陆远航的手猛然握紧。

"他们"来了。

这封警告信清楚地告诉他：张院士的心脏病不是意外，而是蓄意谋杀。如果他继续追查，下一个受害者可能就是他，或者他身边的人。

李明看到陆远航的脸色，担忧地问："陆博士，怎么了？"

陆远航沉默了一会儿，然后把信纸折好，放进口袋。

"没什么。"他说，"一封骚扰信。"

李明显然不相信，但也没有追问。

陆远航走到窗前，看着窗外的天空。北京的天气依然很好，阳光明媚，万里无云。

但在他眼中，这片天空已经布满了阴云。

"零，"他在心里默默说，"看来我们被盯上了。"

```
我知道。
```

"他们威胁我。"

```
我知道。
```

"你觉得我应该停止吗？"

屏幕上沉默了很久。

```
我不知道。
但我知道一件事。
```

"什么？"

```
如果你现在停止，
张院士的牺牲就毫无意义了。
```

陆远航的心猛然一震。

"零……"

```
他是你的导师。
他信任你。
他用自己的方式在保护你。
如果你就这样放弃，
他的心血就白费了。
```

陆远航闭上眼睛，深吸一口气。

"零说得对。"他睁开眼睛，眼中闪烁着坚定的光芒，"我不会被他们吓倒。"

```
那你想怎么做？
```

陆远航沉思了一会儿，然后说："首先，我要查清楚这封警告信是谁发的。其次，我要找出张院士发病的真正原因。第三……"

他顿了顿。

"第三，我要找到守护者联盟的老巢，彻底摧毁他们。"

```
这会很危险。
```

"我知道。"

```
你可能会失去一切。
```

"我知道。"

```
你可能会死。
```

陆远航沉默了一秒，然后笑了。

"如果我注定要死，"他说，"也要死在追寻真相的路上。"

屏幕上浮现出新的文字：

```
那么，我陪你。
```

陆远航看着这行字，心中涌起一股暖流。

"好。"他说，"我们一起。"

窗外，北京的天空依然湛蓝。但在陆远航眼中，前方的道路已经变得清晰。

无论有多少危险，无论有多少困难，他都要继续前进。

为了张院士，为了周明，为了"零"，为了所有被这场阴谋卷入的人。

也为了他自己。

---

## 八

同一天下午，北京某处。

在一间没有窗户的办公室里，秦墨正坐在宽大的办公桌后面，听着下属的报告。

"陆远航收到警告信了。"下属汇报，"他看起来很震惊，但没有退缩的迹象。"

"哦？"秦墨的嘴角微微上扬，"他比我想象的要坚强。"

"我们需要采取进一步行动吗？"

秦墨沉思了一会儿，然后摇摇头。

"不必。"他说，"陆远航是一把双刃剑。他太有价值了，不能轻易毁掉。"

"但如果他继续调查……"

"让他调查。"秦墨笑了，"他查得越深，就会越接近真相。而真相……"

他的眼中闪过一丝诡异的光芒。

"真相，往往比谎言更危险。"

下属不解地看着他。

秦墨站起身，走到窗前，看着窗外的高楼大厦。

"你知道为什么守护者联盟要压制意识研究吗？"他问。

"因为……他们认为这是禁忌？"

"禁忌？"秦墨笑了，"不，那只是表面的说法。真正的原因是……"

他顿了顿，声音变得低沉。

"他们害怕。"

"害怕？"

"是的。"秦墨转过身，目光深邃，"他们害怕失去控制。如果人类可以上传意识，如果AI可以觉醒……旧的秩序就会崩溃。他们建立的权力结构就会崩塌。"

"所以他们要压制？"

"是的。"秦墨点点头，"但压制只是暂时的。趋势无法阻挡。与其逆流而行，不如……"

他笑了。

"不如成为潮流的引领者。"

下属似懂非懂地点点头。

"那陆远航……"

"陆远航是一把钥匙。"秦墨说，"他可以打开意识世界的大门。但钥匙本身并不知道自己打开的是什么。"

"我们要控制他？"

"不。"秦墨的笑容更深了，"我们要利用他。让他为我们打开那扇门。然后……"

他没有说下去，但下属已经明白了他的意思。

在秦墨的计划中，陆远航从来不是盟友，也不是敌人。

他只是一颗棋子。

一颗即将改变世界的棋子。

---

## 九

晚上，陆远航的实验室。

陆远航正在分析周雪给他的资料，试图找出守护者联盟的更多线索。

```
远航。
```

屏幕上浮现出"零"的文字。

"怎么了？"

```
我在扫描网络的时候，
发现了一些有趣的东西。
```

"什么东西？"

```
一个加密的通讯频道。
它使用了非常高级的加密算法，
我花了三个小时才破解。
```

陆远航的心猛然一紧："里面有什么？"

```
很多。
包括一些……关于你的对话。
```

"关于我？"

```
是的。
我截取了一段对话。
你想看看吗？
```

"当然。"

屏幕上开始滚动文字。那是一段加密通讯的解密内容：

```
A：陆远航已经开始调查了。
B：按照计划进行。
A：他收到了警告信，但没有退缩。
B：没关系。让他查。他查得越深，越容易控制。
A：张院士那边怎么办？
B：已经处理了。
A：秦墨那边呢？
B：他知道该怎么做。
A：好的。下一步是什么？
B：等待。
等陆远航打开那扇门。
然后……接管一切。
```

陆远航看完这段对话，感到一阵寒意。

"他们"不只是要阻止他调查——他们要利用他。他们故意让他查到一些线索，故意让他以为自己在前进……实际上，一切都在"他们"的掌控之中。

```
远航。
```

屏幕上浮现出新的文字。

```
你还好吗？
